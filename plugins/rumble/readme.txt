Embed a responsive rumble video.

== Installation ==
To install the Rumble plugin:

* Install Rumble automatically or by uploading the ZIP file.
* Activate the plugin through the \'Plugins\' menu in WordPress.
* Rumble is now activated. Go to the rumble settings and enter publisher ID.
* You are now ready to search and embed videos.
