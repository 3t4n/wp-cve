The Blog Posts Calendar Widget allows you to display your archived or future posts in a calendar as a sidebar widget. 

== Description ==

Monthly grid view calendar widget highlighting archived and/or future posts.

<strong>Additional Features:</strong>
<ul>
<li>Popup details for each date with posts</li>
<li>Optional details include post author and comment count.</li>
<li>Ability to show posts by post type, category, specific taxonomy and/or term</li>
</ul>

[__For more information or to request additional features, please visit the plugin page__]( http://presshive.com/plugins/wordpress-blog-post-calendar-plugin)

== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory, or download and install automatically through your admin panel.
     
2. Activate the plugin through the 'Plugins' menu in WordPress.

3. From your admin console, go to Appeareance > Widgets, drag the Blog Post Calendar Widget to wherever you want it to be and click on Save.

== Frequently Asked Questions ==

= Where can I get more information about using Blog Post Calendar Widget? =

See the [plugin homepage]( http://presshive.com/plugins/wordpress-blog-post-calendar-plugin).

= Customizations =

You have total control over your calendar style. Add wp_calendar.css to your theme directory and customize it as you see fit.
