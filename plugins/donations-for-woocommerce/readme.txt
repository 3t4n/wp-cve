Easily accept donations of varying amounts through your WooCommerce store.

== Description ==

The Donations for WooCommerce plugin facilitates acceptance of donations of varying amounts through your WooCommerce store. It adds a Donation product type that allows the end user to set the price.

== Installation ==

1. Click "Plugins" > "Add New" in the WordPress admin menu.
2. Search for "Donations for WooCommerce".
3. Click "Install Now".
4. Click "Activate Plugin".

Alternatively, you can manually upload the plugin to your wp-content/plugins directory.
