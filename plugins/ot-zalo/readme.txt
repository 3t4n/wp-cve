== Description ==

OT Zalo - Zalo Chat Widget, Follow widget

== Installation ==

1. Upload contents of the plugin zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit Settings Page : Dashboard -> Settings -> OT Zalo
4. Setup Zalo official account ID and welcome message...
