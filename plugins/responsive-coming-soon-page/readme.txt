== Description ==

Coming Soon plugin offers many amazing features to design your coming soon page. Settings is user friendly as it can be, you can setup your coming soon page with one click. You can however customize it in many creative ways, setup custom logo, background graphics, custom content, subscribe forms, popular social networks icons, colors, SEO friendly, and many more.

If your site is under development then use this plugin to display a coming soon page to your visitors. The plugin allows collecting all visitors subscriptions. When your site will live plugin notify all your active subscriber about site launch.

== Installation ==

1. Upload the entire `responsive-coming-soon-page` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Weblizar Coming Soon menu page to customize plugin settings.
4. Live it & Enjoy.
