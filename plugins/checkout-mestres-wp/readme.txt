== Installation ==

= Automatic installation through WordPress =

1. Select **Add New** in your WordPress on the **Plugins tab**.
2. Look for **Checkout Mestres WP**.
3. Click on **Install** Now and **Activate Plugin**.

= For manual installation via FTP: =

1. Upload the **checkout-mestres-wp** to the **/wp-content/plugins/** folder.
2. Activate it in the **Plugins** section of your WordPress.
