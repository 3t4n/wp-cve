Show your posts and pages in tabbed or collapsible way !

== Description ==

The plugin that allows you to display widget with blog posts and pages content in tabbed or collapsed (accordion) view.
The user may also select to show: post thumbnail, readmore link and publish date. Number of panels and their ordering are unlimited, even thumbnail size can be adjusted which makes this widget very friendly and flexible to user. Simple, easy to style, well optimized and clean package.

== Installation ==

1. Upload the 'pe-panels' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the 'Widgets' page found under the 'Appearance' menu item
4. Drag an drope 'PE Panels' to the target widget area and choose your options

**Configuration (see screenshot of backend):**

1. Widget title
2. Type of view ( chose between 2 available views - tabbed and collapsed)
3. Posts and Pages to show as panels
4. Items ordering (depends on items chosen ordering)
5. Content length limit
6. Option to show ‘readmore’ link below content
7. Option to show creation date in certain positions
8. Option to show post thumbnail above content
9. Thumbnail alignment ( to left, to right or center)
10. Thumbnail size (you can chose from all theme sizes or set custom size - thumbnail will be generated automatically)
11. Image width in pixels
12. Image height in pixels
13. Option to crop thumbnails (you can chose crop area).
14. (Tabbed view) Option to resize panel headings to fill available space (each heading will have fixed width)
15. (Tabbed view) Responsive break point (widget will switch desktop <-> mobile layout in this respolution)
16. (Collapsed view) Show or Hide first panel content
17. (Collapsed view) Allow open more than one panel
