== Description ==

### Very simple and lightweight plugin to add any type of tracking code on your website and also block the direct access to thank page. "WP Tracking Manager" has provided an option to add page specific tracking code.

= Key Features =

* Post type specific tracking code
* Page specific tracking code
* Block Direct Access of Thank-You Page
* Adwords Conversion Code
* Facebook Pixel Code
* Google Analytics A/B Testing Code 
* Linkedin Conversion Code


"WP Tracking Manager" provide an important feature to block the thank-you page direct access and allow to add auto "noindex follow" meta robots on all thank-you pages .

= Features = 
 * An option to add tracking code on all pages of the website.
 * An option to add tracking code only on specific pages (i.e Thank-you & Landing pages)
 * An option to block direct access to thank-you page.

== Frequently Asked Questions ==

#Can I add tracking code on thank-you pages?

Yes, We have given an option on pade edit screen to add the tracking/conversion code.

#Can I add tracking code on specific custom post types i.e product, blog post?

Yes, We have given an option to add post type specific tracking code. It means you can add any tracking code for any specific custom post type seperately  

#Can I add tracking code only for a single page?

Yes, We have given an option to add page specific tracking code.

Yes, We have given an option to block thank-you page direct access. Guest user will not able to access thank-you without submit the form.

== Installation ==

Installing the plugin:

1.  Unzip the plugin's directory into `wp-content/plugins`.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  The settings page will be available under settings menu (Settings >> WP Tracking Manager)
