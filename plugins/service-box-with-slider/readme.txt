Service Box Showcase is a powerful &amp; robust but easy to represent your services with grid and slider. It comes with 50 unique layouts and creates unlimited service showcases with 200+ custom fields.

Service Box offers you to add an owl carousel type grid slider containing services name, description, image, icons, external or internal URL.

Service box is an effective tool in WordPress for creating amazing service showcase and sliders with beautiful transition effects. With this awesome plugin, you can add services along with their description and link so that viewers can evaluate the weight of your services so easily. This will certainly help you a lot in your business exposure.

Our rich library of templates will definitely help the user to pick the right one for them. You can build an interactive service page within minutes by using Service Box with Slider plugin. It is very easy to use the provided shortcode into the posts and pages. You can also put the shortcode into the page builders, thus, it can be included in the theme for displaying the services within the theme using PHP function. 

We provide a free 1600+ font awesome icon with our service box with slider plugin along with some cool features to play with. You can find so much flexibility with our plugin.
