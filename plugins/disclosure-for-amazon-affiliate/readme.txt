== Installation ==
Install:
1. From the dashboard of your site, navigate to Plugins –> Add New.
2. Select the Upload option and hit “Choose File.”
3. Upload the Amazon Associate Disclosure directory to the /wp-content/plugins directory
4. Activate the plugin through the ‘Plugins’ menu in WordPress

Configure (none required):
Go to settings - Amazon Associates Disclosure. They will automatically be enabled but you can customize the disclosure and formatting if you choose to. 


== Frequently Asked Questions ==
Can I edit the disclosure? 
Yes, you can edit the disclosure to whatever you want to include. Adding a link to your terms or disclosure page is popular. 

Is Amazon Associate Disclaimer really free?
Yes, 100% free with no need for email opt in or anything else. Enjoy! 

Will this plugin ensure I am compliant with all of Amazon Associates Operating Agreement?
No, this plugin helps deal with 2 of the most commonly incorrectly done parts of the agreement… the affiliate disclosure and trademark disclosure. There are MANY other requirements that this plugin does not address.
