== Description ==
Universal Honey Pot is a powerful and user-friendly WordPress plugin that provides a plug-and-play solution for protecting your forms against unwanted spam. It automatically adds honey pots to all supported form plugins, such as Contact Form 7, Formidable Forms, Forminator, Elementor Pro, Divi and WPforms. With Universal Honey Pot, you can safeguard your WordPress site from spam without any manual configuration.

= Key Features =

* Easy to use: Simply install and activate the plugin, and it will start adding honey pots to supported form plugins.
* Plug-and-play: No manual configuration required. Universal Honey Pot seamlessly integrates with compatible form plugins.
* Wide compatibility: Supports popular form plugins like Contact Form 7, Formidable Forms, Forminator, Elementor Pro, Divi and WPforms, with more to be added in future updates.
* Effective spam protection: Honey pots act as a hidden trap for spambots, reducing the amount of unwanted submissions.
* Lightweight and optimized: Universal Honey Pot is designed to be resource-friendly and ensures smooth performance on your website.

== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings / Universal Honey Pot and check if your form plugins are supported.

== Frequently Asked Questions ==

= Which form plugins are currently supported? =
Universal Honey Pot currently supports Contact Form 7, Formidable Forms, Forminator, Elementor Pro, Divi and WPforms. Support for additional plugins will be added in future updates.

=  Will Universal Honey Pot affect the functionality of my form plugins? =
No, Universal Honey Pot works in the background (with hidden fields) and does not interfere with the normal functionality of your form plugins. It adds an additional layer of protection against spam.

=  Can I customize the honey pot fields? =
Yes, you can add or remove honey pot fields with this filter `universal_honey_pot/fields`. You can also customize the field names and values.

=  I found a bug in the plugin. Where can I report it? =
We appreciate your feedback. Please report any bugs or issues in the Support section of the plugin's WordPress.org page. We'll investigate and address the problem as soon as possible.
