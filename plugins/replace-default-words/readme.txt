Replace the default words in the core, plugins and themes

== Description ==

This plugin helps you replace words in WordPress core, themes and plugins.

Note: Plugins and themes must have a language file to be edited with "Replace Default Words"

**Languages**

* Persian - [صنایع سلولزی ایران](http://packcenter.info/)
* Kurdish - [Mekaeil](http://mekaeil.me/)
* English

Send translation file to: Tansu68@live.com

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Tools > RDW and add new word.
