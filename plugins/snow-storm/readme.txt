== Description ==

Display falling snow flakes on the front of your WordPress website for a festive presentation.

Useful Links:

* <a href="https://tribulant.com/docs/wordpress-snow-storm-plugin/7799">Online Documentation</a>
* <a href="https://tribulant.net/snowstorm/">Live Demonstration of Snow Storm plugin</a>
* <a href="https://tribulant.com">Get Support for Snow Storm plugin</a>

The Snow Storm plugin was developed by <a href="https://tribulant.com">Tribulant Software</a> so that WordPress website owners can display falling snow during the festive season eg. Christmas to decorate their website.

== Installation ==

Installing this plugin is quick and easy:

= Automatic Installation =

1. Go to 'Plugins' in your WordPress dashboard.
2. Click on the 'Add New' button at the top to add a new plugin.
3. Search for "snow storm" and you'll see the Snow Storm plugin by Tribulant Software.
4. Click the 'Install Now' link to install the plugin.
5. After installation, click the 'Activate Plugin' link to activate it.

In case the automatic installation doesn't work due to any reason, use the manual installation below:

= Manual Installation =

1. Download the WordPress Snow Storm plugin.
1. Extract the ZIP archive on your computer to get the `ezgz` folder.
1. Upload the `snow-storm` folder to your `wp-content/plugins` directory.
1. Activate the 'Snow Storm' plugin on your 'Plugins' page in the WordPress dashboard.

== Frequently Asked Questions ==

= How do I configure it? =

Go to <strong>Settings > Snow Storm</strong> in your WordPress dashboard where you will see configuration settings.
