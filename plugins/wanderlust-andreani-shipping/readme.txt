Con este plugin vas a poder obtener costos de envio para WooCommerce utilizando la API publica de Andreani.

 
 == Video ==
[youtube https://www.youtube.com/watch?v=hQHjK9-oQW0]


== Instalacion ==

1. Install the WooCommerce Andreani Shipping - Plugin.
2. Activate the plugin.
3. Go to the WooCommerce Settings Page. 
4. Access Shipping Tab.
5. Select "Shipping Zones" and Add Andreani.
6. Check the Enable/Disable Checkbox.
7. Enter the settings that you would like you to use.
8. Enter the dimensions of the products, (Andreani uses this to determine suitability of products).
