Релевантная реклама для ваших читателей. Рекламодатели сервиса платят в 2-3 раза больше за 1 тыс. показов страниц, чем привычные рекламные сетки. Страница выкупается полностью, на ней размещается максимум четыре рекламных блока, которые выглядят нативно в стиле сайта.

== Description ==
Релевантная реклама для ваших читателей. Наши партнеры зарабатывают в 2-3 раза больше за 1 тыс. показов страниц, чем от всех других рекламных сетей вместе взятых.

= О платформе Native Rent =
Рекламная платформа Native Rent разработала свой плагин для сайтов на платформе Wordpress. Теперь любой медиа-сайт может подключиться к платформе Native Rent через плагин и начать зарабатывать с помощью модели "аренда статей". Native Rent - единственная рекламная платформа в Рунете, которая имеет свой официальный плагин в каталоге плагинов Wordpress.

[Подключить сайт](https://nativerent.ru/register/partner)

= Возможности плагина =
Все коды, необходимые для работы плагина, подключаются автоматически. Вы можете управлять рекламой и правилами её размещения на странице настроек плагина:

* Автоматическая и ручная расстановка горизонтальных рекламных блоков на странице.
* Подключение и настройка всплывающего рекламного блока.

== Installation ==
Для работы с плагином потребуется регистрация на сайте Native Rent в качестве партнера: https://nativerent.ru/register/partner

= Инструкции =
Установите плагин и настройте отображение рекламы:

* Установите плагин и авторизуйтесь в нем с теми же учетными данными, с какими зарегистрировались на платформе NativeRent;
* Проверьте настройки расстановки рекламы и, при желании, задайте собственные селекторы для вывода блоков;
* Проверьте отображение рекламных блоков на любой странице, где есть коды, добавив к ссылке параметр «?_nrdemo».
