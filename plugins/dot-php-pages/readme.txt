Dot Php Pages

== Description ==

This Plugin Adds .php Extension to Your Pages Like http://www.yoursitename.com/yourpage.php WARNING: FIRST DEACTIVATE THE OTHER EXTENSION PAGES PLUGIN.  Just Activate The Plugin And Enjoy PHP Pages.

== Powered By ==
KSM (http://ksmughal.com)

== Installation ==

1. Upload dot-php-pages.php to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Appends dot-php-pages to the URL of PAGES when using permalinks.


== Frequently Asked Questions ==

= Does This Work =

It works On WordPress 2.5.1 to WordPress 3.0.1 

= Plugin Activation Problem while using altranative page extension =

FIRST DEACTIVATE THE OTHER EXTENSION PAGES PLUGIN LIKE .html on pages.

It works On Custom Permalinks.
