== Description ==

WooCommerce Product Configurator appears to be a must-have app for any component and part building pages, particularly PC builder pages. Imagine if customers could go through the process of building a PC and selecting the components and parts they want through steps. That would be incredible, and that is what WooCommerce Product Configurator can provide.

WooCommerce Product Builder is a plugin that allows your customers to build a full product set from small parts step by step. This plugin works as a WooCommerce Product Configurator based on the WooCommerce platform, promising to provide you with many useful features like compatible, email completed product, attributes filters …

### FREE VERSION:

&#9658; **Create Product Builder pages:** Each product builder will display on front-end as a page, customers will build their products on this page. You can add as many product builders as you want.

&#9658; **Multiple steps:** Add as many steps as you need customers to select product parts.

&#9658; **Change step name:** Select a name for each step like STEP 1 – Mainboard, STEP 2 – CPU… Use the shortcode  {step_number} to display the number of currency step.

&#9658; **Select product and categories for each step:** You can select which products categories will appear in each step and then customers can pick one (or more) in that step.

&#9658; **Quantity field:** This option allows customers to select products quantity. If this option is disabled, the quantity will be set as 1.

&#9658; **Preview button:** The preview feature allows customers to view their full product before purchase. The preview page will appear in the last step, by enabling this option you will allow customers to visit the preview page from any step.

&#9658; **Product per page:** Select how many products will be displayed on one page in front-end.

&#9658; **Attribute Widget:** Attribute widgets helps your customers to easier to find the product they want to purchase. To enable attribute widgets, visit Appearance/ Widgets, look for WC Product Builder Widgets and drag/drop them to WC Product Builder Sidebar Area.  Please take note that the plugin will not work with custom attributes. To add/edit attributes, please visit Products/ Attributes

&#9658; **WC Product Builder Price Filter:** Allow customers to find products in a price range.

&#9658; **WC Product Builder Rating Filter:** Allow customers to find product base on ratings.

&#9658; **WC Product Builder Attribute Filter:** Allow customers to find product base on attributes. Like color, brand…

&#9658; **Preview page:** After finish building a product, customers can review it again in the preview page. This page will display the information about product names, prices, quantity, total price. Then customers can select to check-out or email the completed product.

&#9658; **Email:** Customers can send their finished product by emails

&#9658; **“Thank you” message:** A message notice customers that their products have been sent successfully.

&#9658; **Design:** The product builder front-end page is fully customizable to fit with your theme.

&#9658; **Text color:** Change button text color

&#9658; **Background color:** Change the button background color

&#9658; **Primary text color:** Change the button text hover color

&#9658; **Primary background color:** Change the button background hover color

&#9658; **Button Icon:** Select the button icon style. There are 2 styles of text and icon.

&#9658; **Steps** You can added maximum 3 steps.

### PRO VERSION:

&#9658; **All features from free version.**

&#9658; **Select multiple products:** Allow customers to add more than one product in each step. If this option is disabled, when customers select a product they will be automatical redirect to the next step.

&#9658; **Compatible:** Make sure all selected product parts will work properly. The compatible feature works based on attributes values. For example: In STEP 1 – CPU, customers select a CPU with the attribute “Socket” has its value as “LGA 1151”, then in STEP 2 – Mainboard he will find only mainboards which have the socket LGA 1151 (this attribute value must be added to products)

&#9658; **Unlimited steps** You can added unlimited steps.

&#9658; **Custom CSS:** Add your own CSS code to configure the front-end style as you want.
