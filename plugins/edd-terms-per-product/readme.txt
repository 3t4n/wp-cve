== Description ==

Allow terms of use to be specified on a per-product basis for Easy Digital Downloads.

This is an extension for [Easy Digital Downloads](http://wordpress.org/extend/plugins/easy-digital-downloads/) and does not function on its own.

== Installation ==

1. Activate the plugin
2. Add the Terms of Use for each product from the Download edit screen

== Screenshots ==

1. Download product terms
2. Checkout screen with terms of use per-product
