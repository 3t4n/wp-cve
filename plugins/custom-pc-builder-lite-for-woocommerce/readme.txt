Custom PC Builder for WooCommerce Lite allows your customers to build full PCs on their own

== Description ==
Create a product listing page that allows your customers to select components of computer, see estimated costs before deciding to put all of them in the cart.
Insert the shortcode `[custom_pc_builder]` in your post/page

== FREE VERSION ==
* Rows: you can added maximum 7 rows.
* Product searching and sorting
* Display estimated costs
* Add all items to cart

== PRO VERSION ==

* **All features from free version.**

* **Unlimited rows:** You can add unlimited rows, then you can choose to add more product categories .

* **Filter Product Attribute:** Attribute sidebar filter helps your customers to easier to find the product

* **Export Excel file** Your customers can download your excel file which has content is your company information: logo, address, phone number, email.

* **Export Image file:** Your customers can download your image file which has list of products, price, total and logo.

* **Multi PC Builder Page:** You create a PC Builder page, then you will create a shortcode, using the shortcode for any frontpage, customers will build their products on this page.

* **Premium support by domain, forever.**

== Installation ==
1. Upload the entire `custom-pc-builder-lite-for-woocommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Go to Settings - Custom PC Builder
