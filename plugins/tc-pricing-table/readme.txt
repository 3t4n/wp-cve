== Installation ==

Installing  TC Pricing Table is easy like  a regular WordPress plugin. Here goes the step by step Guide to Install TC Pricing Table plugin :

= Install Plugin using WordPress Admin Panel Search Option =
- In your Admin, go to menu Plugins > Add New
- Search TC Pricing Table
- Click to install
- Activate the plugin
- TC Pricing Table post type will show below POSTS .

= Install Plugin using Upload Method in WordPress Admin Panel =

- Download the latest version of TC Pricing Table (.zip file)
- In your Admin, go to menu Plugins > Add New
- click on the Upload Tab
- Click on Choose File from your Hard Disk
- Browse and Select downloaded tc-pricing-table.zip & click on  Install Now  button
- Activate the plugin
- TC Pricing Table type will show below POSTS.

= Install  Plugin using FTP Manager =
- Upload TC Pricing Table folder with all files inside the /wp-content/plugins/ directory
- Go to WordPress dashboard > Plugins & Activate the  TC Pricing Table plugin
- TC Pricing Table post type will show below POSTS with its owl icon.
