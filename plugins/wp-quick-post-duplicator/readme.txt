Copy or Duplicate any post types, including pages, taxonomies & custom fields with a single click.

== Description ==

This plugin allows you to clone/duplicate posts of any type for further editing.

Custom post types are supported, along with custom taxonomies and custom fields.

Hover over a post in the edit screen and click on  "Duplicate This Item" to create a duplicate/clone post.

Plugin uses standard actions and filters only.

== Frequently Asked Questions ==

= How to duplicate custom post types? =

Go to wp-admin -> posts/pages/custom post, On quick edit bar click on Duplicate this item.
