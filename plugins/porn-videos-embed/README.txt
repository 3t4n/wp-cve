= Use =

 1) Manualy

`[xvideos url="url-video"]`
`[xhamster url="url-video"]`
`[pornhub url="url-video"]`
`[tnaflix url="url-video"]`
`[tube8 url="url-video"]`
`[vporn url="url-video"]`
`[gotporn url="url-video"]`
`[drtuber url="url-video"]`
`[youjizz url="url-video"]`
`[sunporno url="url-video"]`
`[eporner url="url-video"]`
`[cliphunter url="url-video"]`
`[homemoviestube url="url-video"]`
`[pervertslut url="url-video"]`

You can set width and height.

`[xvideo url="url-video" width="640" height="360"]`

2) Button 'PVE' in wordpress editor.

= WpForo support =

The shorcodes work with WpForo plugin. You must enable support Wordpress Shortcode in Settings WpForo.
You must add shortcode manualy.
