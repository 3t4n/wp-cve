== Description ==

This is a Flutterwave payment gateway for Easy Digital Downloads.

You can sign up for a Flutterwave merchant account [here](https://dashboard.flutterwave.com)

Flutterwave Easy Digital Downloads payment gateway plugin allows you to accept payment through multiple payment channels via Flutterwave

With this Flutterwave Easy Digital Downloads payment gateway plugin, you will be able to accept the following payment methods in your shop:

* __Card Payment__
* __Bank Account Payment__
* __Mobile Wallet Payment__


= Note =

*	You will need to set your Base Country to either Nigeria, Kenya, Ghana or South Africa in the General tab on the Easy Digital Downloads settings page
*	Currencies supported by Flutterwave: NGN, USD, EUR, GBP, KES, GHS, ZAR

= Plugin Features =

* __Multiple payment channels__ available for your customers.
* __Seamless integration__ into the Easy Digital Downloads checkout page.
* __Add Nigerian Naira, South Africa Rand, Ghanaian Cedi and Kenyan Shilling currency symbol__ to select it go to the Easy Digital Downloads Settings from the left hand menu, then click General from the top tab. From the Currency sub tab select the currency you want to use, then click on Save Changes for your changes to be effected.

== Installation ==

= Automatic Installation =
* 	Login to your WordPress Admin area
* 	Go to "Plugins > Add New" from the left hand menu
* 	In the search box type __Flutterwave Easy Digital Downloads payment gateway__
*	From the search result you will see __Flutterwave Easy Digital Downloads payment gateway__ click on __Install Now__ to install the plugin
*	A popup window will ask you to confirm your wish to install the Plugin.
*	After installation, activate the plugin.
* 	Open the settings page for Easy Digital Downloads  and click the "Payment Gateways" tab.
* 	Click on the __Flutterwave__ subtab from the available Checkout Options
*	Configure your __Flutterwave Payment Gateway__ settings. See below for details.

= Manual Installation =
1. 	Download the plugin zip file
2. 	Login to your WordPress Admin. Click on "Plugins > Add New" from the left hand menu.
3.  Click on the "Upload" option, then click "Choose File" to select the zip file from your computer. Once selected, press "OK" and press the "Install Now" button.
4.  Activate the plugin.
5. 	Open the settings page for Easy Digital Downloads  and click the "Payment Gateways" tab.
6. 	Click on the __Flutterwave__ subtab from the available Checkout Options
7.	Configure your __Flutterwave Payment Gateway__ settings. See below for details.



= Configure the plugin =
To configure the plugin, go to __Downloads > Settings__ from the left hand menu, then click on the __Payment Gateways__ tab. You will see __Flutterwave__ as a subtab. Click on it to configure the payment gateway.


* __Enable Test Mode__ - Check to enable test mode. Test mode enables you to test payments before going live. If you ready to start receving real payment on your site, kindly uncheck this.
* __Test Public Key__ - Enter your Test Public Key here.
* __Test Secret Key__ - Enter your Test Secret Key here.
* __Live Public Key__ - Enter your Live Public Key here.
* __Live Secret Key__ - Enter your Live Secret Key here.
* __Modal Title__ - Enter the text to be displayed as the title of the payment modal.
* __Modal Description__ - Enter the text to be displayed as a short modal description.
* __Checkout Logo__ - Select an image to be shown on the Flutterwave Checkout modal window. Recommended minimum size is 128x128px. Leave blank to disable the image.
* Click on __Save Changes__ for the changes you made to be effected.
