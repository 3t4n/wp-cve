Create sexy flipping cards!

== Description ==

Create beautiful grids with flipping cards (image, text and link). [Here an example](https://www.info-d-74.com/references/)
And there is a Pro version on my shop. [See more here](https://www.info-d-74.com/en/flipping-cards-pro-plugin-wordpress-demos/)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Flipping Cards menu to create your grid
