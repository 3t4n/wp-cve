Move Category description under products for WooCommerce. Move part of the category description under productsonly the part contained in the shortcode will be moved.

Enclose the description you want to jump under the products inside this short code
[mcdusaltasotto]Description jump [/mcdusaltasotto]
Sposta la descrizione della categoria sotto i prodotti per WooCommerce. Sposta parte della descrizione della categoria sotto prodotti verrà spostata solo la parte contenuta nello shortcode.Racchiudere la descrizione che si desidera saltare sotto i prodotti all'interno di questo codice breve[mcdusaltasotto]Descrizione salto [/mcdusaltasotto]


as in the example:
Description in defoult position: Lorem Ipsum has been the industry's standard dummy text ever since the 
[mcdusaltasotto]Description under products: Lorem Ipsum is simply dummy text of the printing and typesetting industry. Lorem Ipsum has been the industry's standard dummy text ever since the [/mcdusaltasotto]


shortcode
[mcdusaltasotto] [/mcdusaltasotto]



Use the shortcode on the woocommerce categories and tags page

This plug-in will allow you to add text in key seo title and subtitles without disturbing the usability
