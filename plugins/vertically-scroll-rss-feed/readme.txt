== Description ==

This plugin scroll the RSS feed title vertically in the widget, admin can add/update the RSS link and style via widget management.

Check official website for live demo : [http://www.gopiplus.com/work/2010/07/18/vertically-scroll-rss-feed/](http://www.gopiplus.com/work/2010/07/18/vertically-scroll-rss-feed/)

* Live Demo : [Live demo](http://www.gopiplus.com/work/2010/07/18/vertically-scroll-rss-feed/)  
* More Information : [More Information](http://www.gopiplus.com/work/2010/07/18/vertically-scroll-rss-feed/)  
* User Comments : [User Comments](http://www.gopiplus.com/work/2010/07/18/vertically-scroll-rss-feed/)  

Vertically scroll rss feed wordpress plugin will create the scroll news in the website widget. We can provide the rss feed as a input for this scroll. That is, This plugin read the given rss feed and create the vertical scroll with rss feed title. We can use only one rss feed at a time.

**Features of this plugine** 

* Easy installation.
* Can add any RSS feed.
* Easy style-override system.
* Scrolling, So occupy less space.

What is rss? RSS (Really Simple Syndication) is a family of web feed formats used to publish frequently updated works—such as blog entries, news headlines.

**Plugin configuration**

* Drag and drop the widget.

* Add directly in the theme.
