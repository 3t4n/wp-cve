Add Metro style social share widget to your sidebar. 5 most popular social networks supported

== Description ==

This plugin ads **Metro style social share widget** to your sidebar.

**5 most popular** social networks supported.

Metro Share Widget is super easy to install and configure.

More [screenshots and live sample](http://www.wpmaniax.com/metro-share-widget/) you can see on plugins' home page

**Main Features:**

*   Easy to install and configure
*   Most popular social networks supported (Facebook, Twitter, Google Plus, LinkedIn, Reddit )
*   Fully responsive
*   Configure where to show (posts, pages, home)
*   Metro style elegant design

== Installation ==

1. Upload `metro-share-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings / Metro Share Widget to configure the plugin
