Delete all comments by selecting the comment status from your WordPress admin.

== Description ==

The Plugin Helps you in Deleting All the Comments by selecting the status (Approved,Pending,Spam and Trashed) of comments Site admin.


== Installation ==

Standard installation required.

1. After downloading the 'delete-all-comments-msbd.zip' file extract/uncompress it.
2. Upload the Downloaded uncompress package to the '/wp-content/plugins/' directory.
2. Activate the plugin using the Plugin Tab in your Wordpress Dashboard.
3. You will find 'Delete Comments' under the 'Settings' menu to delete comments.

== Frequently Asked Questions ==

= Where can I find the admin page of the plugin? =

A menu item (called "Delete Comments") appears after "Comments" menu item on the left sidebar in admin. Please let us know if you don't like a top label menu item for this plugin.

= Do I need a backup of my website to use this plugin? =

It's not mandatory task because we only remove the comments (with comment meta) that you select in the plugin setting page. But we recommend you to get a backup of your site before using Delete Comments By Status plugin, might cause an accidental effect on your site by this plugin. Hope this will go nicely as we have tested it on multiple site server.

= Does this plugin work with all wordpress versions? =

This version works with WordPress 3.0 and better. If you're using an older version, please check for the legacy releases.
