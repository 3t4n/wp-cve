Plug-in for embedding of reservation system reenio into the web presentation.

== Description ==

**System reenio is an on-line reservation system** and represents one of the most effective ways to make reservations and stay in contact with your customers.

Using this plugin, you can **embed reservation system anywhere into your web site**. Just enter your specific reenio identification key and set the parameters.

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/plugin-name directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the 'reenio' menu to configure the plugin.

== Frequently Asked Questions ==

= Is it possible to use this plugin without registering for reenio system? =

No, the plugin serves as a preview and embedding of the reservation system into the web, it is necessary to be a user of this system.
