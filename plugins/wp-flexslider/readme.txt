Simple, easy to use with default WordPress Uploader.

== Description ==

Adding Flexslider using native WordPress Gallery. No extra post type, no complex shortcode like other plugins, easy to use.

- Allow changing slider settings such as effect, speed...
- Compatible to Jetpack Tiled Galleries.

== Frequently Asked Questions ==

= Is this plugin compatible to Jetpack Tiled Galleries? =

Yes, it is.

= Is there any way to change slider speed, effect or hiding slider natigation? =

Yes, Please go to Settings -> WP Flexslider and change as you wished.
