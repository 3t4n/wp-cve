== Description ==

>[Demo Link](http://bit.ly/2qgNLWQ "Demo Link") | [Ezusy](https://www.ezusy.com "Ezusy")

By default setting, WooCommerce only shows the main variation image if you choose a product variation as drop-down product attribute fields.

Ezusy plugin can help you to optimize your WooCommerce product images and boost your sales. You can generate image swatches to display the available product variable attributes like colors, sizes, styles, etc. You can replace the drop-down fields with image swatches. Besides, it makes your site becomes attractive. It is simple for your customers to select products. 

Ezusy plugin is Free and easy to use. It is a part of Ezusy app, which allows you to import AliExpress products to your WordPress WooCommerce with a few clicks. You do not have to order on AliExpress manually; Ezusy can do it for you. Are you interested in it? [Click here](https://www.ezusy.com "Click here") to get more information.

Main features:
 - Showcase product variation images
 - Control image size with zoom option

== Installation ==

1. Upload `ezusy.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
