== Description ==

Timeline Blocks for Gutenberg is a Gutenberg add-on with many additional options for post showcases with the vertical Timeline. It is a standalone Gutenberg solution to give wow design possibilities for your blog posts with a clean design and fully responsive layout. You can say that Timeline Blocks will be your ultimate history showcase for the new Gutenberg Editor.
 
These premade blocks help to build a perfect website in no time. It's time-saving and cost-saving when editing any blocks. Get a wonderful experience on the website building with Timeline Blocks now.

### Where are the Timeline Blocks for Gutenberg Most Useful?
 
Below is one small list of which area you can start using the Timeline Blocks for Gutenberg plugin.
 
* Showcase your Posts with a Timeline

== Installation ==

<strong>From Your WordPress Dashboard</strong>
 
1. Go to Plugins > <strong>Add New</strong>
2. Search for <strong>Timeline Blocks for Gutenberg</strong>
3. Click on <strong>Install Now</strong> Button
4. Click on <strong>Activate Now</strong> After Installed in the Backend 
 
From WordPress.org 
 
1. Download <strong>[Timeline Blocks for Gutenberg](https://downloads.wordpress.org/plugin/timeline-blocks.zip)</strong>
2. Upload the <strong>'timeline-blocks’</strong> folder to the /wp-content/plugins/ directory
3. Activate <strong>Timeline Blocks for Gutenberg</strong> plugin from your plugins page
