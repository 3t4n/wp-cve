easy ads.txt integration for your WordPress

== Description ==
Generates an ads.txt with your custom content. Provides a nice and easy option page for entering text.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the `Plugins` screen in WordPress
3. navigate to yoursite.com`/wp-admin/options-general.php?page=ads.txt`
4. follow instructions on the page and save your ads.txt there
