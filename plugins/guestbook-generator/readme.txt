Instantly generates a guestbook for Wordpress blogs based on the active theme.

== Description ==

Instantly generates a guestbook for Wordpress blogs based on the active theme. Once activated, click on Options > Guestbook Generator to create the guestbook.

== Installation ==

1. Download the plugin [http://www.alleba.com/blog/wp-downloads/guestbook-generator.zip here]
1. Extract and upload the contents of the archive to 'yourserver.com/wp-content/plugins/guestbook-generator/'
1. Login to your Wordpress admin panel and browse to the Plugins section.
1. Activate the Guestbook Generator plugin.
1. Go to Options > Guestbook Generator to create your guestbook.

== Issues and Warnings ==

The latest version of Guestbook Generators work only with Wordpress 2.1 and above.  If you are using a previous version (2.0), use Guestbook Generator v0.7 instead.
