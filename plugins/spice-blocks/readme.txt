==== Description ====

Spice Blocks Plugin is a block plugin that is compatible with all WordPress themes. In plugin block controls are given, that's help to develop a beautiful WordPress theme.

==== Installation ====

1. Upload spice-blocks.zip to the /wp-content/plugins/ directory and extract.
2. Activate the plugin through the Plugins > Installed Plugins menu in WordPress dashboard.
3. Done!

==== Key Features ====

* Heading
* Editor
* Divider
* Button
* Spacer
* Icon
* Section
* Column
* Image
* Blockquote
* Call To Action
* Timeline
* Accordion
* Icon List
* Image Comparison
* Image Gallery
* Image Accordion
* Progress Bar
* Service Box
* Social Icons
