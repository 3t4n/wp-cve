== Installation ==

<h3>From your WordPress dashboard</h3>

<ol>
<li>Visit ‘Plugins > Add New’</li>
<li>Search for ‘c4d woo wishlist'</li>
<li>Activate plugin from your Plugins page.</li>
</ol>
 

<h3>From WordPress.org</h3>

<ol>
<li>Download C4D Woo Wishlist.</li>
<li>Upload the ‘c4d-woo-wishlist' directory to your ‘/wp-content/plugins/’ directory, using your favorite method (ftp, sftp, scp, etc…)</li>
<li>Activate plugin from your Plugins page.</li>
</ol>

<h3>Shortcode</h3>

[c4d-woo-wishlist-cart icon="fa fa-heart-o" label=""]

[c4d-woo-wishlist-button icon="fa fa-heart-o" label=""]
