== Description ==

SiteSuperCharger helps your website look as attractive as possible to search engines helping you to acheive the highest search engine rankings.

Contact Marketing Heroes (https://www.mheroes.com) to get started!

== Installation ==

Install SiteSuperCharger via the plugin directory, or by uploading the files manually to your server. After activating SiteSuperCharger, you'll need to enter your GUID at Settings -> SiteSuperCharger

For info about acquiring a GUID for SiteSuperCharger visit https://www.mheroes.com.


== Frequently Asked Questions ==
