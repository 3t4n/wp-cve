Build your Classified Ads Directory Portal based on Wp Directory Kit and Elementor Plugin

== Description ==

Classified Ads Plugin is Very easy to use and install, plugin will allow you to manage Classified Ads Listings, Categories, Fields from your admin dashboard.

Can be used as Business Directory Plugin to create Combined Classified ads website with multiple type of ads like: job directory, booking website, local business directory and service directory.

This is addon plugin for WP Directory Kit Plugin, so will configure fields, categories and import demo listings for Classified Ads Portal into [WP Directory Kit](https://wpdirectorykit.com/).

Almost everything possible to design visually in Elementor, including front pages, resutls pages, listing preview pages etc.

Visual designing support for search form and result item/cards.

== üèÜ Classified Ads Control ==
This classified ads WordPress plugin gives you a choice between automatic and manual approval of your listings. If you have too many users, automatic approval is best. On the other hand, manually approving listings help screen ads in detail before posting them.
