Tiny Google Analytics adds Google Analytics Tracking Code using an optimzed code, useful for MultiSite.(Low resource usage).

== Description ==

Inserts tracking code only, view your stats in your Google account.

**Features**

* Drop-dead simple and easy to use
* Uses latest version of Google Analytics Tracking Code
* Inserts your tracking code on all theme pages
* Ultra Lightweight, and born of simplicity, no frills

== Installation ==

1. Unzip, Upload, activate
2. Visit the "Tiny Google Analytics" options page, enter your Google Analaytics Property ID, and done.

For more information, visit the [Tiny Goolgle Analytics Plugin Homepage](http://todoestamal.com.ar/tiny-google-analytics-plugin-for-wordpress/).
