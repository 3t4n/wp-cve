== Description ==
* WordPress3.5のため、ップロードするファイルの保存場所とファイルへの完全な URL パスを表示させる。
* メディアのデフォルト状態に戻したい場合、プラグインを無効にして、アップロードするファイルの保存場所とファイルへの完全な URL パスのデータを削除する.
