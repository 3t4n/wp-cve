Search and Replace is used for change text in WordPress with going to edit page or database.

== Description ==
Search and Replace is used for change text in WordPress after page load, Like if you wanna change some text in WordPress website that is very hard for you to change just install the Search and Replace Text plugin search what text you want to find after replace with you can change or remove word or paragraph from whole website or specific page.

== Installation ==
1. Upload the plugin files to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Go the Tools -> Search and Replace Text screen to configure the plugin

== Frequently Asked Questions ==
= Will this plugin replace text in database =
No, this plugin just replace the text on screen load, It will not affect any database table

= What if I want to remove the text filter in plugin =
No problem you just have to get to the plugin setting page \"Tools > Search and Replace Text\" And remove the filter you want to be removed.
