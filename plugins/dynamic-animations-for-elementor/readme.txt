Extension for Elementor, you can move widgets with amazing effects.

== Description ==

It's an Elementor addons, so you must install Elementor FREE before activate the plugin.
For every widget you can set many advanced and incredible effects and animations.

[View FULL FEATURES, DEMO and MORE](https://www.dynamic.ooo/widget/dynamic-animations/)

We also suggest this amazing plugin: [Dynamic Visibility for Elementor](https://wordpress.org/plugins/dynamic-visibility-for-elementor/)

= How it works =

Simply open a page in Elementor mode. Select your widget, go to Advanced tab.
You'll find a new “Dynamic” configuration. Open it, enable Animations and set as you prefer.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/dynamic-animations-for-elementor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. There is nothing to configure. It's necessary have installed and activated Elementor FREE
