== Description ==

DevVn Float Left Right Ads help you add ads to left and right fixed position

== Installation ==

1. Upload the entire `devvn-float-left-right-ads` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

Go to Setting -> Float Left Right Ads config this plugin

For basic usage, you can also have a look at the [plugin homepage](http://levantoan.com/devvn-float-left-right-ads/).
