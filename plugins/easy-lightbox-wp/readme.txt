== Installation ==

Installing this plugin as regular WordPress plugin.

After install, you are ready to add Lightbox in images. By default, the plugin auto assign WordPress regular gallery with Lightbox. But, you have to use link to attribute as file. Example this gallery shortcode will auto add Lightbox: <code>[gallery link="file" columns="5" ids="357,356,354,353,352"]</code>. 

If you want to add a lightbox in single image, you just have to use <strong>litebox</strong> in your link. Example code: <code><a href="BIG_IMAGE_LINK" class="litebox"><img src="THUMBNAIL_IMAGE_LINK" alt="" /></a></code>.

If you want to add lightbox image image set, just use <code>data-litebox-group="galone"</code> in your link. Example code: 

<code>
<a href="BIG_IMAGE_LINK" data-litebox-group="galone" class="litebox"><img src="THUMBNAIL_IMAGE_LINK" alt="" /></a>
<a href="BIG_IMAGE_LINK" data-litebox-group="galone" class="litebox"><img src="THUMBNAIL_IMAGE_LINK" alt="" /></a>
<a href="BIG_IMAGE_LINK" data-litebox-group="galone" class="litebox"><img src="THUMBNAIL_IMAGE_LINK" alt="" /></a>
<a href="BIG_IMAGE_LINK" data-litebox-group="galone" class="litebox"><img src="THUMBNAIL_IMAGE_LINK" alt="" /></a>
<a href="BIG_IMAGE_LINK" data-litebox-group="galone" class="litebox"><img src="THUMBNAIL_IMAGE_LINK" alt="" /></a>
</code>

If you want to add video Lightbox, it is same like single Lightbox. Example code: <code><a href="YOUTUBE_VIDEO_LINK" class="litebox">Open Youtube Video</a></code>

Currently this plugin support 4 video sharing websites. 

<ol>
	<li>Youtube</li>
	<li>Vimeo</li>
	<li>Dailymotion</li>
	<li>Kickstarter</li>
</ol>
