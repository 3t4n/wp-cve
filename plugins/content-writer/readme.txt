== Installation ==

1. Download plugin to your local drive from WordPress.org
2. Open your WordPress blog and click Plugins => Add New
3. Click Upload Plugin and select the content-writer.zip file
4. Click Activate Plugin
5. Enter Your SteadyContent.com Email and Password and click Connect!
6. If you do not have an account, click the link below Connect and follow the instructions.
