Integration of the Wiredminds LeadLab trackingcode.

== Description ==
This Plugin will provide a Tracking-Code for LeadLab (powered by WiredMinds). The code will be placed in front of the closed body tag to each page rendered by WordPress.
The User has only to use his customer number in the plugin configuration.

== Installation ==
1. Upload \"leadlab.php\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Go to Settings.
4. Go to Wiredminds LeadLab. 
5. Enter your Wiredminds LeadLab custom number.
6. Save your entry.

== Frequently Asked Questions ==
Where can i find my customernumber?
You can find your "Customernumber" in LeadLab under configuraion and trackingcode.
