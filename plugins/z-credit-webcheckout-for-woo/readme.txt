== Description ==
This plugin allows you to accept credit cards from you woocommerce site.
## תוסף הסליקה מאפשר:
* סליקת כל כרטיסי האשראי בחנות שלכם
* סליקה באמצעות עמוד חיצוני או באמצעות עמוד פנימי באתר
* עמוד סליקה שתומך בעברית, אנגלית ורוסית לפי שפת האתר
* משלוח חשבונית/קבלה אוטומטית בסיום העסקה
* אפשרות לחיוב דחוי, ושינוי סכום ההחיוב לפני הוצאת המשלוח
* תמיכה בש"ח ובדולר
* אפשרות לזיכוי ישירות מתוך ההזמנה
* תמיכה מובנית ב GooglePay , ApplePay וביט

להפעלת התוסף נא התקשרו לשירות הלקוחות: 077-3233190

After installing the plugin, you will need a key to use it.
