This Custom Shotcode Creator plugin allows you to quickly define custom shortcodes via admin dashboard without any hassle.

== Description ==

1 .This plugin allows you to easily replicate redudant information/data on different pages by defining them as shortcodes.
Requiring no access for core php files, the plugin is secured and easy to use.

2 . You can write php code inside the editor and generate shortcodes

== Installation ==

1. Put the Custom ShortCode Creator to plugin folder "/wp-content/plugins/"
2. Go into the WordPress admin pannel and activate the plugin
3. Enjoy
