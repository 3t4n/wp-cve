== Description ==

== Installation ==

= Minimum Requirements =

* WordPress 5.5 or greater
* WooCommerce 4.5 or greater
* PHP version 7.2 or greater

= How to Install plugin =

* Visit the WordPress Plugin Store
* Download and activate the Invoice123 plugin
* Provide the API key and configure the necessary settings
