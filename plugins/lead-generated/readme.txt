Save the leads to our lead management system CRM generated by Contact Form 7

== Description ==

This is the official plugin for Lead Generated. This plugin creates the connection between WordPress and your Lead Generated account. Form submissions are instantly available and ready to  leverage all the powerful features of our lead management system within the Lead Generated system. For more information on features, plans and more check out [leadgenerated.com](https://www.leadgenerated.com/).

== Installation ==

1. Upload `lead-generated` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
	And you are ready to generate the leads from CF7 form submissions.
3. Go to Settings -> Lead Generated Settings to add your api key.
4. After adding valid api key the 'Send to Company' field will be populated with companies from your Lead Generated account.
5. Now select the company you want to send the leads to in the Lead Generated account.
