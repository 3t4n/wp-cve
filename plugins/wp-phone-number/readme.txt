Makes sure telephone numbers are clickable on mobile devices

== Description ==

Your mobile visitors are already holding their phone it will allow them to call your phone number.

This plugin is based on the <a href="http://code.google.com/p/php-mobile-detect/" target="_blank">PHP Mobile Detect class</a>.



== Installation ==

1. Upload plugin folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're done!
