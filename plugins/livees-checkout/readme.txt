Livees Checkout permite recibir pagos electrónicos en Bolivia a través de todos los medios de pago electrónicos regulados por ASFI.

== Description ==

Livees Checkout® es la principal pasarela de pagos en Bolivia creado con el fin de ofrecer múltiples canales de cobro integrados, entre ellos tarjetas de crédito/débito Visa, Mastercard, billeteras electrónicas, banca móvil y QR Simple.

Entre los beneficios que incluye Livees Checkout están:
* Tus clientes pueden finalizar sus compras y pagar con la pasarela sin abandonar tu tienda online
* Programa 3D secure 2.0 incorporado
* Visa Secure® y Mastercard ID Check® incorporado
* Protección completa certificada con SSL, Geotrust® y SiteLock®
* Experiencia de Pago con 1 clic (ONE CLIC BUY®)
* Control anti fraude con triple capa de seguridad
* Abonos a cuenta en cualquier entidad bancaria de Bolivia
* Identificador automático de moneda Dolares/Bolivianos¹
* Identificador automático de idioma Español/Inglés¹
* Soporte extendido 24/7/365
* Facturación electrónica incorporada¹
* ¹Complementos adicionales a requerimiento

Para todos las tiendas online, se requiere una llave y token privadas. 

== Installation ==

Instale el Plugin Livees Checkout desde el panel de Plugins o descargando el Plugin en formato ZIP, instálelo, actívelo y luego ingrese su TOKEN y LLAVE. Si no tiene una cuenta visite: https://www.livees.net/liveescheckout
