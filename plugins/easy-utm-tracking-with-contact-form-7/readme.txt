== Description ==

Easy UTM Tracking with Contact Form 7 is a simple plugin that lets you track UTM parameters and referrer in your Contact Form 7 lead emails with just one-click installation. The plugin saves any UTM values coming in the URL and stores them for later retrieval, so users can browse your website normally without losing the UTM data. When a user submits any Contact Form 7, the stored UTM parameters and referrer data are automatically included in the email.

This plugin is a great tool for those who want to track and analyze their website's UTM data and referrer traffic sources in a convenient and effortless manner. 

**The following UTM tags are tracked:**

* utm_source 
* utm_medium 
* utm_term 
* utm_content 
* utm_campaign 
* Landing page URL
* Referrer (traffic source)

== Installation ==

You can install Easy UTM Tracking with Contact Form 7 using either of the following methods:

* Installation using WordPress

1. Log in to your WordPress admin panel. 
2. Navigate to the Plugins section.
3. Click on the "Add New" button.  
4. Search for 'Easy UTM tracking with Contact Form 7' in the search bar. 
5. Click the "Install Now" button. 
6. Activate the plugin once installation is complete.

* Installation by downloading the plugin *

1. Download the plugin from the WordPress repository.
2. Extract the plugin files into the `/wp-content/plugins/` directory either by using FTP or uploading the zip file directly from the Plugins page.
3. Navigate to the Plugins section in your WordPress admin panel.
4. Activate the plugin through the 'Plugins' screen in WordPress
5. You're ready to start tracking UTMs!

== Frequently Asked Questions ==

= What is UTM tracking? =
UTM tracking is the process of tracking user engagement on your website through the use of UTM (Urchin Tracking Module) parameters in URLs. These parameters help you to understand the effectiveness of your marketing campaigns by tracking the source, medium, campaign name, content, and term of your visitors.

= How do I use this plugin? =
Easy UTM Tracking with Contact Form 7 is a one-click installation plugin that automatically tracks and saves UTM parameters and referrer in Contact Form 7 lead emails. Simply install the plugin and start tracking UTMs effortlessly.
