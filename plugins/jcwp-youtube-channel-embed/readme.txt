== Description ==

This plugin embeds a custom channel to wordpress page or post

This plugin is ShortCode based. Shortcode can be generated by a click of a button from settings page. Shortcode format is shown below

[jcorg_youtube_channel mode="list|thumbnails" videoWidth="640" thumbnailWidth="240" showTitle="yes|no" maxResults="6" startIndex="1" orderBy="published|relavance|viewcount" filterKeyword="" channelName="" embedType="frame|object"]

== Installation ==

1. Upload unzipped plugin directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Plugin Options on left menu under Settings > JCWP Youtube embed.
