Make post, page or partial content visible to registered users only


== Description ==
Increase user registration on blog by making post or page available only for registered user.
This plugin allows to make post, page or partial content visible to registered users only. You can hide partial  content using shortcode `[RegUserOnly][/RegUserOnly]`. Or you can hide complete post or page by selecting Visibility to `Registered Users` in post/page editor.

= Plugin Features =

* Easy installation and integration
* Custom tinymce shortcode button
* Cross Browser capability
* Translation ready

== Installation ==

= Using the Plugin Manager =

1. Click Plugins
2. Click Add New
3. Search for `Only for registered users`
4. Click Install
5. Click Install Now
6. Click Activate Plugin

= Manually =

1. Upload `only-for-registered-users` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
