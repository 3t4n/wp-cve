== Description ==

All Embed Addon For Elementor is a collection of powerful audio, video, social media widgets that works perfectly with Elementor page builder. It has 20 plus widgets so you can easy-to-use elements those were designed to make your next WordPress page.
Embed anything you want, 

== Installation ==

Download the latest version of All Embed Addons For Elementor (.zip file)
In your Admin, go to menu Plugins > Add New
Select “Upload Plugin”
After Install The Plugin, Just Active It And Use it Smoothly


Note: This plugin works with Elementor. Make sure you have [Elementor](https://wordpress.org/plugins/elementor/) installed.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress.
