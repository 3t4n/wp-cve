Loading paged content with Ajax.

== Description ==
Loading paged content with Ajax. Works with standard WordPress pagination, WooCommerce pagination, plugins.
[Plugin demo](https://processby.com/blog/)

= Features =

* Ajax pagination;
* "Load more" button;
* "Load more" button and pagination;
* Infinite Scroll;
* Customisable styles.


== Screenshots ==

1. WP Ajax Pagination settings


== Installation ==

1. Unzip the downloaded zip file.
1. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
1. Activate `WP Ajax Pagination` from Plugins page
