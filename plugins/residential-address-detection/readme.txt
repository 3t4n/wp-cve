Real-time identification of residential and commercial address types.
 
== Description ==

An add-on plugin developed by Eniture Technology. The plugin works with Eniture Technology's Small Package Quotes and LTL Freight Quotes plugins. It identifies the address type of the ship-to address provided by website visitors. When the address type is residential, the residential delivery fee is included in the shipping rate estimate returned by the carrier.

**Key Features**

* When enabled, automatically identifies residential and commercial addresses.

**Requirements**

* WooCommerce 5.7 or newer.
* The installation of at least one of Eniture Technologyâ€™s Small Package Quotes or LTL Freight Quotes plugins.
