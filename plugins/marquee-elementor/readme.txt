= About the plugin = 

it is a free plugin to make marquee in elementor in an easy way.

= Features = 

1- write any text you want to make a marquee.
2- use dynamic properties in elementor instead of static text.
3- Use your posts as marquee.
4- Select posts by category.
5- Select number of posts.
6- Change the order of posts as you want.
7- you can write simple html tags in your text.
8- you can link your marquee to any link you want.
9- you can wrap your text in many tags available.
10- you can style your text as you want.

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 5.8 or greater
* Elementor Free Plugin

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Pages Or Posts
4. Press the 'Edit with Elementor' button.
5. search for marquee and enjoy.
