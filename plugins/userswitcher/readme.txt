A helper tool to help you switch between user account without logging in and out.

== Description ==
A simple tool that allows you to switch between user accounts without the hassle of logging in/out. It allows you to view, review and do what yours does to your site.
= Features = 
* Switch between user account and to guest user.
* Inherit all capabilities of the selected account, as if you are that very same user.

* Helpful for developers, designers, and site owners.

== Installation ==
1. Upload `userSwitcher` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click `User Switcher` menu at the top (admin bar). If admin bar is disabled, you will see a small box at the bottom left corner of your page.
