== Description == 

Before After Image Comparison Slider for WPBakery is an image comparison slider plugin for WPBakery Page Builder. This plugin allows you to create the effect for comparing two before and after images.

This plugin let you compare two before and after images with WPBakery Front-end Drag and Drop Builder. 

You can add unlimited numbers of slider with this plugin. This plugin also works on the frontend editing mode and let you see the preview of how the slider looks on the website.

== Features ==

   * **Responsive and Touch Friendly** -– It is fully responsive on any browser and Touch Friendly for mobile and touch devices.

   * **Image Dimensions** -- You can use any image sizes used in your theme.

   * **Custom CSS** -- Option to Override Default CSS and add your own style with Custom CSS

   * **Front-end Editing** -- You can add new slider or edit existing sliders from the WPBakery Frontend Editing Mode

   * **WebP Image Support** -- This plugin supports **WebP Images**. It is the only plugin which supports **WebP Images**.  You can now use **WebP Images** to compare with this plugin

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin folder to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Visual Composer Editor, and looking for the addon under "Web Builders Elements" category named as "Before After Image Comparison Slider" and select it.
4. Upload Two Images and set your desired image sizes and configure other options as you do on other elements.
