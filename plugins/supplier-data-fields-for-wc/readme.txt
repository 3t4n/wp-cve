Adds supplier fields to the product data section of each product.

== Description ==

Adds 10 custom supplier fields to the General product data section of each product. Fields are Supplier Cost, Supplier Minimum Price, Supplier SKU, Supplier Description, Supplier Name, Supplier Address, Supplier Phone, Supplier Email, Supplier Location and Supplier Notes.

== Frequently Asked Questions ==

= I need a another custom field added =

* If there is another field that you think would be beneficial (and is related to suppliers), then send a support request and I will consider adding it in.
