== Description ==

**Content Snippet Manager plugin** allows you to create and manage unlimited numbers of HTML and WordPress shortcodes in your WordPress content
You will be able to choose the position and the destination of each snippet.

For example, you may need to insert an Amazon banner or a subscription newsletter shortcode at the middle or at the end of all posts, you can do it using our plugin, without being forced to edit manually each WordPress post.

**Important** Javascript code, conversion script or custom css codes are not allowed on the free version, they will be removed from the WordPress editor.

To insert an header, body, footer or conversion snippet, you can configure one of available shortcode.
[https://demo.themeinprogress.com/content-snippet-manager-pro/conversion-shortcodes/](https://demo.themeinprogress.com/content-snippet-manager-pro/conversion-shortcodes)

**Premium features.**

> <strong>Header snippets</strong><br>
> You can insert your own Javascript and CSS code on the wp_head hook.<br><br>

> <strong>Body snippets</strong><br>
> You can insert your own Javascript and CSS code on the wp_body hook.<br><br>

> <strong>Footer snippets</strong><br>
> You can insert your own Javascript and CSS code on the wp_footer hook.<br><br>

> <strong>Conversion snippets</strong><br>
> You can insert your own Javascript and CSS code inside the final thank you page of WooCommerce.<br><br>

> <strong>Dynamic conversion values</strong><br>
> The dynamic conversion values allow you to use specific parameters of a WooCommerce order inside your snippet, when a user has been redirected to the final thank you page.<br><br>

> <strong>Custom post type support</strong><br>
> You can include a snippet inside all available custom post types available on WordPress, instead of only the WordPress posts, pages and WooCommerce products.<br><br>

> <strong>Custom taxonomies support</strong><br>
> You can include a snippet inside all available custom taxonomies available on WordPress, instead of only the WordPress post categories, tags and WooCommerce categories.<br><br>

> <strong>Device selection</strong><br>
> Select one or more devices where you can load the content snippet.<br><br>

> <strong>User Role</strong><br>
> If needed, you can hide each snippet for specific user roles, like the administrator.<br><br>

You can get the premium version, from the following link:
[https://www.themeinprogress.com/content-snippet-manager/](https://www.themeinprogress.com/content-snippet-manager)

= Installation =

Once you have installed the plugin, you just need to activate the plugin in order to enable it.

= Configuration =

Content Snippet Manager will add a new admin page, where you can manage all snippets.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/content-snippet-manager' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
