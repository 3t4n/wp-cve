This plugin lets you insert HTML content exported from InDesign using in5 into a Wordpress post or page.
 
== Description ==

in5 lets you export interactive HTML from Adobe InDesign.

The Ajar in5 Embed plugin lets you insert your in5 HTML right inside a post with no coding. 
 
== Installation ==
 
1. Use Plugins > Add New to Upload the .zip directory or upload `ajar-in5-embed.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

Where do I get in5?

in5 is available at [http://in5.ajar.pro](http://in5.ajar.pro "InDesign to HTML5").
 
Do I zip the files or the folder?

Either one. The plugin will figure it out for you.

Can I upload a .hpub file?

Yes, a .hpub file will work as well as a .zip.

Does the in5 Embed plugin work with the Gutenberg text editor in Wordpress?

Yes, the latest version does.

== Screenshots ==
1. The Add in5 Embed button appears in the Post Editor.
2. Easily click to upload a zip file (or choose a previously uploaded package).
3. The plugin automatically populates the title, width, and height from the in5 content. You can apply addition options from here as well.
4. The content is automatically inserted into your post.
5. You can also use the in5 embed plugin with the Gutenberg editor.
6. Filter by in5 to easily find it in the Gutenberg editor.
7. Click to embed from the Gutenberg editor block.
8. As a block, you can as view as HTML, or preview the content itself.
