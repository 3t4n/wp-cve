BDroppy products management

== Description ==

BDroppy is auto importing product in you shop and auto manage stack from bdroppy server.

requirements:

1. Bdroppy account
2. WooCommerce

== Installation ==

How to install


1. create account on bdroppy.
2. create your catalog on bdroppy products.
3. install plugin.
4. setup api.
5. select catalog.
6. enjoy it.
