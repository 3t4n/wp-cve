== Description ==
FX Live Prices WordPress Plugin provides live forex rates and indicators and cross-exchange rates. Prices quote update frequency is 1-5 seconds. It is a simple solution to embed forex widgets and all tools are within the WordPress dashboard.

=== EXTERNAL SERVICES USED IN PLUGIN ===
The plugin has been providen by [FXpricing](https://fxpricing.com), and using websocket for live quotes from [FCS API](https://fcsapi.com). All forex data are display in HTML &lt;iframe&gt; source from third party service [FXpricing.com](https://fxpricing.com).

== Installation ==
<ol>
<li>First download plugin zip file.</li>
<li>In your WordPress Admin Menu, go to "Plugins > Add" New.</li>
<li>Click on "Upload Plugin" button, found on top left corner of page.</li>
<li>Click on Browse (1), Select the .zip file of your plugin in your computer, and click Install Now (2) button.</li>
<li>At this point, the plugin is installed. You can click on Activate Plugin link to work with it.</li>
<li>Or Activate the plugin through the 'Plugins' menu in WordPress</li>
</ol>



== Use a Plugin ==
<ol>
<li>In your WordPress Admin Menu, go to "Forex Widget" -> Add New".</li>
<li>Select your widget format from dropdown.</li>
<li>Update your custom settings.</li>
<li>Publish your widget.</li>
<li>Copy shortcode and paste it where the widget is to be displayed in Post or Pages.</li>
</ol>

== Frequently Asked Questions ==
= How many FX currencies support? =
2000+ currency are supported, but top 400 FX currency has live exchange rates with 1 seconds update frequency.

= How many FX currencies we can show on-site? =
There are multiple widgets in a plugin, if any widgets support multiple currencies then there is no limit. 

= From where you get live rates? =
We get live rates from third-party fcsapi.com API and show in &lt;iframe&gt; from source fxpricing.com in WP site.

= Is it free to use? =
Yes.

= Why live rates are not updating? =
Check if today is not weekends or a holiday because the market is closed. else you can contact us
