== Description ==

<p>Simple, <strong>yet effective</strong> plugin that once activated, it will <strong>eliminate every CSS version tag</strong> of WordPress plugins, core & themes.</p>
<p><strong>Bonus</strong>: your website will receive a small boost of SEO, you will experience no CSS caching issues anymore and hackers wont be able to recognize your version numbers.</p>
<p>Try its demo <strong><a target="_blank" href="http://apps.8web.gr/demos/wordpress-remove-versions-from-code-plugin/">here</a></strong></p>

== Installation ==

<p>All you have to do, is to upload our plugin within your Wordpress Plugins menu and activate it (in case of manual install).</p>
<p>Ta daa, magic happens :) All versions are removed from the source code.</p>
<p>In case you face any issues, feel free to raise a support ticket <strong><a target="_blank" href="https://wordpress.org/support/plugin/remove-code-version-tags">here</a></strong></p>
