== Description ==

Its a very simple plugin to disable the password reset in the wordpress.

If a user attempts to reset their password they will receive the error message

**Password reset is not allowed for this user**

Kindly let us know your feedback or comments to add more features in this plugin.

== Installation ==

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **Simply Disable Password Reset** in the search box and click on search button.
3. Find Simply Disable Password Reset plugin.
4. Then click on Install Now after that activate the plugin.
