== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to _Plugins &rarr; Add New.
2. Type "Plum" into the Search and hit Enter.
3. Locate the Plum plugin in the list of search results and click **Install Now**.
4. Once installed, click the Activate link.
5. You will be redirected to the Plum Getting Started page.

It can also be installed manually.

1. Download the Plum plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate Plum in the list and click the *Activate* link.
5. You will be redirected to the Plum Getting Started page.
