== Description ==
"Tilted Tag Cloud Widget" is a free plugin for Wordpress, developed by the Whiletrue.it staff to generate a cloud with the website's most used tags.

The words inside the cloud are overlapping and tilted, creating a cluster. 

= Options =

The following options are customizable:

* number of tags shown
* link to tag page
* tag color (random if none inserted)
* smallest and biggest font size
* horizontal and vertical spread

This plugin displays the most used tags in a sidebar widget.

== Installation ==
1. Upload the `tilted-tag-cloud-widget` directory into the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Inside the `Themes->Widget` menu, place the Tilted Tag Cloud Widget inside a sidebar, customize the settings and save
4. Enjoy!
