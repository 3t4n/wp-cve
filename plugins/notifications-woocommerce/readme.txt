Notificações para WooCommerce é um plugin de código aberto para notificações em tempo real no WhatsApp integrado ao WooCommerce.

== Description ==

= O PLUGIN DE NOTIFICAÇÕES NO WHATSAPP MAIS SIMPLES DE USAR =

Já imaginou sua loja integrada ao WhatsApp, enviando notificações em tempo real para seus clientes?

Pois é, nós imaginamos!

Notificações para WooCommerce é um plugin que irá permitir enviar notificações em tempo real para o WhatsApp do cliente sempre que o mesmo fizer uma compra na loja ou quando o status do pedido for alterado.

https://youtu.be/JU4LjDoUt8c

= FUNCIONALIDADES =

- Envio de notificação sempre que uma nova venda é feita
- Envio de notificação sempre que o status de um pedido é alterado
- Personalização de todas as mensagens
- Diversos shortcodes
- Integrado com o [Plugin Rifa](https://dropestore.com/plugin-rifa/)

== Installation ==

= Requisitos mínimos =

* WordPress 5.0 ou superior
* PHP versão 7.4 ou superior
* MySQL versão 5.0 oou superior

= Recomendamos que seu host suporte: =

* PHP versão 7.4 ou superior
* MySQL versão 5.6 ou superior
* Limite de memória do WordPress de 64 MB ou superior (128 MB ou superior é o recomendado)

= Instalação =

1. Instale usando o instalador de plugins do WordPress ou extraia o arquivo zip e solte o conteúdo no diretório 'wp-content/plugins/' da sua instalação do WordPress.
2. Ative o plugin através do menu 'Plugins' no WordPress.
3. Vá para WC Notificações.
4. Insira o token do dispositivo e o número que irá fazer o disparo das mensagens.
5. Agora veja a mágica acontecer sempre que um cliente efetuar uma compra no site ou quando o status de algum pedido for alterado.

== Frequently Asked Questions ==

= Como eu instalo o plugin? =

Para instalar a versão gratuita do Notificações para WooCommerce, siga as etapas abaixo:
No painel do WordPress -> Vá para Plugins -> Clique em 'Adicionar novo' -> No campo Pesquisar, digite Notificações para WooCommerce e selecione o plugin.
Clique em Instalar -> Após a instalação, clique em Ativar.

= É necessário contratar algum serviço para o plugin funcionar? =

Sim. O plugin só é funcional com uma assinatura ativa na [DW-API](https://dw-api.com/).

= O plugin só funciona com WooCommerce? =

Sim. Para que o plugin Notificações para WooCommerce funcione corretamente você precisa ter o plugin WooCommerce instalado em seu WordPress.

= Atualizações  =

As atualizações automáticas devem funcionar sem problemas, mas ainda recomendamos que você faça backup do seu site.
