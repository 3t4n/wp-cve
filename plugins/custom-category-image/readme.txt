== Description ==

Are you looking for uploading image to a category ? Yes, Custom Category Image plugin exactly does that. 
Just install it and it will add an image field to your categories.
use following function to get your category image in front-end 'MGC_Custom_Category_Image::get_category_image($term_id, $size);'

You can retrieve your desired image size also by passing size to this function.

Very easy to use. For any further customization feel free to raise a ticket on our [website](http://www.mingocommerce.com/contact-us/)

If our plugin helps you, please leave a review as that would be an ecouragement for us.

For any issue triggered while using our plugin please feel free to use support section. We will resolve that as soon as possible.

== Installation ==

Manual

1. Upload the plugin folder to the '/wp-content/plugins/' directory .

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Find image upload field to each category

Automatic

1. Go to plugin list from 'dashboard'. Search our plugin and click to install

2. Activate it.

3. Find image upload field to each category
