== Description ==

Repair table fixes any corrupted table .
If you need to repair your database then install this plugin.This plugin is very easy to use and that is why it is named "Simple Database Repair" . 

== Installation ==

1. Upload the 'simple-database-repair' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Plugin can be used from "Simple Database Repair" menu in left panel.
