== Installation ==
1. Upload the folder "wp-easy-replace-howdy" to "/wp-content/plugins/" path.
2. Activate the plugin through the "Plugins" menu in WordPress backend
3. After activating the plugin, howdy will automatically be replaced with "welcome {username} Message.
4. Enjoy!
