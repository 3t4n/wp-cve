== Description ==

Displays the metadata information in the Media Library. Regenerates the metadata and thumbnails even if they exist. This plugin is really useful when the metadata is broken. This is an extremely light plugin.

I am using this tool for debugging many installs and decided to make it available here. Try it, discuss it. Remarks are very welcome :)

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Check your Media Library in List Mode.
