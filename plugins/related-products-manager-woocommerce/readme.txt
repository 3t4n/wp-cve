== Description ==

Related Products Manager for WooCommerce plugin lets you customize the associated products in WooCommerce.

You can change displayed WooCommerce related products number, columns, order, relate by tag, category, product attribute. You may need to clear all product transients to immediately see the front-end results after changing the plugin's settings. Alternatively, you can update each product individually to remove its transients.

### Key Features of Related Products Manager for WooCommerce

There are the following reasons to choose Related Products Manager for WooCommerce.
 
**General Options**

<strong>1. Related Products Number</strong>
 
You can show the total number of displayed related products.
 
<strong>2. Related Products Columns</strong>
 
You can display the number of columns.

**Order Options**

<strong>1. Order By</strong>
 
You can order the related products via the following options.
 
* Random
* Date
* Title
* ID
* Modified
* Menu Order
* Price

<strong>2. Order</strong>
 
It ignores if order by "Random" is selected in "Order by." There are two possible values.
 
* Ascending 
* Descending

**Relate Options**

<strong>1. Relate By Category</strong>

It is related to the current product category.
 
<strong>2. Relate By Tag</strong>
 
It is related to the current product's tag.
 
<strong>3. Relate By Product Attribute</strong>
 
It is related to the current product's attributes.

<strong>4. Attribute Type</strong>
 
If using the "Global attribute," enter the attribute's slug in "Attribute name." Here are two options you can set.
 
* Global Attribute
* Local Attribute

<strong>5. Categories and Tags</strong>
 
If enabled, it will override relations by categories and tags when relating manually or by-product attributes. Here are three values.
 
* Override
* Do Not Override (AND)
* Do Not Override (OR)

**Hide Option**
 
<strong>1. Hide Related Products</strong>
 
You can hide related products globally.

**Exclude Options**
 
<strong>1. Exclude from Related</strong>
 
To enable or disable exclude Option via a checkbox.
 
<strong>2. Categories</strong>
 
Add categories of products that, if you don't want to show in related products.
 
<strong>3. Tags</strong>
 
Add tags of products that, if you don't want to show in related products.
