== Description ==

Widget that displays the chess puzzle from [shredderchess.com](https://www.shredderchess.com/daily-chess-puzzle.html).

= Example =

An [example at my local chess club Pegasus](https://svpegasus.nl) is available in the footer of the website..

= Compatibility =

This plugin is compatible with [ClassicPress](https://www.classicpress.net).

= Contributions =

This plugin is also available in [Codeberg](https://codeberg.org/cyclotouriste/wp-shredderchess).


== Installation ==

1. Use the plugin installer built into WordPress to search for the plugin. WordPress will then download and install it for you.
2. Activate the plugin.
3. Go to Appearance > Widgets, and drag the widget to the desired widget area.
4. Enjoy.
