Useful post types and customizer options.

== Description ==

This plugin ads some extra options to your Customizer (like social icons),  registers taxonomies and custom posts types for:
- Portfolio items
- Testimonials
- FAQ
- Team Members

This plugin doesn't change how new post types are displayed in your theme. e.g You'll need to add templates for archive-portfolio.php and single-portfolio.php if you want to customize the display of portfolio items.
In the theme integrated with smartlib library you don't need do anything

== Installation ==

1. Upload `Smartlib Tools` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go te the admin area options page (Settings-> Admin Area Options)
