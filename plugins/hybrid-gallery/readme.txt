== Description ==

<h4 id="item-description__hybrid-gallery-visual-premium-wordpress-gallery-plugin">Hybrid Gallery | Responsive WordPress Visual Gallery Plugin</h4>
<p>Hybrid Gallery is the 1st <a href="https://hybrid-gallery.com"><strong>WordPress Gallery Plugin</strong></a> with Visual Shortcode Editor, which allows you to create and edit galleries with just one click. It’s the easiest way to set awesome galleries with different layouts, skins, effect and much more…</p>

https://www.youtube.com/watch?v=w1jOO70xVHE

<h4 id="item-description__why-hybrid-glalery">Why Hybrid Gallery?</h4>
<p>Well, first of all, unlike most gallery plugins, Hybrid Gallery has very simple usage with extremely good usability. There is no admin panel with difficult instructions, hard manuals, all you need to do is just open popup panel (from Wordpress editor), add some images and that’s all! This gallery plugin is fully integrated with wordpress <strong>TinyMce editor</strong>, which means that you can set galleries everywhere editor exists: in posts/pages, custom post types, meta fields… </p>

<h4 id="item-description__why-hybrid-glalery">Responsive and Flexible</h4>
<p>Our <strong>gallery plugin</strong> is mega responsive. You can set your own <strong>responsive options</strong> for custom <strong>screen sizes</strong>. For example, you can set 3 columns and left alignment for <strong>grid layout</strong> if screen size is smaller than 820px and 4 columns with right alignment if screen size is smaller than 640px. It’s the most responsive wordpress gallery plugin.</p>

<p>What about flexibility, it is possible to work with any width, use every pixel, increase or decrease the gallery width how much you need. It’s more flexible than ever!!!</p>

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add post/page, click on 'Hybrid Gallery' button next to 'Add Media'.
4. Add images, select options and save it.
