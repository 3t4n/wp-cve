== Description ==
"Multiple Email Recipient" plugin helps to Send two additional email addresses that can be used as email recipients for WooCommerce new order and cancelled order emails. You can select which type of order email you want to have with up to two email recipients (Bcc) via the settings.
If you are in the sales, marketing or recruitment field, then most of the time, you need to send a single email to multiple recipients. With the help of this plugin , the store managers/sales admins can manage their new order notifications and cancelled emails to two email recipients. This is a free feature of this plugin.

== Installation ==
1. Upload the plugin files to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the /Plugins/ screen in WordPress
3. Settings/WooCommerce Multiple Email Recipients
3.1. Option to select Order email notifications
3.1.1. New order email
3.1.2. Cancelled Order email
3.2. Can enter up to two email addresses
3.2.1. Email Id 1
3.2.2. Email Id 2

== Free Features of Plugin ==
Option to select the new order mail and cancelled email for Bcc participant.
Sales admin can add up to two email recipient for receive email Bcc for sale new order and cancelled order notification.
Easy to understand and simple to operate.

= What is this method called of multiple email receipant? =
This method is called The BCC (Blind Carbon Copy) method (The most common approach to send email to multiple recipients at the same time. Emailing to multiple recipients using the BCC feature hides other recipients from the recipient making it look like the sole recipient of the email.
