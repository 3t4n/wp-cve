This plugin adds a custom HTML post type, with shortcode to place anywhere on your site.

== Description ==

WPS HTML Blocks, adds custom html blocks to WordPress. The custom blocks have shortcode to give you the ability to add these HTML blocks to anywhere on the wordpress site. It has been tested against most themes. Elementor and WP Bakery both are compatible.

== Installation ==

Upload the plugin files to the /wp-content/plugins/WPS HTML Blocks directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress

== FAQ ==

Q. What is this for? 
A. We found the need to the ability to create small HTML blocks to be able to place around various pages/posts and WooCommerce products. This plugin solves that problem.
