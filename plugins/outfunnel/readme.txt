Easily sync leads from various Wordpress forms to Pipedrive, Copper, HubSpot and other CRMs. Includes web visitor tracking.

== Description ==

[Outfunnel](https://outfunnel.com) plugin lets you track website visits and sync form submissions to your CRM. Works with Pipedrive, Copper, HubSpot CRM and Salesforce.

Supported forms: Elementor, Contact Form 7 and Gravity Forms. Learn more about syncing new form submissions to CRM [here](https://support.outfunnel.com/en/articles/5234038-wordpress-forms-crm-connection?utm_source=Wordpress&utm_medium=marketplace).

You need to have an Outfunnel account to use this plugin â€“ register [here](https://app.outfunnel.com/register).

For instructions to get the unique tracking code please register for an account and visit the Outfunnel [web tracking page](https://app.outfunnel.com/new-web-tracking/setup).

== Installation ==

1. Go to the Add New plugins screen in your WordPress Dashboard
1. Click the upload tab
1. Browse for the plugin file (outfunnel.zip) on your computer
1. Click "Install Now" and then hit the activate button
