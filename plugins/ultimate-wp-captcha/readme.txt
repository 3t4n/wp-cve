Enables Google reCAPTCHA and hCaptcha for the WordPress website forms.

== Description ==

Ultimate WP Captcha plugin is an effective security solution that prevents spam Login and registration of your WordPress website, WooCommerce and LearnDash. We are offer you to integrated Google reCaptcha and hCaptcha.

The Users need to confirm that they are not a robot before submitting WordPress forms.

= Features =

* Add reCaptcha to:
	* WordPress Registration form
	* WordPress Login form
	* WordPress Reset password form
	* WordPress Comments form
	* WooCommerce Registration form
	* WooCommerce Login form
	* WooCommerce Reset password form
	* WooCommerce Checkout page
	* LearnDash Registration form
	* LearnDash Login form

== Installation ==

1. Upload the `google-captcha` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Plugin settings are located in WUC Captcha menu.

== Frequently Asked Questions ==

= How to get Google reCaptcha keys? =

Follow the below steps in order to get and enable Google reCaptcha:
1. Open your WordPress admin dashboard.
2. Left Navigate “UWC Captcha” to go plugin Settings page.
3. Select Google reCaptcha.
4. Click the “Click here” link to create or view keys for Google reCaptcha.
5. Enter your domain name and click “Create Key” button.
6. Copy your site key and secret key them and paste to the appropriate fields on plugin Settings page.
7. Save changes

= How to get hCaptcha keys? =

Follow the below steps in order to get and enable hCaptcha:

1. Open your WordPress admin dashboard.
2. Left Navigate “UWC Captcha” to go plugin Settings page.
3. Select hCaptcha.
4. Click the “Click here” link to create or view keys for Google reCaptcha.
5. Register your account Or Create Key.
6. Copy your site key and secret key them and paste to the appropriate fields on plugin Settings page.
7. Save changes.
