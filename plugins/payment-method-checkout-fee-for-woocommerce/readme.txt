This is a WooCommerce plugin, that allows you adding an additional checkout fee for each of your payment methods.

== Description ==

This is a plugin for WooCommerce that allows you adding an additional checkout percent fee for each of your payment methods.

You can find the settings under "Settings" menu in Wordpress, titled 'Checkout Fee'.

You have an option to choose which payment methods should have the additional fee by ticking Enable.

Then you can add the percent you desire to be added as an additional fee for the selected method.
