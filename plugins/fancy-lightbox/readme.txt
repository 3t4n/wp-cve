== Description ==

### Fancy Lightbox

Add fancy lightbox easily, responsive lightbox and easy to use, without options and without complexity, compatible with all major browsers.

### The Features

1. Easy to use, just activate the plugin.
2. Responsive lightbox using Fancy Lightbox jQuery.
3. Without options and without complexity.
4. Compatible with any slider.
5. Compatible with all major browsers.

== Installation ==

### Installation

1. Upload 'fancy-lightbox' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Now plugin is ready! but [read explanation of use](http://wp-plugins.in/fancy-lightbox).
