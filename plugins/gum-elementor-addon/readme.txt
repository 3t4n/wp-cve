== Description ==

This plugin addon for Elementor Website Builder it mean you need install <a href="https://wordpress.org/plugins/elementor/">Elementor</a> plugin and activate before.

== Widgets ==

The plugin offers inbuilt widgets that help to create design more attractive.

* **Slideshow** - Slideshow based <a href="https://github.com/nicinabox/superslides">Superslides</a> engine with part: heading, subheading, text and two action buttons.
* **Gum Heading** - Three colors heading with line and image decoration.
* **Pricing Table** - Pricing table with two pricing systems.
* **Recent Post** - Show recent post list.
* **Blog Grid** - Show blog post ( including pagination ).
* **Blog Term** - Show blog term list ( tag, categories ).
* **Post Meta** - Show sortable post meta for single post.
* **Post Term** - Show sortable post term and divider for single post.
* **Post Related** - Show related post for single post.
* **Post Adjacent** - Show previous/next post for single post.
* **Post Share** - Share post to social media.
* **Post Image** - Show featured image for single post.
* **Post Slider** - Show recent post with carousel.
* **Popover Button** - Hotspot button icon.
* **Site Menu** - Show main menu as site navigation.
* **Special Option** - Inner section width like Top section, Inner section position like common widget.
* **To Top Button** - Show to Top Button.
* **Image Box** - Show image box with button and text.
* **Image Box Carousel** - Show image and button with carousel.

== KEY FEATURES ==

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Make sure Elementor Website Builder was installed and active.
