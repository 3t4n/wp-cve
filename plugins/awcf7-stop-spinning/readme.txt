== Description ==
 
Problems with the Contact Form 7 spinner? Is your Contact Form spinning forever? In this plugin we fixes the problem on you CF7 forms! This plugin is simple like 1,2,3. 
Download, activate and clear your website cache. 

FEATURES
* Simple code
* No settings required. This plugin works out of the box. 
 
== Installation ==
 
1. Log into your website administration panel.
2. Then go to Plugins page Add New > Upload Plugin .
3. Click “Choose file” then select our plugin .zip file.
4. Then Install and activate the plugin.
