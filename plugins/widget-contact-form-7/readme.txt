== Description ==

Makes Contact Form 7 easier to use by adding a widget so no shortcodes are needed.

Contact Form 7 is the most widely used form plugin for WordPress as it's secure, easy to use and has advanced options. This widget allows you to save time and energy implementing and using Contact Form 7.

Please note, you need Contact Form 7 plugin activated so this widget can find your created forms.

This plugin was created as part of the [Marketer WordPress business theme](http://themersclub.com/marketer)

== Installation ==

1. Upload `widget-contact-form-7` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to widgets and see the new Contact Form 7 widget available!

== Frequently asked questions ==

= How do I use this plugin? =

This plugin create a widget that will lookup contact form 7 forms. You will need contact form 7 installed:
http://wordpress.org/plugins/contact-form-7/

= Why contact form 7? =
This is the most widely used form system for WordPress and generally has a great easy to use user interface with some advanced options. The one great omission is a widget, which this plugin solves.

= Why a widget? =
Yes, you can just use shortcodes to achieve the same thing, but this just makes it easier especially for non-tech savvy users.
