== Description ==
[ManyContacts](https://www.manycontacts.com/) is a CRM connected with WhatsApp that allows you to manage hundreds of conversations with a small team.

Use ManyContacts to get more leads and grow your sales.

### Features: ###

*   Unlimited number of agents.
*   Manage hundreds of conversations.
*   Organize your conversations with tags and our powerful search engine, never miss a deal.
*   Use short responses, internal notes, teams, and some other tools to improve your customer service.
*   IVR: Set and interactive response to categorize the conversations automatically.
*   WhatsApp Business: Connect your number in one minute scanning the QR code.
*   WhatsApp API: We help you to setup you number in the official WhatsApp API.
*   API and integrations: We provide an API to allow you to integrate with your own software. Also we have integrations with the most common CRM and Marketing providers like Hubspot, Pipedrive, ...

== Installation ==
1. Go to the Wordpress Pluglins Page.
2. Select the ManyContacts pluglin and click Install.
3. On your WP Pluglins section, look for ManyContacts and click Enable.
4. Go to [manycontacts.com](https://app.manycontacts.com/register) and create an account.
5. Scan the QR Code.
6. On ManyContacts -> Go to "Settings > Widget" and copy the code snippet.
4. Paste your ManyContacts snippet from [manycontacts.com](https://www.manycontacts.com/) into the text area and click save!

Manual Installation:
1. Upload the `manycontacts-bar` folder and all its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the same instructions as the automated installation.
