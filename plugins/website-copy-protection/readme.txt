== Description ==
Website Copy Protection prevents will protect the content of you website. On activating this plugin, copying the text, image, source code, taking printout, saving the page etc won't be available.  The activation is a one click process and no other settings are required! Deactivation is equally easy.

**Easy to Install**:

Installation is a one click process and does not require any coding or additional settings.

**Features:**
<ul>
<li>Secure your website content without being rude to your readers.</li>
<li>No one can download images from your website.</li>
<li>Disable right click.</li>
<li>Disable the following key combination Ctrl + A, Ctrl + C, Ctrl + X, Ctrl + S, Ctrl + V, Ctrl + U, Ctrl + I and Ctrl +J.</li>
</ul>


== Installation ==
<ul>
<li>Click Plugins > Add New</li>
<li>Search for Website Copy Protection</li>
<li>Activate Website Copy Protection from your Plugins page.</li>
</ul>
<p>Plugin would start functioning as soon as you activate it. </p>
