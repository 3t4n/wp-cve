== Description ==

The bbPress Shortcodes plugin provides a TinyMCE dropdown button in the visual editor that users can access to use any bbPress shortcodes.

It also parses short codes placed in topic and reply content.

You can use plugin settings in the admin area to customize its functionality.

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins -> Add New -> Search` and search for bbPress Shortcodes).
* Alternatively, you can [download](http://downloads.wordpress.org/plugin/bbpress-shortcodes.zip "Download bbPress Shortcodes") the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* Use the `Settings -> bbPress Shortcodes` to configure the plugin

== Frequently Asked Questions ==

= Why does TinyMCE dropdown button not display in post type editor? =

Please make sure you are using visual editor and have configured the plugin settings from Settings -> bbPress Shortcodes
