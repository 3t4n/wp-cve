Display FREE if WooCommerce Product Price is Zero or Empty.

== Description ==
Free Product for WooCommerce is a lightweight and user-friendly plugin that allows you to replace the price display of WooCommerce products with the word "Free" instead of "0.00". 

It's a simple solution to provide a more intuitive shopping experience for customers when they encounter free products in your WooCommerce store. 

With this plugin, you can easily indicate which products are offered at no cost, eliminating any confusion about pricing. 

Enhance your customers' experience and boost conversions with the Free Product for WooCommerce plugin.

== Features ==
- Automatically replaces the price display of WooCommerce products with the word "Free".
- Improves the shopping experience by clearly indicating which products are free of charge.
- Lightweight and easy to use, with no complex configuration required.
- Compatible with the latest version of WooCommerce.

== Installation ==
1. Upload the "free-product-for-woocommerce" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it! The price display of your WooCommerce products will now show "Free" instead of "0.00" for free products.

== Usage ==
1. After activating the plugin, visit your WooCommerce store and navigate to the product edit screen.
2. For any product that should be displayed as "Free", set the regular price to "0.00" as usual.
3. Save/update the product, and the price will be displayed as "Free" on the product page and shop/catalog pages.

== Frequently Asked Questions ==
= Will this plugin affect the checkout process or payment gateways? =
No, this plugin only changes the display of product prices. It does not interfere with the checkout process or payment gateways.

= Is this plugin compatible with the latest version of WooCommerce? =
Yes, the Free Product for WooCommerce plugin is designed to be fully compatible with the latest version of WooCommerce.
