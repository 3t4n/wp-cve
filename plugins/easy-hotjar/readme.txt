Set up hotjar on your site in a matter of seconds

== Description ==

Set up hotjar on your site in a matter of seconds. The Easy hotjar WordPress plugin helps you to set up hotjar on your site.

1. Open hotjar configuration page

2. Enter your hotjar site id

3. That's it! :)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin folder `easy-hotjar-wordpress` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

or

1. Install the plugin from the 'Plugins' page

== Frequently Asked Questions ==

Q. Where can I find the site Id?

A. You can find it easyly on your address bar. 
Just log in your hotjar account. Click on your site dashboard and take a look of the address bar.
