== Description ==
WP Cookie Law Info allows you to easily show Cookie Law Popup for showing
a notice to the user only the very first time that visits your website.

Message, position and the style can be easily modified through the plugin menu.

The WP Cookie Law Info plugin supports the following features:
<ul>
<li> Choose Custom background Color</li>
<li> Choose Custom Button Color</li>
<li> Add Custom Message</li>
<li> Options for Show/Hide Cookie Law Popup</li>
<li> Options for Change Popup Position</li>
<li> Options for Change Popup Layout (4 Layouts available )</li>
<li> Options for Change Color Style</li>
<li> Options for Add Custom policy link and Label Text</li>
</ul>

== Installation ==
1. Upload the wp-cookie-law-info folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > WP Cookie Law Info
