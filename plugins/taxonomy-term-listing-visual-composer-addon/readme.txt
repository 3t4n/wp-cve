Taxonomy Term List Visual Composer Addon lets you give list of different categories, taxonomies and nested categories of different post types.

== Description ==

Taxonomy Term List is the WordPress Visual Composer Addon that shows the list of categories or taxonomies of selected post types. It gives various options for customizing the lists. It can also show the list of child categories of any particular category. It is compatible with Visual Composer of version 4.11 or higher.

Features
•   Easy installation
•   Show all categories, nested categories and taxonomies
•   Free support
•   Multi-lingual support
•   Gutenberg Compatible
•   WPBakery Page Builder compatible
•   Translation ready (Gujarati)

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Important: You need to have installed and activated Visual Composer
2. Install the plugin via WordPress or download and upload the plugin to the /wp-content/plugins/
3. Activate the plugin through the 'Plugins' menu in WordPress
4. You can see the 'Taxonomy Term List' element under Visual Composer Element Grid.
