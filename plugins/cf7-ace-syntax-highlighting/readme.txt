Adds syntax higlighting to the Contact Form 7 admin screens. Requires the Contact Form 7 plugin.

== Description ==

Are you using HTML code in your Contact Form 7 forms and email bodies? Ace syntax highlighting enhanches the Contact Form 7 backend and makes it easy to code HTML for complex forms.

== Installation ==

The plugin assumes you're already using Contact Form 7, if not install it first before you proceed. 

1. Install the plugin from the Plugins menu and then activate it.
2. When using Contact Form 7, you will now have syntax highlighting in all form and email edit textarea's.

= Manual Instructions =

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
