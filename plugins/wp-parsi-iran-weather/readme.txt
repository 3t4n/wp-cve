Get accurate and beautiful weather forecasts for Iran cities powered by 'weather.com' for your site.

== Description ==

Full ajax (jQuery powered) and multilingual widget with functionality shows current weather and forecasts Iran cities. This plugin powered by weather.com for your site.

Language Support:

* English
* Persian - Thanks [Farzad](http://forum.wp-parsi.com/user/485-farzad/)

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Widgets' and choose 'WP-Parsi Iran weather'

* Structure of shortcode:
`[wppwhtr]`
