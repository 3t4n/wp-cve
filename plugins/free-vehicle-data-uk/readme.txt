== Description ==

Free Vehicle Data UK plugin makes adding a vehicle search by registration to your WordPress site quick and easy, plug in to the free version of the Rapid Car Check API in minutes.



Includes PDF guide and short code list to implement vehicle data by registration into your WordPress site, entirely free API access and usage statistics in admin backend.

== Installation ==

Download and install the plugin. Either by the WordPress marketplace or via a .ZIP file.



Proceed to "Plugins" under the WordPress Administration Page and click "Activate" under the "Free Vehicle Data UK" plugin.



Go to WordPress admin backend, click "FVD UK" menu link.



Click the link that says “Sign Up Free Now”.



Enter your details and click “Activate API (Instant)”.



Go back to wp-admin page “FVD UK” page and reload or refresh the page to show API usage statistics.



The plugin should now be activated! 



Go back to wp-admin page “FVD UK” and click "IMPORT DEMO PAGE CONTENT" to automatically create vehicle data page. Full list of short codes, PDF documentation and API usage statistics will be available in admin area.



== Frequently Asked Questions ==

Q: Is API Access Free?

A: Yes, we do offer free API access to WordPress plugin users.



Q: What vehicle data is included in API?

A: Make, Model, Colour, Year, Fuel Type, Vehicle Type, Body Style, BHP, TopSpeed, 0-60MPH Speed, Insurance Group, Age, Date of registration, Cylinder capacity, Mileage, MOT, Road tax status, MOT history



Q: Can I only lookup UK vehicles?

A: Yes, at the moment we only provide data for UK vehicles.



Q: Does this plugin require 3rd party plugins?

A: Yes, but only if you require the mileage graph feature.



Q: Does this plugin track vehicle searches and requests?

A: When a request is made, the vehicle registration and your WordPress site URL is passed to the API endpoint for fair usage reasons and to process requests.
