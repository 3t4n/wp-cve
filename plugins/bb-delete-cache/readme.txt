Add delete beaver builder cache button in admin bar!

== Description ==

With BB delete cache, you have now a button in the admin bar to clear Beaver Builder cache for the current post or all the website.

== Installation ==

1. Upload the plugin files to the plugin directory, or install the plugin through the WordPress plugins screen directly.
1. Activate through the 'Plugins' screen in WordPress.
