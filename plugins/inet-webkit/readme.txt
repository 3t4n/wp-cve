iNET Webkit được xây dựng theo tiêu chí gọn nhẹ, tinh giản tối đa việc tìm hiểu và công cụ cài đặt hỗ trợ vận hành website Wordpress.
Chúng tôi tham khảo và lựa chọn ra 1 loạt các công cụ phổ biến cần thiết trên hầu hết các website để tích hợp theo hướng
"All in one", đưa các tùy chọn vào một plugin thống nhất, việc sử dụng chỉ cần bật / tắt / nhập số liệu.

Việc giảm số plugin, giảm dung lượng cũng như tối ưu tốc độ giúp website của bạn thân thiện hơn, chuẩn SEO và dễ dàng lên TOP Google hơn.

==> Các tiêu chí iNET Webkit hướng tới <==
1. Giảm tối đa số lượng plugin cài lên hệ thống.
2. Giảm dung lượng từ các plugin mới tải lên.
3. Cải thiện bảo mật, tốc độ tải trang.
4. Giảm thời gian tìm hiểu, cài cắm thêm các plugin hỗ trợ vận hành.
5. Cài đặt sẵn các tiêu chí hỗ trợ bán hàng trên website.

== Hướng dẫn Cài đặt ==
Sử dụng phương thức cài đặt Plugin iNET Webkit từ Bảng điều khiển WordPress hoặc thông qua upload FTP thủ công.

== Từ Bảng điều khiển WordPress ==
1. Điều hướng đến Plugin -> Cài mới từ bảng điều khiển WordPress của bạn.
2. Tìm kiếm “iNET Webkit” và cài đặt.
3. Kích hoạt Plugin từ menu Plugins.
4. Thực hiện cấu hình Plugin.

== Cài đặt thủ công ==
1. Tải xuống tệp plugin: “inet-webkit.zip”.
2. Giải nén tệp tin.
3. Copy thư mục “inet-webkit” vào thư mục “/wp-content/plugins” trên hosting của bạn (không đổi tên thư mục)
4. Kích hoạt Plugin từ menu Plugins.
5. Thực hiện cấu hình Plugin.
