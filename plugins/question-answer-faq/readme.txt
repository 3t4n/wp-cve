== Description ==

Question answer, ajax, bootstrap, gravatar avatar plugin with email notification. It looks like a chat. Add shortcode [mideal-faq-form] on your page to set the form question. Add shortcode [mideal-faq] on your page to implement the question list.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/question-answer-faq` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Question -> 'Setting' screen to configure the plugin
4. Use shorcode [mideal-faq-form] on a page for add form.
5. Use shorcode [mideal-faq] on a page for add list Question and answer.

== Screenshots ==

1. Form for adding questions.

2. List question-answer. If you have permission you will see 3 buttons more - replay, publish, delete.
