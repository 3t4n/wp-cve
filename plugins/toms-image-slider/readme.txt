== Description ==

<a href="https://toms-caprice.org">TomS Image Slider</a> is a Simple image slider block. Absolutely light weight and **easy to use** even without explanation. This is a image slider block and all settings inside the block, 
if your theme support the block editor, you can add our block any where you want. No support the shortcode. Of course, you don’t need to upload image files to the wordpress media library. You can paste image url directly .You can store your image files on a CDN server to reduce the burden on the wordpress server.


= TomS Image Slider Features =

Here is the Slider Settings
* **Image Slider:** Temporarily disable Image Slider, if you don’t want to use it for now.
* **Auto Play:** Enabled automatic switching the image.
* **Thumnbail：** Enabled the image thumnbails.
* **Modal:** Enabled the image modal.
* **Modal preview in Editor:** Enabled Modal preview in current Editor, to make this option work you need to enable Modal first.

== Frequently Asked Questions ==

= Why the Auto Play not work in editor? =

When you Enabled the Auto Play option, you need to save the post/page, and reload the post/page.

= This plugin support shortcode ? =

No, this is a block of gutenberg.
