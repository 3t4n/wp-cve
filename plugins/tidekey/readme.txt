TiDeKey - Title + Description + Keywords are the first things you do for OnPage SEO, so why don't you customize your SEO meta template easy?

== Description ==

This SEO plugin for WordPress allows you to create fully customizable:

- SEO Title

- SEO Decription
 
- SEO Keywords

To create any of SEO meta you will need to add a template to different types of content, as Post/Categories/Taxonomies/Custom Post Types/etc.

So, if you have a few thousands of pages on your wordpress of one kind, let's say recipes, you can create template for all recipes with just one sentences.

For example:
Using title template like this: "%TITLE% recipe - How to make %TITLE% easy!" , will get your page titles with recipes look like this: "Potato salad recipe - How to make potato salad easy!" .

As you can see with just a small title template, you can get a greatly SEO optimized titles for thousands of your pages with recipes (or whatever product you have).

GOOD NEWS for those who are struggling with How to do Multilingual SEO for WordPress Sites (using WPML). 

This small plug-in is fully compatible with WPML plugin for Multilingual Websites, which is really helpfull as you can create all SEO meta templates for each language in one place.

== Installation ==

1. Unzip the archive on your computer  
2. Upload `tidekey` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to the 'Settings' -> 'Tidekey' and set the settings you want.
