Mail Debug for WooCommerce allows you to debug WordPress and WooCommerce emails.


== Description ==
Mail Debug for WooCommerce allows you to debug WordPress and WooCommerce emails.
After installing and activating the plugin, you'll see a new "Mail Debug" menu, in which you'll be able to see each email sent by WordPress.
Furthermore, you can see additional information about the emails sent; you can view also an email preview.

= Features: =

* A specific panel to see each email sent by WordPress.
* Possibility to enable/disable debug.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder and upload it into the directory `wp-content/plugins/` in your WordPress site.
3. Activate `Mail Debug for WooCommerce` from Plugins page

Mail Debug for WooCommerce adds:
* a new entry in admin menu called "Mail Debug", from which you can see each email sent by WordPress;
* a submenu called "Settings" in "Mail Debug" menu. Here you can find the plugin settings.
