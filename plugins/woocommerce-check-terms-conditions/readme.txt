WooCommerce Check Terms & Conditions  will check terms & conditions by default on the woocommerce billing page.

== Description ==
Installs WooCommerce Check Terms & Conditions on your wordpress blog so it will check terms & conditions by default on the woocommerce billing page .

== Installation ==

* Use WordPress’ builtin plugin installation system located in your WordPress admin panel, labeled as the "Add New" options in the "Plugins" menu to upload the zip file you downloaded
* Extract the zip file and upload the resulting "ink" folder on your server under `wp-content/plugins/`.

All you need to do after that is navigate to your blog’s administration panel, go in the plugins section and enable WooCommerce Check Terms & Conditions.


For more information, see the ["Installing Plugins" article on the WordPress Codex](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins).
