== Installation ==

= English =
1. Upload the directory `/pwd-wp-login/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on "Customize" > "Login page" and upload your favicon

= Français =
1. Envoyez le dossier pwd-wp-login dans le dossier /wp-content/plugins/
2. Activez PWD WP Login sur la page des plugins de WordPress
3. Configurez le plugin en allant dans le menu Personnalisé placé dans le menu Apparence
