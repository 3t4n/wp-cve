== Description ==

Hoo contact form plugin.

== Installation ==

Unzip plugin files and upload them under your '/wp-content/plugins/' directory.

Resulted names will be: './wp-content/plugins/hoo-contact-form/*'

Activate plugin at "Plugins" administration page.

Shortcode

Use the Hoo Contact Form shortcode to display the contact form on any WP Post or Page:

[hoo_contact_form]
