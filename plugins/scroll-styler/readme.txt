Scroll Styler offers a solution to add custom scrollbar style on your WordPress website.

== Description ==

The Scroll Styler allow you an easy customization. The RGB color picker helps you to create the best color balance between the website and the scrollbar.

= Browser Compatibility =
Scroll Styler works only on WebKit browsers.

== Features ==

* Custom Scrollbar
* Scrollbar Width
* Scrollbar Track Padding
* Scrollbar Track Background Color
* Scrollbar Thumb Background Color
* Scrollbar Thumb Radius

== Installation ==

- First, log in to the Administration area of your Wordpress website and select the Plugin.
- Click the Add New button on the top of the page.
- Select the Upload Plugin button on the top of the page again.
- Browse the plugin zip file from your computer and click the Install Now button.
- Click on the Activate Plugin link and the plugin will be active.
- After successful plugin activation, you will redirect to the plugins' list page.
- You can find the Scroll Styler plugin in the list.
