== Description ==
[Disable Right Click and Content Copy Protection] is a free wordpress plugin. It gives users the ability to prevent right Click on wordpress site.The plugin is lightweight. Use it and Enjoy. <br><br> **But when Administrator or Site Editor is logged in, he can access everything without any of the above restrictions.**
<br><br>

== Installation ==
1. Go to the Add New plugins screen in your WordPress Dashboard
2. Click the upload tab
3. Browse for the plugin file (disable-right-click-and-content-copy-protection) on your computer
4. Click \"Install Now\" and then hit the activate button.
