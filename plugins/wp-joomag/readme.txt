== Description ==

This plugin is for WordPress 2.1 and higher which simplifies the embedding of [Joomag](http://joomag.com) publications in blog posts. Simply copy the WordPress embed code provided by Joomag and paste it into your post.

== Installation ==

Here's how to install the plugin and how to make it work.

1. Upload the `wp-joomag.php` file to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Copy WordPress embed code from Joomag.com and paste it to your post: `[joomag magazineid=XXX width=400 height=350 ... ]`.
