Built-in PCI compliance, FX and settlement into your preferred currency.

= Why choose Rapyd? =


Start Accepting 900+ Payment Methods on Your WooCommerce site. To begin accepting payments through Rapyd with Wordpress and WooCommerce, you must create a Rapyd account and have a Wordpress site running WooCommerce. 




== Installation ==


1. Sign up for a Rapyd Client Portal Account at https://dashboard.rapyd.net/sign-up
2. Activate Your WooCommerce Plugin in the Rapyd Client Portal by going to Settings > Plugins and clicking the Activate Plugin button in the WooCommerce window.
3. Plugin Installation on WooCommerce:
1. A. Sign in to your WordPress site and navigate to Tools > Plugins.
2. Select Plugins > Add New from the main menu.
3. In the Search Plugins text-box enter Rapyd Payments Plugin. The Rapyd Payments Plugin for WooCommerce appears.
4. Click Install Now.
5. When the Installing plugin page appears click Activate Plugin.
6. Rapyd Payments Plugin Configuration.
You can find the complete installation guide here https://docs.rapyd.net/docs/rapyd-payments-plugin-for-woocommerce-new 


= Does this support recurring payments, like for subscriptions? =
The Rapyd plugin does not support recurring payments. 


= Does this require an SSL certificate? =
Rapyd does not require an SSL certificate.


= Does this support both production mode and sandbox mode for testing? =
Rapyd offers both a sandbox environment for testing and a production environment when you want to go live
