== Description ==

Easy Code Snippets is light weight WordPress plugin, it allows you to create CSS or JS code snippets. Where you can manage analytics code, tracking code and other custom JS and CSS code. you can use there snippets to in your website at header or footer.

= Main Features =

* Light weight plugin
* Dynamic JS/CSS code snippets
* Easy to setup and configure
* Manage CSS or JS codes to your website pages
* WordPress default HTML code editor
* Option to display code in header or footer
* 100% Multi language

= Pro Features =

* Unlimited JS/CSS code snippets
* Display code snippets to particular pages only.
* Shortcode ready

== Installation ==

Following the steps below, you can install plugin and running in seconds.

1. Upload 'easy-code-snippets' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the new menu item "Easy Code Snippets"
