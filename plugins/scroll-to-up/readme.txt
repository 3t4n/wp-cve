Scroll To Up is a lightweight plugin that creates a full customizable "Scroll to top / Back to top" feature in your WordPres site
== Description ==

Scroll To Up plugin makes the scrolling of a page easier. It appends an icon at bottom of the page which attracts user attention. When the user click on this button it smoothly scrolls to the top of Webpage. `Scroll To Up` offers custom settings which no other plugin have.
While you activate the plugin you will redirect to the plugin setting page. You can go to the `Scroll To Up` setting page from wordpress settings menu.


= Features =
* Pre-Designed Icons.
* Fluid/responsive icon.
* Compatible with all WordPress themes.
* It is user friendly and very easy to install.
* Zero coding skills required to set up this plugin.
* Customizable options via the admin panel
* You can change background color.
* You can add custom text.
* You can change text color.
* You can set required height/width.
* You can set the position of icon.
* It is user friendly and very easy to install.
* Zero coding skills required to set up this plugin.
* You can set your custom icon by editing the custom icon from settings menu.

== Installation ==

= For automatic installation: =
* The simplest way to install is to click on 'Plugins' then 'Add New' and type 'Scroll Up' in the search field.
* Click 'Install Now' and have Fun!

= For manual installation: =
* Login to your website and go to the Plugins section of your admin panel.
* Click the Add New button.
* Under Install Plugins, click the Upload link.
* Select the plugin zip file (scroll-to-up.zip) from your computer then click the Install Now button.
* You should see a message stating that the plugin was installed successfully.
* Click the Activate Plugin link.


= Installation from server: =
* You should have access to the server where WordPress is installed. If you don't, see your system administrator.
* Copy the plugin zip file (scroll-to-up.zip) up to your server and unzip it somewhere on the file system.
* Copy the "scroll-to-up" folder into the /wp-content/plugins directory of your WordPress installation.
* Login to your website and go to the Plugins section of your admin panel.
* Look for "Scroll To Up" and click Activate.
