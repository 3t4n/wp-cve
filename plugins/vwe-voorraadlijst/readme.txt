Voor autobedrijven - Zet eenvoudig uw voorraad online in uw Wordpress website. Kijk voor meer informatie op https://www.vwe.nl/Aanmelden-als-klant

== Description ==
Door middel van deze plugin kunt u eenvoudig uw aanbod integreren binnen uw Wordpress website. U heeft hiervoor een account nodig van VWE. U kunt uzelf aanmelden via onze website https://www.vwe.nl/Aanmelden-als-klant.

== Installation ==
1. Navigeer naar 'Plugins' -> 'Add New' en klik 'Upload Plugin' selecteer daarna de .zip of pak de .zip uit en plaats de bestanden in de `/wp-content/plugins/` map
2. Activeer de plugin via het 'Plugins' menu in WordPress
3. Navigeer naar 'Settings' -> 'VWE' of druk op Instellingen en vul het Dealer ID in dat u van VWE heeft gekregen
4. Plaats de shortcode [ad-voorraadlijst] op de gewenste plek
5. Bij gebruik [ad-carousel] of [ad-search-block] tevens Basis URL meegeven naar de URL waar de [ad-voorraadlijst] is geplaatst, vul deze URL aan met ?AdFrameSource=https://svl.autodealers.nl/
6. Parameters veld optioneel invullen bij instructie door VWE
