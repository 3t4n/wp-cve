Hide Any Page is a simple and lightweight plugin that allows you to hide any page from your website's visitors and search engines such as Google. It's great for hiding pages that you don't want to appear on your website, such as thank you pages and download pages.

== Description ==

The Hide Any Page plugin allows you to hide any page on your WordPress website or blog.

The plugin lets you select pages that you want to prevent from appearing in standard menus, lists and searches. It will add meta tags to your selected page which tell search engines such as Google not to index the page or to keep a cached copy.

The Hide Any Page plugin is useful for download pages, thank you pages and other pages that you want to hide from your visitors.

== Installation ==

1. Unzip the download file if necessary and upload the 'hide-any-page' folder and all its contents to your '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in your WordPress admin.
3. Click the 'Hide Any Page' menu under the Settings menu in your WordPress admin.
4. Check the page(s) that you want to hide from your visitors and search engines such as Google.
5. Click the Save button to apply.
