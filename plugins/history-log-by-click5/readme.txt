== Description ==

Best WordPress plugin to <b>track user activity</b> and <b>log changes</b> on your website.

Keep a detailed record of important events that occur in the WordPress Dashboard of your website. It helps to know what has been changed or what plugins have been updated while troubleshooting possible errors. Easy to navigate report, which can be easily filtered down only to the events of your interest, will make the task much easier. No more wondering who deleted the last post, or who installed a new plugin. == Installation ==

= AUTOMATIC INSTALLATION =

Automatic installation method is the easiest and most recommended option.

1. Log in to your WordPress Dashboard, navigate to the Plugins menu, and click on ‚ÄúAdd New‚Äù.
2. Enter ‚Äòclick5‚Äô (without quotes) in the search field and click the ‚ÄòSearch Plugins‚Äô button.
3. On the list of search results, click the 'Install Now' button for the 'History Log by click5' plugin.
4. Then click the red 'Active' button to finalize the installation.
5. You will be automatically redirected to the plugin page.

= MANUAL INSTALLATION =

1. Download the plugin and unzip the contents. You should see a single folder named 'history-log-by-click5'.
2. Connect to the FTP server with your WordPress installation.
3. Upload the 'history-log-by-click5' plugin folder to the /wp-content/plugins/ folder on your FTP server.
4. Continue to step 3 of Automatic Installation instructions above.

== Frequently Asked Questions ==

= How long are the logs stored for? =

Events in the history log are stored for 6 months (by default). Events older than this will be automatically removed.
Depending on your needs, history log data can be stored for 1 week, 1 month, 3 months, 6 months, 12 months, or Indefinitely.
