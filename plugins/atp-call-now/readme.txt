== Description ==

###Custom easy plugin call with ATP Call Now
An easy way to edit the call button's color main and icon, text's color and background. Setup position left, right, top, bottom. Edit size Call button, change icon for Call button.

###Optimized for mobile
Optimized for responsive websites.

== Installation ==

1. From your WordPress Dashboard go to 'Plugins' > 'Add new' and search for 'ATP Call Now'.

2. Click 'Install Now' under the title of the ATP Call Now plugin.
3. Click activate
.
4. Go to 'Settings' > 'ATP Call Now' and enter your phone number on input Text and enter link tel: on link when click.

5. Click 'Save' and you're done!
