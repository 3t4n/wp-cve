== Description ==

百度推送百度收录SEO-Baidu Push是一款解决把你文章内容手动推送或者自动推送到百度里，让百度第一时间抓取你的内容.

## 插件功能

* 支持文章手动推送到百度普通收录里
* 支持后台自定义域名接口和token
* 支持对应https
* 支持新发布文章自动推送到百度普通收录，无需要人工干预+
* 支持老文章自动推送到百度普通收录，无需要人工干预
* 支持采集回来文章自动推送到百度普通收录，无需要人工干预
