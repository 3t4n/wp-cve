== Description ==
Very Simple plugin to enable shortcods inside Widget , Comments, Experts, Category, Tag, and Taxonomy Descriptions.
 
<strong>Feature</strong>

* Plugin allows to ability to use the shortcodes inside:
1. Text Widgets
2. Comments
3. Experts
4. Categories / Taxonomy Descriptions

== Installation ==
1. Upload the folder "enable-shortcodes-inside-widgets-comments-experts" to "/wp-content/plugins/"
2. Activate the plugin through the "Plugins" menu in WordPress
3. Enjoy! as there is setting no setting.
