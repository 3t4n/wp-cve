== Description ==

You can easily place your page into maintenance mode. You can customize and create your own maintenance page.

== Installation ==

This section explains how to install and operate the plugin. <br>


1. Plug-in file to `/wp-content/plugins/WP-Maintenance-page` directory or directly install the plug-in from the WordPress plug-in screen. <br>
2. Enable plug-ins from WordPress "plug-in" screen<br>
3. Configure the plugin using the setting-> plugin name screen <br>
