"Easy Pixels" puts tracking codes for Google Analytics, Google Ads, Facebook Ads, Bing Ads, LinkedIn Ads, Twitter Ads, Yandex, Google Tag Manager

== Description ==

"Easy Pixels" sets the main tracking codes for Google Analytics, Google Ads, Bing, Facebook, LinkedIn, Yandex and Twitter pixel.

Sets the Google TagManager Tracking code.

Google tracking sets the main Gtag label.

Some plugin extensions supports Contact Form 7 tracking and WooCommerce tracking for advanced marketing purposes.

= Extensions =
You can install these extensions too:

- [Easy Pixels CF7 extension](https://wordpress.org/plugins/easy-pixels-contact-form-extension-by-jevnet/) supports tracking for the ["Contact Form 7" plugin](https://wordpress.org/plugins/contact-form-7/) form submissions without extra configurations.
- [eCommerce extension](https://wordpress.org/plugins/easy-pixels-ecommerce-extension-by-jevnet/) supports tracking for WooCommerce plugin. Includes enhanced ecommerce transaction events.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/easy-pixels-by-jevnet` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to "Easy Pixels" Menu option and set the codes you will use.

Nothing else!
