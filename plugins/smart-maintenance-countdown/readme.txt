This is a very easy and simple plugins to maintain the under maintenance and Coming soon countdown period for wordpress. This plugin three template from where user can choos any of them.

== Description ==

Using this easy manitenance and countdown user can easily setup the maintenance phase for any peticular user role. 

*	Status			: Using this plugin user can change the status On / Off to his wordpress site.
*	Page Title		: Usser can set his custome Page title for under construction page.
*	Company Name	: Can set his company name.
*	Message			: Can set custome messege.
*	Templates		: User can choose / switch from 3 nice and beautiful templates
*	Roles To Deny 	: User can deney (Editor, Author, Contributor, Subscriber) these roles to visit by selecting them.

User also can set the following parameners which will show in the Maintanence page according to user settings

*	Comeback Time.
*	Contact Email.
*	Contact Number.
*	Facebook Link.
*	Twitter Link.
*	Google+ Link.

== Installation ==

1. Upload `smart-maintenance-countdown` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `SMaintenance` menu and set the status ON and save settings.
4. Now 'Log out' and go to your Website font.
