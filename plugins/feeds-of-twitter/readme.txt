### GS Twitter Feed’s shortcode usage

`
[gs_tweet username="wordpress" theme="gstf_theme1"]
`

<strong>GS Twitter Feed’s Shortcode attributes Usage</strong>
`
[gs_tweet username="wordpress" tweet_number="10" theme="gstf_theme1" link="_blank"]
`

<strong>Shortcode PHP Usage</strong>
`
<?php echo do_shortcode( '[gs_tweet username="wordpress" theme="gstf_theme1"]' ); ?>
`

Template Usage – Add the shortcode anywhere you need to display GS Twitter Feed in template files (header.php, front-page.php, etc.)
`

== Installation ==

Install WordPress Posts Grid plugin as a regular WordPress plugin. Here is different ways to install WordPress Posts Grid plugin:

*   [Visual Installation & Usage Guide &raquo;](https://docs.gsplugins.com/wordpress-twitter-feed)

= Install by Search Plugin =
- In your Admin, go to menu Plugins > Add New
- Search - GS Twitter Feeds
- Click to install
- Activate the plugin
- Twitter Feed settings will found under GS Plugins > Twitter Feed Settings

= Install by Upload Plugin =
- Download the latest version of GS Twitter Feeds (.zip file)
- In your Admin, go to menu Plugins > Add New
- Select **Upload Plugin**
- Click on **Choose File**
- Select downloaded gs-twitter-feeds.zip & click on ***Install Now*** button
- Activate the plugin
- Twitter Feed settings will found under GS Plugins > Twitter Feed Settings

= Install Plugin using FTP =
- Upload gs-twitter-feeds folder inside the /wp-content/plugins/ directory
- Go to WordPress dashboard > Plugins > Activate the GS Twitter Feed
- Twitter Feed settings will found under GS Plugins > Twitter Feed Settings

* Put [gs_tweet username="wordpress" theme="gstf_theme1"] shortcode or select another theme, where you need to display WordPress Posts.

= How to use shortcode inside page templates? =
WordPress has a great function, do_shortcode(), that will allow you to use shortcodes inside your theme files. For example, to output Tweets in a Theme file, you would do this: 
`
<?php echo do_shortcode('[gs_tweet username="wordpress" theme="gstf_theme1"]'); ?>
`
