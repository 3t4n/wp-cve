== Installation ==

= Requirements =

To install OTP-less, you need:

* WordPress Version 5.5.1 or newer (installed)
* PHP Version 7.4 or newer

= Instructions =

1. Log in to your WordPress admin panel.
2. Go to Plugins > Add New.
3. Search for the **OTPLESS** plugin.
4. Click on **Install Now** and wait until the plugin is installed successfully.
5. You can activate the plugin immediately by clicking on **Activate** now on the success page.
