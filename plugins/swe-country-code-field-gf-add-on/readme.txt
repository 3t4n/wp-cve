Gravity Forms Addons for Inetrnational Phone code in drop down with flags 

== Description ==

This is an addon of gform to create a field in Advanced fields section for phone number with international phone code in dropdown with country flag in front-end form display. It's very easy to use and good to look on the form. 
This is fulfilling the requirement for phone number with phone code fields in Gravity Forms. 


== Installation ==

1. Unzip and upload plugin to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Form creat or edit section and click on Advanced fields section, Under dropdown of Advanced fields section, there is fields with name Phone Code. 
4. Just Add the field phone code and check the form in fron-end.
