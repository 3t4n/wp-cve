Conditionally restrict partial content on divi page. Divi visual builder compatible. Grant access to logged-in users, users belonging to any/all selected roles. Integration with third-party plugins like "WooCommerce Membership", "MemberPress", "Restrict Content Pro" & "Custom WordPress Filter".

== Description ==

## Restrict Content On Divi Page

https://www.youtube.com/watch?v=FH2uT690fSs

**Conditionally restrict** access to **partial content** on a **divi section** 

> 100% divi visual builder compatibility.

This plugin restricts content on divi section. Its 100% compatible with divi visual builder.

We've added **Restrict Content** & **Un-Authorized Text** settings toggle on divi section module. Its on the **Advanced tab**

In the **Restrict Content** toggle, setup all the conditions for restricting content like

* Show content to logged-in users
* Show content to users with specific roles (Premium Version)
* Show content based on third-party plugin's like
    - [Restrict Content Pro](http://bit.ly/2JNNyEA) (Premium Version)
    - [MemberPress](http://bit.ly/2X8dk9w) (Premium Version)
    - [WooCommerce Membership](http://bit.ly/3rWA0Ib) (Premium Version)

In the **Un-Authorized Text** settings toggle, set the un-authorized text message and text styles.

> Restrict content on all section types. For example regular, full-width and specialty divi sections

== Installation ==

1. Upload `content-restrictor-for-divi`  to the `/wp-content/plugins/` directory
2. Activate the "Divi Content Restrictor" plugin through the 'Plugins' menu in WordPress
