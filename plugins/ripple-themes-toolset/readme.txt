Import Dummy data for themes developed by Ripple Themes.

== Description ==

An easy plugin to import dummy data for themes by Ripple Themes using [Advanced Import](https://wordpress.org/plugins/advanced-import/). The plugin also adds additional functionalities to the theme from the Ripple Themes.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for Ripple Themes Toolset
3. Activate Ripple Themes Toolset plugin from your Plugins page.
4. Go to the Appearance -> Demo Data

= From WordPress.org =

1. Download ripplethemes-toolset.
2. Upload the 'ripplethemes-toolset' directory to your '/wp-content/plugins/' directory, using your favourite method (ftp, sftp, scp, etc...)
3. Activate Ripple Themes Toolset from your Plugins page.
4. Go to the Customizer (Appearance -> Customize) to add custom colour to your site.
