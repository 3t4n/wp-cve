== Description ==

LatePoint Manager is a new role for LatePoint - Appointment Booking & Reservation plugin. This Plugin Assign a new role. LatePoint Manager can control LatePoint plugin 'Pending Approval' page, edit 'Appointments', and after verify, 'Appointments' can be approved.

== Installation ==

Installation procedure:

1. Deactivate plugin if you have the previous version installed.
2. Extract "latepoint_manager.zip" archive content to the "/wp-content/plugins/latepoint_manager" directory.
3. Activate "LatePoint Manager" plugin via 'Plugins' menu in WordPress admin menu.
4. Go to the "Users"-"LatePoint Manager" menu item and change your WordPress standard roles capabilities according to your needs.
