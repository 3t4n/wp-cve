== Description ==
<p>Recent post thumbnail slider widget plug-in provides you post/page thumbnail slider that allows you to display featured image of any posts and pages in widgetized sidebar as slider effect. Great ability to customize slider with slider options, post management options.</p>
<p>User can slide recent post by give sort by date option. Even user can give specific category ids, post ids, what they want to display in slider.</p>
`Features`
<ul>
<li>Multiple slider effect, slider effect like (fade, move, random).</li>
<li>Ability to give slider speed.</li>
<li>Option to give slider auto run.</li>
<li>Ability to give Hide/Unhide next previous button and</li>
<li>Ability to give opacity to Next/Previous button.</li>
<li>Option to give Thumbnail Dimension where you can change thumbnail height/width.</li>
<li>Choose the number of related items to show.</li>
<li>Ability to sort post by date, title, number of comments or can sort randomly.</li>
<li>Post slider can order by Ascending and Descending too.</li>
<li>Ability to Hide/Unhide post title.</li>
<li>Ability to change post title color (default color is white #FFFFFF). Just write the color code, what you want.</li>
<li>Can set post title position in the slider area with Top/Bottom position.</li>
<li>Ability of all custom post types.</li>
<li>Customize post slider display feature where you can give post and category IDS.</li>
</ul>
== Installation ==

1. Upload the recent-posts-thumbnail-slider-widget folder to the /wp-content/plugins/ directory `OR` Upload from Plugins > Add New > Upload zip file.
2. Go to the 'Plugins' page of your WordPress administration area and activate the plugin.
