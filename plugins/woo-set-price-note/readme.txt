== Description ==

**Woo Set Price Note (Units, Offers, Editions)** – is the plugin that facilitate WooCommerce Shop-manager to display **units/custom texts/messages** with product price.


### The Plugin Features:
* **New feature** - Price note added to cart details.

* **New feature** - Price note added to checkout details.

* **New feature** - Separate control panel.

* **New feature** - Added new options to **include/show** price note on **Order details** and **Emails**.

* **New feature** - Facilitated **include/exclude** "price" and "separator" options for the **Order details** and **Emails** note.

* **New feature** - Added **label/texts customization** options for the **Order details** and **Emails** note.

* Supports **any product type**.

* **Price Note** - enables WooCommerce shop manager to display custom note with product price.

* **Units** - Custom note might be product price per unit ( $ 123.45/ piece ).

* **Offers** - Custom note might be festival/occasion offers ( $ 123.45 - Christmas Offer ).

* **Stock/Edition Messages** - Custom note might be edition of products ( $ 123.45 | Limited Edition ). 

* **Anything** - Whats on your mind.  



== Installation ==
* Download to your plugin directory or simply install via WordPress admin interface.
* Activate.
* Go to specific woo-commerce product page.
* Go to product data section and in general section you can find couple of extra fields.
* Set your message and set separator between price and the message.
* Update your product.
* Enjoy your message.
