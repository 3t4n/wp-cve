== Installation ==

1. Install and activate "Inkedjoy POD" Plugin at wordpress.
Before connect to Inkedjoy POD, the following conditions must be met:
(1) WooCommerce plugin has been installed and activated;
(2) Set permalinks to anything other than ��Plain�� in Settings > Permalinks;
(3) Your website must be an SSL connection.
2. Click "Connect to Inkedjoy POD" and authorize.
4. Select [WooCommerce Shipment Tracking Plugin](http://www.woothemes.com/products/shipment-tracking/) for automating order fulfillment.
