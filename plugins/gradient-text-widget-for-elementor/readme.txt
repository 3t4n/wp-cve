"Gradient Text Widget for Elementor" is a powerful and easy-to-use WordPress plugin that allows you to add stunning gradient text to your website. With this plugin, you can add a gradient effect to your text in just a few clicks, without the need for any coding skills.
This plugin is fully compatible with the popular drag-and-drop page builder plugin, Elementor, which makes it easy to add gradient text to your website's pages and posts. Simply drag and drop the widget into your desired location, choose your preferred text and colors, and you're good to go!


= How to Use Gradient Text? =

Here's how to use the plugin:

1. Install and activate the plugin.
2. In your Elementor editor, click on the "Add Widget" button and search for "Gradient Text".
3. Drag and drop the widget onto your page.
4. In the widget settings, enter the text you want to display.
5. Choose a gradient style from the dropdown menu or create your own custom gradient using the color pickers.
6. Adjust the direction and intensity of the gradient using the sliders.
Customize the text style, size, and color as desired.
7. Preview your changes in the live editor and save when you're happy with the result.
8. And that's it! With just a few clicks, you can add eye-catching gradient text to your website.

We hope you enjoy using the Gradient Text Widget for Elementor plugin. If you have any questions or need assistance, don't hesitate to reach out. Happy designing!
