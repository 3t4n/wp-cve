Required Fields can help you write your Posts, Pages without forgetting fields, if you forget something you'll be alerted about that!

== Description ==

### Required Fields

You Can Require Categories, Tags, thumbnail, title if you have forgotten something We can remind you With an simple Alert.

### The Features

* Enable / Disable The Plugin
* Set On / Off all the settings
* Set On/Off Required Fields For Drafts
* Set Custom Alerts Message
* Set Required Fields For Pages
* Set Required Fields For Posts

== Installation ==

1. Extract the `Required-Fields` folder to your `wp-content/plugins` directory
2. Activate the plugin through the admin interface

== Frequently Asked Questions ==

= Are there any settings I can adjust? = 
Yes!
Enable `Require For Posts` For Posts Alerts
Enable `Require For Pages` For Pages Alerts
Enable `Require For Drafts` For Drafts Alerts

= Can i set Custom Error Alerts? =
Yes go to `Set Error Alerts` and write your custom alerts.
