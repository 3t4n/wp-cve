== Description ==

WordPress does not allow upload ePUB and MOBI formats.
This plugin will enable the uploading of ePUB and MOBI files.

= Social Media =
[Like Us](https://www.facebook.com/eKnizky/) on Facebook and [follow us](https://twitter.com/eKnizky) on Twitter.


== Installation ==

1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Allow ePUB and MOBI formats upload' 
3. Click 'Install Now' and then activate the plugin
