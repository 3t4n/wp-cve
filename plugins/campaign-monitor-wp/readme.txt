== Description ==
= Campaign Monitor Optin Cat WordPress Plugin =

The **Campaign Monitor Optin Cat** WordPress Plugin lets you create gorgeous Campaign Monitor sign-up forms to your WordPress site in minutes.

> #### Gutenberg Compatible
> This plugin is fully compatible with WordPress 5.0's new content editor ("Gutenberg"). Simply create your optin form, then add your Campaign Monitor block to your website.


[youtube https://www.youtube.com/watch?v=9d3HPWbbn58]

= Opt-in Types =

1.   Post boxes let you add a Campaign Monitor form to a post or page using shortcodes or by automatically appending the form to the bottom of your post or page.
2.   Sidebar widgets. You can add Campaign Monitor forms to your sidebar using widgets.
3.   Popups. Capture more emails using lightbox popups.


= Features =

*   Popup targeting based on time spent on page and frequency.
*   Intuitive user interface, set up your Campaign Monitor forms in seconds.
*	Full customization options, such as the ability to change your copy, font-size, colors and more.
*   Campaign Monitor form works with responsive sites.
*   Form validation.
*   Redirect users to a "Thank You"-page after submission.
*   Build unlimited Campaign Monitor forms.
*   Analytics feature so you can learn which Campaign Monitor forms work well and which ones don't.
*   WordPress 5.0 Gutenberg support. Add Campaign Monitor blocks to WordPress' new visual editor with ease.

= Developer Friendly =

*   You can customize your Campaign Monitor forms using our "Custom CSS" feature.
*   You can inherit your theme's form styling by choosign our "No CSS" layout.

== Installation ==
1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In your sidebar, select 'Optin Cat -> Add New' to create a new table

== Frequently Asked Questions ==
= How to disable impression & conversion tracking? =
Simply add `define( 'FCA_EOI_DISABLE_STATS_TRACKING', TRUE);` to your theme's functions.php file.

== Privacy Disclosure ==

This plugin stores personal data (eg. names & emails collected using this plugin) in the WordPress database.

This plugin integrates with the WordPress GDPR privacy controls for data export & data deletion added in WordPress 4.9.6.

This plugin uses a browser cookie to track successful optins.

This plugin can be configured to connect to 3rd party service providers such as Campaign Monitor.

If you use this plugin to connect to a 3rd party, personal data may also be shared with that party.

Additional privacy policy information for 3rd party services can be found here:

[Campaign Monitor](https://www.campaignmonitor.com/policies/#privacy-policy)
