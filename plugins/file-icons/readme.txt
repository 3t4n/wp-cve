== Description ==

File Icons adds icons via CSS to your links. 
Configure and displays icons before links on your WordPress website by css (on firefox, opera, etc.)
Icons come from the famous free icons set at [famfamfam](http://www.famfamfam.com/lab/icons/ "Creative Commons licenced icons").
Localized: english and deutsch(german)

== Installation ==

1. Download, unzip and upload to your WordPress plugins directory
2. Activate the plugin within you WordPress Administration Backend
3. Go to your WordPress Administration Backend > Presentation > File Icons for configuration options.

== FAQ ==

= I have messed up my configuration =

Deactivate and reactivate the plugin again.
