== Description ==

Warning: It seems the crawler detection isn't working properly. There is a public git repo here https://bitbucket.org/ChaosKiller/cookie-consent-wp so please try to work with my to find the problem

Dutch law states that before a cookie is even placed on someone’s computer, you need their consent. The problem is that other EU countries only require you to notify them and remove the cookies if the user disagrees. The big difference is when the cookie may be placed.

This plugin will show a warning to the user that this site uses cookies. If the user clicks "yes" a cookie will be placed that he agrees. If not, he/she can't 
browse the site.

== Installation ==

1. Upload `cookieconsent.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

OR

2. Install through your wordpress site.
