== Description ==
Reservas online es el plugin la gestión de tu reservas desde tu sitio WordPress.<br> Podrás controlar en tiempo real las nuevas reservas realizadas, recibir notificaciones, cobrar de forma anticipada etc. <br>Tus clientes podrán reservar desde tu web de manera muy sencilla gracias al calendario que ofrece el plugin. <br><br>Seleccionado como mejor plugin de reservas &#9733;&#9733;&#9733;&#9733;&#9733;

== Installation ==
1. Activar plugin
2. Ir al menú de Reservas Online
