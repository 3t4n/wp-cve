== Description ==

The ELEX Hide WooCommerce Shipping Methods is a free plugin allows you to hide certain shipping methods based on shipping class, order weight, other existing shipping methods, etc. You can hide WooCommerce default shipping methods as well as third-party shipping services from DHL, USPS, Stamps.com, UPS, FedEx, EasyPost, Australia Post, and so on.

= Features & Use Cases of Hide WooCommerce Shipping Methods Plugin =
* Hide all shipping methods when Free Shipping is available.
* Hide all shipping methods when Local Pickup is available.
* Hide WooCommerce Shipping Methods based on Shipping Classes.
* Hide WooCommerce Default Shipping methods & Shipping Methods from Third Party Developers.
* Hide Shipping Methods & Options based on Various Conditions.
* Configure the Conditions based on Various Filtering Options. 
* Create and Manage an Unlimited Number of Rules to Hide Shipping Methods.
