== Description ==

Perfit es la aplicación de email marketing te permite enviar a cada contacto lo que desea recibir.

A través del plugin de Perfit para Wordpress podrás crear formularios de suscripción de forma sencilla y hacer crecer tus listas de Perfit para llegar a más contactos.

Es posible personalizar los campos y textos de los formularios, los colores y la modalidad: embebido en línea, popup con botón o popup automático.

Para poder activar el formulario debes obtener el API key de tu cuenta de Perfit. Esta funcionalidad está disponible sólo para planes pagos.

== Installation ==

La manera más sencilla es utilizar el instalador de plugins de WordPress.

Una vez instalado en tu sitio, debes ir a la pantalla de conifguración desde el menu de la izquierda 'Perfit Optin' e ingresar el API key de tu cuenta de Perfit. Esta funcionalidad está disponible sólo para planes pagos.
