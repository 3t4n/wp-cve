Блок Яндекс.Поделиться для вашего сайта на WordPress.

== Description ==

Данный плагин позволит вам вставить блок "поделиться" от Яндекс в конец каждой статьи и страницы на вашем сайте.

После активации плагина зайдите в раздел Настройки - Яндекс.Поделиться для того, чтобы настроить внешний вид кнопки и доступные сервисы.

== Installation ==

1. Распакуйте архив с плагином
2. Загрузите папку с плагином в директорию `/wp-content/plugins/` на вашем сервере
3. Активируйте плагин Яндекс Метрика в разделе Плагины в панели администрирования WordPress
4. Зайдите в раздел Настройки - Яндекс Поделиться для конфигурации вашего блока Поделиться
