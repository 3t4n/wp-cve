Быстрая и удобная вставка любых математических формул.

== Description ==

Добавьте на сайт поддержку математики. Вставляйте и редактируйте формулы через удобное окно в виде блока или строки.
Просматривайте и контролируйте формулу прямо во время набора. Изменяйте цвет формул.

= Особенности =

* Поддержка математики на сайте и в админ панели
* Удобное окно для вставки и редактирования формул
* Два типа формул для вставки: блочные и строчные
* Изменение цвета для каждой формулы
* Присвоение формуле ID или классов
* Выделение фоном формул в визуальном редакторе
* Добавление полосы прокрутки к формулам при маленькой ширине браузера

== Installation ==

Удобнее всего устанавливать прямо из Wordpress.

Но и вручную установить тоже легко. Всего 2 шага:

1. Загрузите папку `insert-math` из скачанного архива в папку `/wp-content/plugins/`
2. Активируйте плагин на странице "Плагины" в админ панели Wordpress

Все! Можете использовать математику на сайте!

== Frequently Asked Questions ==

= При загрузке страницы формулы сначала отображаются текстом, а потом в нормальном виде. =



= Какая библиотека используется для отображения математики? =

Плагин использует библиотеку MathJax. Она существует давно (с 2009 года) и поддерживает большинство браузеров.
Для ускорения загрузки из MathJax убраны все лишние расширения.

= Как именить цвет лишь части формулы? =

Цвет части формулы можно изменить с помощью LaTeX команды `\color{color}{text}`. Весь текст во вторых скобках будет покрашен в цвет, указанный в первых скобках. Пример: `\color{red}{a^2}=\color{blue}{b^2}+\color{green}{c^2}`.
