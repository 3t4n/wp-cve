== Description ==

A beautiful and easy customizable team member Gutenberg block for the new editor. Easily adjust the following parameters:

Increase avatar Size
Change border width and color
Change avatar shape from round to square
Adjust font and icon size
Change icon hover color
Social icons from Font Awesome library
Custom social icon support (JPG, PNG, SVG)
Create multiple rows
Increase offset between blocks
Set background color
Wide and narrow layout support

== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to _Plugins &rarr; Add New_.
2. Type "Kenzap Team Members" into the Search and hit Enter.
3. Locate the Kenzap Team Members plugin in the list of search results and click **Install Now**.
4. Once installed, click the Activate link.
5. You will be redirected to the Kenzap Team Members Getting Started page.

It can also be installed manually.

1. Download the Kenzap Team Members plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate Kenzap Team Members in the list and click the *Activate* link.
5. You will be redirected to the Kenzap Team Members Getting Started page.
