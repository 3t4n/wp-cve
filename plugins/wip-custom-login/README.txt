== Description ==

WIP Custom Login allows you to customize the login section of WordPress and you can replace the admin WordPress logo, set a background image and much more.

[youtube https://www.youtube.com/watch?v=w5jFYlFjlz0]

**Premium features.**

> <strong>Default Skins</strong><br>
> You can set a default plugin skin and replace the saved options.<br><br>

> <strong>Background slideshow</strong><br>
> You can create a beautiful body background slideshow and make your admin page unique.<br><br>

> <strong>Unlimited colors</strong><br>
> You can change the colors of each section, like the login button.<br><br>

> <strong>Backup section</strong><br>
> You can create a backup of plugin settings ( included the slideshow) or restore the default settings.<br><br>

> <strong>Sample data</strong><br>
> You can import the preview settings, from the backup section.<br><br>

= Installation =

Once you have installed the plugin, you just need to activate the plugin in order to enable it.

= Configuration =

WIP Custom Login will add a new admin page, where you can customize the login page.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/wip-custom-login' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
