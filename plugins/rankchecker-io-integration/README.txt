Simple integration plugin for Rankchecker.io

== Description ==

Rankchecker.io is a free Google website rank checking service for webmasters and SEO professionals.

It allows you to monitor your websites search engine rankings for various keywords on Googleâ€™s SERPs.
While other keyword tools on the market often cost hundreds of dollar per month, ours is completely free.
You can use our free keyword rank checker to check rankings for any country or language.

This plugin helps you easily verify and connect your domain with your Rankchecker.io account and lets you see your domain's top keyword rankings in your WordPress admin panel.

== Installation ==

1. Upload `rankchecker.zip` to the `/wp-content/plugins/` directory
1. Unzip plugin folder to `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit 'Appearance' -> 'Widgets' -> 'Rankchecker Badge'
