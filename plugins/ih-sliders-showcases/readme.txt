Originally Designed to work with IH Business Pro WordPress theme. This Plugin Offers Users options to create Parallax Boxes, Sliders and Showcases in their themes easily.

== Description ==

<strong>This Plugin is Designed to work IH Business Pro WordPress theme:</strong> 
<br />
If you are interested you can download the IH Business Pro theme from here - <a href="https://wordpress.org/themes/ih-business-pro/" target="_blank">IH Business Pro</a>
<br />
Once the Plugin and Theme both are activated together, you will see Following New Sections in the theme.
<br/>
<ol>
<li>Sliders</li>
<li>Showcases</li>
<li>Parallax Hero</li>
<li>Counters</li>
<li>Featured Areas</li>
<li>Team</li>
<li>Testimonials</li>
</ol>


== Installation ==

This Plugin is Generally Recommended by themes which Support it. If you theme doesn't support this plugin, you will get a notice as soon as your activate this plugin.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Once its Activated, Go to Appearance > Customize to Configure the Slider.
