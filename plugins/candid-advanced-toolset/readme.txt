Import Dummy data for themes developed by Candid Themes.
== Description ==
A easy plugin to import dummy data for themes by Candid Themes.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for Candid Advanced Toolset
3. Activate Candid Advanced Toolset plugin from your Plugins page.
4. Go to the Appearance -> Import Demo Data
Note: Make sure, One click demo import plugin is installed and activated first.

= From WordPress.org =

1. Download candid-advanced-toolset.
2. Upload the 'candid-advanced-toolset' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Candid Advanced Toolkit from your Plugins page.
4. Go to the Customizer (Appearance -> Customize) to add custom color to your site.

== Frequently Asked Questions ==
1. Where I can see the Settings?
Check all the available options inside Appearance > One Click Demo Import
