Shipping by city WooCommerce Add-on plug-in.
== Description ==

Shipping by city WooCommerce Add-on plug-in.

Features of the plugin include:

 * Provide complete solution for Shipping by city option in our E-commerce site instead on Shipping by Postal code for every Zone.
 * it include Option to configure all cities for every Zone under Woocommerce settings.
 * Simple-configuration - just install,setup and enjoy.
 
 == Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `shipping-by-city-for-woocommerce.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the menu name WooCommerce->settings.
4. Now we can see new option call "Advance Shipping Zone".
5. After click on this page we can see configuration form.
6. Now just configure & see the result on checkout page.
