== Installation ==

1.    Copy Meta SEO BenignSource in your plugin directory
2.    In your WordPress administration, go to the Plugins page
3.    Activate the Meta SEO BenignSource plugin and a subpage for the plugin will appear in your Manage menu.
4.    Go to the Settings > Meta SEO BenignSource and make the settings step by step
5.    Enjoy!
