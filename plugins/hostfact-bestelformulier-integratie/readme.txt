Eenvoudige manier om het bestelformulier van HostFact in de Wordpress website te integreren. 

== Description ==

Zie https://www.hostfact.nl/help/artikel/87/bestelformulier-integreren-in-een-wordpress-website/ voor meer informatie.

== Installation ==

1. Upload de bestanden naar de `/wp-content/plugins/` map
2. Activeer de plugin vanuit de 'Plugins' sectie in WordPress
3. Gebruik [bestelformulier url=https://www.uwdomein.nl/bestellen/?cart=1] in de betreffende Wordpress pagina
