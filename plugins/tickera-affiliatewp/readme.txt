Integrate AffiliateWP with Tickera event ticketing system for WordPress

== Description ==

By utilizing [AffiliateWP](https://affiliatewp.com/?ref=1059 "Easily create your full-featured affiliate program on WordPress") integration with [Tickera](https://tickera.com/ "Event ticketing system for WordPress") you can allow registered users on your website to share referral links to your event. That way, you can gain more visits while they are able to get a cut of each ticket that is sold via their referral link.

== Installation ==

You can find here full [installation and configuration instructions](https://tickera.com/tickera-documentation/affiliate-wp-integration/ "AffiliateWP with Tickera")
