WPUrdu Make Possible Urdu writing in WordPress editor Urdu -> اردو

== Description ==

WP Urdu is simple and efficient Urdu writing plugin, this plugin will allow you to write Urdu with in the WordPress editors. When you will write roman Urdu like Text "Mujhay Urdu Likhna Pasand Hai" in editor It will convert that to "مجھے اردو لکھنا پسند ہے". You can enable this option by clicking the button over editors.

<strong>Plugin Features</strong><br />

* Urdu Writing.
* Urdu Fonts (Nafees Nastaleeq famous Urdu font) and many more.
* Gutenberg Editor Block.
* Gutenberg Editor Sidebar Block Urdu Fonts (Nafees Nastaleeq) and Font Size.

== Installation ==

= Install from dashboard =
1. Visit 'Plugins > Add New'
2. Search for <strong>'WPUrdu'</strong>x
3. Click on the 'Install Now'
4. Activate WPUrdu from your Plugins page.
5. You will see Enable/Disable WPUrdu button over WordPress editor.

= Manual upload =
1. Upload `/wpurdu/` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently asked questions ==
= I can't find the enable/disable option. =
*  You will see Enable/Disable WPUrdu button over WordPress editor in post and page section.
