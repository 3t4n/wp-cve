Make a frame that can be easily used for the main points.

== Description ==

Enclose the main points in a frame to make them stand out.

== Installation ==

1. Upload `point-maker` folder into your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
