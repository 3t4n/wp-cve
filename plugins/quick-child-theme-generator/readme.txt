Quick Child Theme Generator is a WordPress plugin and it is helpful for creating child themes quickly.

== Description ==

Quick Child Theme Generator is a very easy, quick and user friendly plugin to create a child theme for any theme supported by WordPress CMS. It allows you to edit or add functionality to your template without having to overwrite any of the parent theme files and it will helpful for restrict losing data on WP update time. You do not need to deal with any code and configuration to create child themes. It will reduce your `time` and `effort` to create a child theme.

This plugin is providing you a way to create basic custom templates pages, those will help you to add custom code in specific pages.

== Installation ==

**From your WordPress dashboard:**

- Go to **Plugins > Add New**
- Search for **Quick Child Theme Generator** in search box
- Click **Install Now** to install **Quick Child Theme Generator**
- Activate **Quick Child Theme Generator** from your Plugins page.

**From WordPress.org (FTP)**

- Download the **Quick Child Theme Generator**
- Unarchive the **Quick Child Theme Generator** plugin
- Copy and Paste the unarchive folder to `\wp-content\plugins\` directory
- Go to Admin Panel > Plugins menu and activate the **Quick Child Theme Generator**

== Frequently Asked Questions ==

= How to use this plugin? =

Install the plugin and activate it. Then click on **Create New** link and you will reach plugin page. Then select a parent theme, fill out the fields (Theme Name is required field) and the plugin will generate a child theme for you.
