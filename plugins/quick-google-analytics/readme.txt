== Description ==
The quick solution for adding your Google Analytics Code into your header.php file - without coding.

Add the New gtag.js into your header.php file

== NEW == 
Google Analytics 4 Functions!!

= Features =

* Add UA-Code to your Header File
* IP Anonymizing Yes or No
* A switch Button between the old UA-Code and the new G-Code
