== Description ==

= This Plugin Integrates,  Automates,  Sync & displays Google Sheets with WordPress and its most popular plugins. =

WordPress events like new user registration, delete user, new post, update post, new page, user login, user logout, new database table row along with [Advanced Custom Fields](https://wordpress.org/plugins/advanced-custom-fields/) fields. This plugin also integrates [WooCommerce](https://wordpress.org/plugins/woocommerce/), [Contact form 7](https://wordpress.org/plugins/contact-form-7/), [ninja form](https://wordpress.org/plugins/ninja-forms/), [Formidable](https://wordpress.org/plugins/formidable/), [Fluent forms](https://wordpress.org/plugins/fluentform/), [WPform](https://wordpress.org/plugins/wpforms-lite/), [gravity form](https://www.gravityforms.com/), [forminator form](https://wordpress.org/plugins/forminator/) submission events with [Google Sheets](https://www.google.com/sheets/about/). 

*This plugin also **UPDATES / CREATES** WordPress posts, pages, users, WooCommerce products, WooCommerce orders, and **Database table rows** from Google Sheets.* *

## üîù KEY FEATURES ##

1. **Unlimited integrations:** Create as many integrations as you want and collect all the data you need with as many fields you want.

2. **One click Update WordPress site:** Just click your Google Spreadsheets to update WordPress Post and Page ! Awesome !!

3. **Privacy and Security:** Very Secure, just share your google sheet with service account email address, no 3rd party authentication access, the private sheet only.

4. **Drop Down & Select:** Build the integrations of your dream with its fully featured Drop Down & Select  options.

7. **One Click Google Spreadsheet Columns title Create  :**  just click to create integrated sheet column titles.

8. **Auto-sync :** Auto-sync Google sheet data and display data as table content in your post or page.

9. **Google [service account](https://cloud.google.com/iam/docs/service-accounts)  Authentication :** Spreadsheet Integration is the first & only one service account Authenticated plugin in the WordPress plugin repository .

== Installation ==

= Modern Way: =
1. Go to the wordPress Dashboard "Add New Plugin" section.
2. Search For "Spreadsheet Integration".
3. Install, then Activate it.
4. Follow the step-by-step documentation on settings page

= Old Way: =
1. Upload `Spreadsheet Integration` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the step-by-step documentation on settings page


== Frequently Asked Questions ==

=  Where does it save logs and service account credentials?  = 
 It saves those data in the installed site database. 

=  Does this plugin support Update WordPress Post from Google Sheet?  = 
 Version 3.6.0 support update WordPress Post and page from google sheet. 

=  Does this plugin support custom post type as a data source?  = 
  Professional version 3.4.0 support custom post type as a data source.You can use MetaData as a data Source in the Professional version.

=  My service account configuration and credentials is ok but why not spreadsheets are Displaying ? = 
 You have to share your service account email address with Google spreadsheets.

=  What is google service account?  = 
 A service account is a special type of Google account intended to represent a non-human user that needs to authenticate and be authorized to access data in Google APIs.
