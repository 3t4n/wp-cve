Con WP ONiAd añadirás sin problemas el tú código de seguimiento único a tu página web

== Descripción ==

Rellenando un solo campo pudes añadir el código de seguimiento en tu web en cuestión de segundos y empezar a usar ONiAd.

== Instalación ==

1. Sube los archivos del plugin a 'wp-content/plugins/wp_oniad' o instala el plugin a través del directorio de plugins de WordPress.
2. Activa el plugin en la sección 'Plugins' de tu WordPress.
3. Busca tu código único en https://oniad.com/platform en la sección de audiencias en el menú de la izquierda.
4. Escribe tu código en la sección del plugin de ONiAd que encontraras en el menú de WordPress en el backend y guarda los cambios.
