== Description ==
Creates a simple Swiper slider for all your categories. Categories will have new properties: "image", "color start" and "color end". Use those to set a custom background color and image to the slider items.
If you use WooCommerce it will pick the category images.

== Installation ==
* install [Elementor](https://be.elementor.com/visit/?bta=207645&brand=elementor)
* install the plugin
* insert the widget
* define properties
* add additional styles to your theme to style the colors or other elements
