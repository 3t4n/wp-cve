DSGVO konforme Cookie Hinweise, Datenschutzerklärung, Google Analytics und Google Fonts Lösung

== Description ==

* Dieses Plugin gibt Ihnen die Möglichkeit einen Cookie Hinweis einzublenden.
* Google Analytics kann eingebunden werden, natürlich anonymisiert und mit OPT-OUT.
* Impressumgenerator
* Datenschutz-Generator

=== Deutsch ===
Fall Sie Google Fonts verwenden, kann dieses Plugin gegen eine Nutzungslizenz, alle Scripts von Google Fonts mit einer URL unseres deutschen Proxy-Servers ersetzen.

Dieser kostenpflichtige Dienst ändert alle Links von Google Fonts zu fonts.woolink.de.
Der dort verwendete Proxy-Server speichert keine Logfiles und übermittelt KEINE Daten Ihrer Besucher an Google.
Dieser Dienst setzt eine Nutzungslizenz vorraus, die sie unter https://www.bytecity.de/fontsproxy erwerben können.
Dieser Dienst wird von bytecity UG betrieben, die auch die Plugin-Entwickler sind (woolinks)
Alle Server von uns stehen in Deutschalnd und unterliegen EU Recht.

=== English ===
If you use Google Fonts, this plugin can replace all script calls to google fonts with a german proxy server wich is operated by us directly.

The plugin modifies all urls from google fonts to font.woolink.de. The proxy server does not forward any information about your visitors to google and it does not save any logfiles.
To use this service, you have to buy a subscription which you can do here https://www.bytecity.de/fontsproxy
The servers are operated by us in germany and all EU law is in place.
This service is operated by bytecity UG which is also the contributer (woolinks)
No information or data will be forwarded to any 3rd party person or company.

== Installation ==
        
1. Upload the plugin folder to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the BC-DSGVO screen to configure the plugin
