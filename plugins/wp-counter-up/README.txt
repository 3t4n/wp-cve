== Description ==

Counter Up is a lightweight animated number counter plugin that counts up to a targeted number when the number becomes visible. It can display multiple animated counters on your WordPress site.

### Counter Up by [LogicHunt.com](https://logichunt.com)

This animated number counter is a very easy-to-use and super responsive WordPress number counter which can be used on the website to showcase the number of clients, satisfied customers, or anything related to a numerical countdown. Counter Up animates a number from zero (counting up towards it). 


= What Can You Count Up? =

*   Floats: 1.234
*   Integers: 1234
*   With commas: 1,234.56
*   Commas and dots: 12.345,67
*   With non-numeric characters: $1,234.56
*   With non-numeric characters: $1,234.56
*   Multiple countable values: 604,800 seconds in 10,080 minutes in 168 hours in 7 days

== Installation ==

= Using The WordPress Dashboard =

This is the simplest method of installing a plugin.

1. Log in to your WordPress dashboard
2. Navigate to "Plugins > Add New"  
3. Find the search form in the top-right and type "wp-counter-up"
4. To install the plugin click the "Install Now"
5. Click "Activate" to activate the plugin

= Uploading in WordPress Dashboard =

1. Download the "wp-counter-up.zip" from WordPress.org
1. Log in to your WordPress dashboard
1. Navigate to "Plugins > Add New"  
2. Click the "Upload Plugin" at the top of the screen.
3. Select "wp-counter-up.zip" from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard
