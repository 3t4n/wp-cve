== Description ==

= Jetpack Subscription Form =

This a plugin based on a feature in the jetpack by wordpress. This plugin prvides you a widget that will help to customize the jetpack subscription
box. This widget provides options to add placeholder , remove p tags surrounding the text box and submit button, also you can add custom classes to
email filed and submit button field.

== Installation ==

= Install Jetpack Subscription Form: =

1. Download Jetpack Subscription Form to your computer
2. Log into your WordPress admin
3. Click **Plugins --> Add New**
4. Click **Upload** (in the header)
5. Search your computer for **jetpack-subscription-form.zip**
6. Click **Install Now**
7. Click **Activate Plugin**
8. Click **Appearance --> widgets** to Find the widget ( Widget Name Blog Subscriptions Customized(Jetpack)).

* Note: If your computer automatically unzips your download, you'll need to re-compress it first.
* On Mac OSX: Ctrl+click the plugin folder --> Compress.
* On Windows: Right-click the plugin folder --> Send To --> Compressed (zipped) Folder.
