=== Omnipress Blocks ===

* Button: Advanced Button Block enhances the functionality of the Gutenberg editor by offering users greater flexibility. It provides additional options, including customizable icons, rel attributes, button labels, links, fill and outline layouts, custom sizing, typography adjustments, spacing and margins control, padding settings, color customization, responsiveness features, and a variety of advanced settings

* Inner column: Inner Column blocks allow easy content structuring, making visually appealing and responsive web page layouts

* Container: The Container block in WordPress simplifies content organization and design by grouping multiple column blocks. It ensures responsive layouts, efficient content editing, and is perfect for diverse uses, including multi-section pages, grid layouts, and enhanced blog posts.

* Countdown: Countdown Block allows you to effortlessly add a visually appealing countdown timer to your website. Create anticipation for upcoming events, product launches, or promotions with customizable styles and precise time settings. Engage your audience and build excitement with this easy-to-use block.

* Custom CSS: Custom CSS blocks in a FSE Block-based theme allow users to add their own styling code to customize the appearance of specific blocks without modifying the theme files directly.

* Dual Buttons: Design a Dual Button in the Gutenberg editor with unique features and capabilities for each button. You can easily insert icons, text, and various customization options between the buttons to serve as dividers and enhance the overall design

* Heading: Advanced Heading blocks offer enhanced customization options for visually appealing headings, subheadings, and separators on WordPress websites, boosting their aesthetic appeal.

* Icons: The Icon Block lets you easily add your own icons, SVG images, and graphics to your content using the block editor in Gutenberg.

* Mega Menu: The Gutenberg Mega Menu Block is a versatile container for custom menu items and nested drop-downs, offering vertical and horizontal menu layouts. You can customize it with menu icons, position settings, and various other options. It simplifies the creation of custom navigation menus within Gutenberg.

* Menu Item:  Menu blocks provide advanced customization options for creating custom Menu with features like background images, colors, content images, margins, paddings, and more..

* Slide: Slider blocks provide advanced customization options for creating custom sliders with features like background images, colors, content images, margins, paddings, and more..

* Slider: Slider blocks provide advanced customization options for creating custom sliders with features like background images, colors, content images, margins, paddings, and more..

* Tab Labels: Easily customize and label your tabs with the Tab Labels block, providing a simple and intuitive way to guide users through your content.

* Tabs Content: Organize diverse content effortlessly within each tab using the Tabs Content block, allowing you to create engaging and structured pages for your audience.

* Tabs: Streamline the overall settings for your tabs with the Tabs block, giving you control over the appearance and functionality of the entire tabbed interface on your website.

* Team: Team Blocks provides customizable blocks for teams, enabling personalized features and functionalities to enhance collaboration and productivity.

* Products Carousel: Product Carousel -- Create an engaging display of your products that scrolls horizontally, allowing you to showcase multiple items in a visually appealing way.

* Products Category: Woocommerce Product Category -- Display a list of products from WooCommerce product category with a chosen preset style, offering flexibility in how it looks with different settings.

== Installation ==

### How to install the plugin ###

1. Open WordPress admin, go to Plugins, and click Add New
2. Enter "Omnipress" in search and hit Enter
3. Plugin will show up as the first on the list, click "Install Now"
4. Activate & open the plugin's page from the admin menu

Or if needed, upload manually;

1. Download the latest stable version from from <a href="https://downloads.wordpress.org/plugin/omnipress.zip">downloads.wordpress.org/plugin/omnipress.zip</a>
2. Unzip it and upload it to _/wp-content/plugins/_
3. Open WordPress Dashboard > Plugins and click "Activate" next to "Omnipress"
4. Open the plugin's page from the admin menu
