== Descrição ==

Este plugin WordPress insere uma nova opção de integração para os formulários com o leadlovers de forma nativa. Com ele é possível adicionar uma ação após a submissão dos formulários que envia os dados do lead capturado para a plataforma leadlovers.

Com ele você poderá:
* Selecionar uma máquina, funil e sequencia para enviar os novos leads conquistados pelo seu formulário;
* Salvar dados dos lead como nome, email, telefone, data de nascimento, cidade, estado, gênero e empresa na leadlovers;
* Atribuir tags aos leads capturados;
* Visualizar um log de erros e capturas pelo painel do Wordpress;
* Ter múltiplas integrações por página;


== Instalação ==

1. Após instalar o plugin, navegue até Plugins -> Plugins instalados e ative o plugin "leadlovers forms".
2. Após ativar, navegue até Leadlovers -> Configurações . 
3. Preencha o campo "Token Pessoal" com o seu token pessoal disponível nas configurações da sua conta na plataforma leadlovers.
4. Após salvar seu token pessoal, acesse qualquer página que possua um formlário, no botão de ação de cada form, haverá um icone com a logo do leadlovers. Clique nesse link para inciar uma nova integração. Após salvar, a integração já estará ativa.
