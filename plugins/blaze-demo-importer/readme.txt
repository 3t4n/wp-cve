<h4>Plugin Features</h4>
<ul>
<li>Replicate demo you want in just a click</li>
<li>Clean previous website data - Optional</li>
<li>Automatically installs recommended and required plugins</li>
<li>Imports customizer settings, post, pages, media, navigation menus and widgets</li>
</ul>

== Installation ==
The easy way to install the plugin is via WordPress.org plugin directory.
<ol>
<li>Go to WordPress Dashboard > Plugins > Add New</li>
<li>Search for "Blaze Demo Importer" and install the plugin.</li>
<li>Activate Plugin from "Plugins" menu in WordPress.</li>
</ol>
