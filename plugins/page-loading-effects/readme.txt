== Description ==

Bring life to your website's loading pages with our new WP plugin. With its visually stunning loading animation, your visitors will be kept entertained while they wait for the page to fully load. Say goodbye to boring loading screens and hello to an enhanced user experience with our must-have plugin.

If you have concerns or inquiries, you can check us out [DATACAREPH](http://datacareph.com/)

Go and test the [DEMO](http://tobiasahlin.com/spinkit/ "DEMO") here. Thanks to @tobiasahlin this guy. 

Best coding practices was applied to this plugin by strictly following the WP Docs, this will be available on Github soon.

== Installation ==

Installation is very easy, you can either upload or install it via WP plugin section (the easy way) or Upload using FTP to wp-content/plugins/

1. Of course, Activate it via plugin section! 
1. Go to your WP Dashboard > Page Loading Effects (Just to see if it works :D)
1. The rest is just self-explanatory and You're done!
