نمایش آپارات به صورت کاملا رسپانسیو در نوشته های وردپرس
کافیه پلاگین رو نصب کنید و در طبق اسکرین شات ارائه شده در پایین روی دکمه آپارات در ادیتور وردپرس کلیک کنید، یک باکس باز می شود که باید آی دی ویدئوی آپارات رو در اون وارد کنید و گزینه ثبت رو بزنید و تمام.
آی دی ویدئوی آپارات از آدرس ویدئو قابل استخراج است به مثال زیر دقت کنید
http://www.aparat.com/v/abc
همان آی دی ویدئو است abc در آدرس بالا عبارت
و شورتکد اضافه شده با این فرمت است
[aparat id="abc"]


this plugin is used to play aparat videos in your posts. just click on aparat button on editor and enter aparat id from url.

Shortcode :
[aparat id="video_id"]


what's aparat Video ID :
For example in url 'http://www.aparat.com/v/abc' the ID is 'abc'
this Plugin is a new version of aparat plugin that support fully video responsive.

== Installation ==

1. به برگه 'افزونه‌ها' در مدیریت وردپرس رجوع کرده و بر روی 'افزودن' کلیک کنید
2. 'Aparat' را جستجو کنید
3. افزونه ذکر شده را نصب کرده و توضیحات صفحه‌ی مدیریت آن را مطالعه کنید
