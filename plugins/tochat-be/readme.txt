== Installation ==
* plugins -> Add New
* search for 'tochat.be'
* click on Install Now and then Active.

= using FTP or similar =
* unzip 'to-chat-be-whatsapp' file and 
* Upload 'to-chat-be-whatsapp' folder to the /wp-content/plugins/' directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
