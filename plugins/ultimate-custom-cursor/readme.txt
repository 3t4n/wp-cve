== Description ==

Using Ultimate Custom Cursor for customize your website cursor or mouse pointer, you will get a very elegant and unique site. You can change colors, selectors etc from this settings page.

<strong>Plugin Features</strong>

<ul>
<li>Simple installation & activation</li>
<li>Unlimited colors.</li>
<li>Lightweight.</li>
<li>Enable/Disable Option.</li>
<li>Easy Option Panel.</li>
<li>All Modern browser supported</li>
</ul>

<strong>Options</strong>
<ul>
<li>Custom Cursor : You can enable or disable custom cursor using this option.</li>
<li>Mobile Custom Cursor : You can enable or disable custom cursor using this option for mobile devices.</li>
<li>Cursor Color : You can cursor color using this option.</li>
<li>Cursor Color Opacity : You can change cursor color opacity from this option.</li>
<li>Cursor Color Hover Opacity : You can change cursor color hover opacity from this option.</li>
<li>Hover Trigger Selectors : You can change cursor hover trigger selectors from this option.</li>
</ul>

== Installation ==

1. Unzip the downloaded ultimate-custom-cursor.zip file and upload to the `/wp-content/plugins/` directory or install the Ultimate Custom Cursor by DynamicLayers plugin from WordPress repository. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Installation Instructions =

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After activate plugin you will see “Custom Cursor Options” menu under the "Settings" menu.
