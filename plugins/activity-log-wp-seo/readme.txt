== Description ==

### IMPORTANT: This extension has been merged into [WP Activity Log](https://wordpress.org/plugins/wp-security-audit-log/).

This WP Activity Log extension for Yoast SEO have been merged into the main plugin, [WP Activity Log](https://wordpress.org/plugins/wp-security-audit-log/). This means that you no longer need to use this extension to keep a log of changes that happen on your Yoast SEO plugin.

#### About WP Activity Log
[WP Activity Log](https://melapress.com/wordpress-activity-log/) is the most comprehensive real time activity log plugin for WordPress. It helps thousands administrators and security professionals keep an eye on what is happening on their websites and multisite networks.

WP Activity Log is also the most highly rated WordPress activity log plugin and have been featured on popular sites such as GoDaddy, ManageWP, Pagely, Shout Me Loud and WPKube.

### What Yoast SEO plugin changes can WP Activity Log keep a log of?

Below are some of the user and plugin changes you can keep a log of when you install [WP Activity Log](https://wordpress.org/plugins/wp-security-audit-log/):

* Yoast SEO plugin features such as readability anaylises
* Template on-page SEO title and meta description
* Changes to on-pages SEO in the Yoast SEO metabox

Refer to the [activity logs event IDs for WooCommerce](https://melapress.com/support/kb/wp-activity-log-list-event-ids/#yoast-seo) for a complete list of the changes the plugin can keep a log of.

=== Install WP Activity Log from within WordPress ===

1. Visit 'Plugins > Add New'.
1. Search for 'WP Activity Log'.
1. Install and activate the plugin.
