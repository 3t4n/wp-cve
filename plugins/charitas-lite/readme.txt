== Description ==

Charitas Lite by WPLook is a WordPress Plugin to extend the [Charitas Lite WordPress Theme](https://wordpress.org/themes/charitas-lite/).

!!! If you don't use the [Charitas Lite WordPress Theme](https://wordpress.org/themes/charitas-lite/) You don't have to use this plugin. Don't install it, it will not work for your theme.

[Charitas Theme PRO is Available Here](https://wplook.com/product/themes/non-profit/charitas-charity-nonprofit-wordpress-theme/?utm_source=wporg&utm_medium=link&utm_campaign=Charitas_lite_plugin)

= HOW TO USE THE PLUGIN =
After you activate the plugin, on the left menu will appear 3 Custom post Type ([Causes](https://themes.wplook.com/charitas-lite/causes/?utm_source=wordpress_org&utm_medium=plugin&utm_campaign=Charitas_lite_plugin), [Projects](https://themes.wplook.com/charitas-lite/projects/?utm_source=wordpress_org&utm_medium=plugin&utm_campaign=Charitas_lite_plugin) and [Staff](https://themes.wplook.com/charitas-lite/our-team/?utm_source=wordpress_org&utm_medium=plugin&utm_campaign=Charitas_lite_plugin)). You can open each of them and publish new posts.

If you want to display the [Cause List](https://themes.wplook.com/charitas-lite/causes/?utm_source=wordpress_org&utm_medium=plugin&utm_campaign=Charitas_lite_plugin), you have to Create a new page **Causes** and from **page attribute** select **Causes List**. Now you can add this page to the Menu. Go to **Menu** > Select the **Main menu** and Add the Causes page to the menu.

All custom Post Type comes with a Widget. So go to Appearance > Widgets > Add the Cause Widget to any widget area.

REPEAT the Steps for **Staff** and **Projects**.

== Installation ==

You have to search for the "**Charitas Lite**" via the **Plugins -> Add New** dialog and click install, or download and extract the plugin, and copy the plugin folder into your wp-content/plugins directory and activate.
