== Description ==

Add Smiley Button to TinyMCE.

== Installation ==

1. Upload the folder `tinymce-smiley-button` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all :)
