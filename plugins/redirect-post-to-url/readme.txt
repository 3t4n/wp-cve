== Description ==
Redirects a post (or any other any post-type) with a custom field <code>’redirect’</code> to another URL

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add a custom field <code>redirect</code> with an url value to a post. Easy.
4. View the post.
