A plugin which allows you to disable parent menu link.

== Description ==
This plugin is bug free version of [Disable Parent Menu Link](http://wordpress.org/extend/plugins/disable-parent-menu-link/).
Since it was not updated for a long time and I don't have access to make any changes so I decided to make an upgrade.

Note! It is better to deactivate Disable Parent Menu Link before installation!

== Installation ==
1. Upload the `advanced-disable-parent-menu-link` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
