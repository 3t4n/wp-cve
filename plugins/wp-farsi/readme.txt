== Description ==


#### مبدل تاریخ میلادی وردپرس به خورشیدی
+ تنها مبدل کامل تاریخ و اعداد 
+ نصب زبان فارسی در صورت نصب نبودن
+ تبدیل اعداد انگلیسی به فارسی
+ رفع مشکل هاست با زبان و تاریخ
+ مکمل و سازگار با افزونه‌های مشابه
+ بسیار سبک، سریع، پایدار و ایمن

هم در کنار این پلاگین استفاده کنید
wp-jalali, wp-parsidate 
**توجه:**
میتوانید از امکانات پلاگین‌های

== Installation ==

آموزش نصب افزونه

1.  Extract wp-farsi.zip to "wp-content/plugins" directory
2.  Go to "wp-admin/plugins.php" url then activate WP-Farsi
