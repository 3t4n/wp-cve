Wplms bigbluebutton addon is a social login integration plugin for BuddyPress.

== Description ==

This plugin integrates bigbluebutton plugin with wplms . 


1. Create,edit,delete meetings from wp editor 
2. Set reminders before meetings,send email notifications before meeting.
3. Meeting privacy
4. Specific meeting start time.
5. Adds meetings tab in profile .
6. Instructor privacy.

Tutorial On how to setup and get started : <a href="http://vibethemes.com/documentation/wplms/knowledge-base/wplms-bbb-addon/">link</a>

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Wplms bigbluebutton addon'
3. Activate Wplms bigbluebutton addon from your Plugins page. 

= From WordPress.org =

1. Download Wplms bigbluebutton addon.
2. Upload the 'wplms-bbb' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Wplms bigbluebutton addon from your Plugins page. 
= Once Activated =

1. You will see a button meetings above wp-editor from which you can add delete meetings.
2. Use [wplms_bbb token="23qdqwqd2d"] shortcode to embed meeting,where "23qdqwqd2d" is meeting token.
