== ‚úç Description ==
Advanced FAQ Manager allows you to easily create and manage frequently asked questions (FAQs) in your WordPress pages.

Using this FAQ plugin, 

* Show any number of FAQs based on category grouping
* Display FAQs in accordion style and in sync with your theme‚Äôs stylings. 
* Add FAQ Post (in the FAQ plugin behaves like a Category) and add the each questions & answers as a content within the same, there by making it easy manage a set/category of FAQs 
* Bring more detail to FAQs by adding images to the content/answers.
* Customise your FAQs using Advanced FAQ Manager with options to change the text and background colour of the FAQ title and content, and choose to display them in colors that suit your WordPress website.
* Edit/remove individual FAQs from a set easily if you find them unnecessary.
* Display the last updated date of the FAQs on top, so that the viewer could know that you have the latest updates for them.

With Advanced FAQ Manager Plugin, all you need to do is create a new FAQ category/set, add in the list of frequently asked questions as the content. Copy the displayed short-code of the post, and paste it in your web page wherever necessary.

== Installation ==
= Minimum Requirements =
* WordPress 5 or greater
* PHP version 7.2 or greater 
* MySQL version 5.6 or greater 

= Automatic installation =
1. Log in to your WordPress dashboard.
2. Navigate to the Plugins menu, and click "Add New".
3. Search and locate 'Advanced FAQ Manager' plugin.
4. Click 'Install Now', and WordPress will take it from there.

= Manual installation =
Manual installation method requires downloading the 'Advanced FAQ Manager' plugin and uploading it to your web server via your FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

== Frequently Asked Questions ==

=How can I create and manage FAQs?=
Go to FAQs ‚Üí Add New button and will create a new FAQ category for the FAQs.To manage, you can FAQ ‚Üí Edit the FAQ.

=How do I get to show the FAQs on my page?=
While an FAQ is added, a short-code is automatically generated for the FAQ set. Go to Wordpress page you need the FAQ to be displayed, edit and add this short-code into the description area of the page. FAQ will now be displayed in the page.

=How can I customise the text colour and background colour of my FAQs?=
The plugin provides a feature: Class for modifying the styling properties of the fields. 
<ol>
<li>Go to Dashboard ‚Üí FAQs ‚Üí Settings</li>
<li>From the <strong>Display settings</strong> section, set the title text colour, background colour, content text colour and background colour</li>
<li>Once the necessary changes are made, click on the ‚ÄúSave changes‚Äù button to confirm the changes</li>
</ol>

=Is FAQ Plugin for WordPress responsive?=
FAQ Plugin is completely responsive
=Is it possible to add FAQs as Widgets?=
Yes, you can add FAQs as widgets. To configure FAQs as widgets,
<ol>
<li>Go to Dashboard ‚Üí  Appearance ‚Üí  Widgets</li>
<li>Choose the "Text" widget. Drag and place the Widget in the sidebar</li>
<li>Copy paste the short-code of the required FAQ set to the content area of the text widget </li>
<li>Click on the Save button</li>
</ol>

FAQs from the respective short-code will now be displayed in the sidebar. 

=Can I edit my FAQs later?=
Yes, it is possible. To edit the FAQ please follow the below steps,
<ol>
<li>Go to Dashboard ‚Üí  FAQs ‚Üí All FAQ</li>
<li>Choose the FAQ that needs to be edited. Click on the edit button to edit FAQ accordingly</li>
<li>Save the Changes</li>
</ol>

=Is it possible to give custom colours to text and background of a particular set of FAQs?=
Yes, it's possible. 
<ol>
<li>Navigate to the FAQ that needs custom colour to be set</li>   
<li>Enable "Override General Settings"</li>
<li>You will be provided with the option to set the colour for the text and background to that particular set of FAQs</li>
<li>Save to confirm changes </li>
</ol>

=How many questions can I add in one page?=
You can add as many FAQs you need. There is no limit defined.
