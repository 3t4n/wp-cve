== Description ==

Demo Importer Plus is one of the best and powerful demo importer plugin. It allows to easily import the demo content, widgets, customizer settings along with theme settings with a single click.

In addition to this, you can also import a single page template on your existing website.

You can import the starter sites via Appearance > Demo Importer Plus.
