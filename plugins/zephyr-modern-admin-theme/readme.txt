Zephyr Admin Theme gives you the power to transform your WordPress dashboard into a more modern and beautiful dashboard, with custom colors, predefined themes, login screen customization options and much more!

== Description ==
Zephyr Admin Theme allows you to transform your WordPress dashboard and make it look more modern and beautiful. It also includes custom colors selection, beautiful predefined themes, a dark and light mode, login screen customization and much more!

= Core Features =

**Spice up your Dashboard**

* Update the look and feel of your dashboard and make it look modern.
* Zephyr Admin Theme offers you many options and transforms your dashboard into a beautiful and modern design, to change it up a bit and look forward to using the dashboard.

**Customization**

* There are many customization options to change the color and look of all the elements, including:
* Complete color selection
* A variety of predefined custom themes to choose from
* Login screen customization and custom login screen
* Set custom login screen logo, or hide the logo altogether
* Create, select and save your own theme templates to reuse in the future
* Custom fonts and font selection
* Custom WordPress dashboard logo
* Set custom login redirection URL
* Plus much more. Give it a whirl and see if you like it :)

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress 'Plugins' > 'Add New' screen directly.
1. Activate the plugin through the 'Plugins' screen in your WordPress Dashboard
1. Navigate to `Appearance` > `Zephyr Admin Theme` to change the colors and settings
1. If you experience any problems, please do not hesitate to contact me at dylanjkotze@gmail.com and I would be more than happy to help.

== Frequently Asked Questions ==
= Where can I change the colors and theme settings? =
You can customize the theme by heading to Appearance > Zephyr Admin Theme and can customize all the settings to fit your site.

= What if something doesn't look right? =
If there is anything that doesn't look right or another plugin that might be causing issues, please let me know and I will fix and improve it as soon as possible.

= Does the plugin change the frontend at all? =
No, the Zephyr Admin Theme plugin only transforms the backend dashboard and will not change anything on the frontend.

= What if I want to customize the look even more? =
I would be more than happy to help with further customizations if you contact me at dylanjkotze@gmail.com :)
