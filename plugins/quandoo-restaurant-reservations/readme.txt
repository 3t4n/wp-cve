The official Quandoo restaurant reservation plugin. Quickly integrate a button or calendar widget to start receiving reservations straight away.

## Description ##

This plugin is a user-friendly way of integrating [Quandoo's](https://www.quandoo.com "Quandoo Portal")  restaurant reservation system with any WordPress website.

If your restaurant is already a Quandoo partner, you’ll need your unique reservation key to use this plugin. If your restaurant is not yet a Quandoo partner, join [here](https://b2b.quandoo.com/qualification/ "Join Quandoo") to obtain a new reservation key and then use the plugin.

## Installation ##

Installation is easy, just like any other WordPress plugin:

1. Download the plugin from [WordPress](https://wordpress.org/plugins/quandoo-reservation "Quandoo Restaurant Reservation").
1. Use WordPress `/wp-admin/plugin-install.php` page to install the plugin or upload the plugin quandoo-restaurant-reservation directory to the `/wp-content/plugins/` directory.
1. Activate Restaurant reservation from the 'Plugins' (`/wp-admin/plugins.php`) page in WordPress.
1. Open Reservations -> Quick Start.
1. Paste your unique reservation key. If your restaurant is not listed on Quandoo yet, please follow [this link](https://b2b.quandoo.com/qualification/).
1. Click 'Quick activate' -> 'Save Changes' -> Then receive new reservations through your own website.
