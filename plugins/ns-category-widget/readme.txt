A plugin to add widget for listing Categories and Taxonomies. Extending Default WordPress Category Widget.

== Description ==

Are you frustrated by the lack of customization options in default Category widget in WordPress? Then this plugin is for you. Works in similar fashion but with several customization options. Cheers !

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
1. Select 'ns-category-widget.zip' from your computer
1. Install
1. Activate the plugin on the WordPress Plugin Dashboard
1. Place the 'NS Category Widget' into a Widget area through the 'Appearance -> Widgets' menu in WordPress.

= Using FTP =

1. Extract 'ns-category-widget.zip' to your computer
1. Upload the 'ns-category-widget' directory to your '/wp-content/plugins/' directory
1. Activate the plugin on the WordPress Plugins dashboard
1. Place the 'NS Category Widget' into a Widget area through the 'Appearance -> Widgets' menu in WordPress.

== Frequently Asked Questions ==

= Is only Category is supported in listing? =
You can list Category as well as Custom Taxonomy also.

= There is something cool you could add... =

Nice, make a post on the [Support Forum](https://wordpress.org/support/plugin/ns-category-widget/#new-post) and let us know.
