WooCommerce Plugin for accepting payment through Konnect Gateway.

== Description ==

A Konnect payment process is as follows:

1. On the seller's web page where the product or service can be purchased, the buyer clicks on the Pay button.
2. The buyer Gets redirected to the payment gateway page.
3. The buyer confirms or cancels the order.

Once the payment has been validated, the buyer is redirected to the merchant site.

== Frequently Asked Questions ==

= How to top up a Konnect account ? =

Konnect account can be topped-up using credit cards and bank wires

= How to cash-out ? =

You can simply ask for a wire transfer to your bank account.

= Is the gateway secure ? =

All Konnect Gateway requests are SSL encrypted.
