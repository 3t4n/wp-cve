WooComerce Pricing helps to create an unlimited pricing table based on a clean and flat design on Your WordPress website. 

You can add unlimited packages with unlimited functions. Variety of style support themes and table elements to display according to your needs. Perfect fit for creating a catalog of products for your store, art gallery, restaurant and much more!

The plugin has created for those who really appreciate beauty and taste. You can display your products in high quality and in best design. The plugin is perfectly combined with existing HTML and CSS, which makes it to be an excellent choice for flexible layouts. Any version of WooCommerce Pricing can be customized in accordance with your preferred preference, and can be connected to your website.

= Icon =

You can choose the icon for each field from the select menu, and there is the opportunity to select the color according to your needs.

= Plugin Capability (Features) =

* Unlimited price table anywhere.
* Can be created unlimited tables with unlimited rows and columns.
* You can display the image in each column.
* You can display icons in each line.
* Recommended column.
* You can set other colors for the each table
* Background image for the table area.
* Drag and drop items for easy sorting.
* URL button and text.
* Font size settings.
* You can create pricing in any language.
* Advanced settings for each table.
* Each type has unique customizable parameters.
* You can create clone an existing table to create a new one with just one click.
* 24/7 technical support.

== Installation ==

Here's how you install and activate the WooCommerce Pricing plugin:

Download the plugin.
Upload the .zip file in your WordPress plugin directory.
Activate the plugin from the "Plugins" menu in WordPress.
After activating video gallery choose the type of version you wish to use.

== Frequently Asked Questions ==

= 1). There are some restrictions for adding pricing table? =

* There is no limit for the amount of pricing table. 
* You can add as many table as you want. Plugin has no limitations.

= 2.) How many table may I create for WordPress website? =

* There is no limit for the creation of columns in plugin. You may create unlimited pricing tables by this plugin and by many options.

= 3.) Where can I see a working demo? =

* A fully featured [WooCommerce Pricing demo](https://total-soft.com/wp-woocommerce-pricing-table/) is now available.

= 4.) I want to change the colors for each separately column? =

You can change the colors of each column as you like. From Theme Option you can change colors and make them unique for your site.

= 5.) Where can I get support or talk with other users? =

If you are stuck, you can contact to Plugin Forum for help.

= 6.)  Can I have two different price tables on one page? =

Of course. You can place more than one price table on one page, and it will work without any problems.
