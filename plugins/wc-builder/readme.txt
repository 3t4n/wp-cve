== Description ==
WC Builder is a WooCommerce Addons for WPBakery Page Builder. WC Builder is included in this plugin to build a custom product page and archive page.

[Download Pre-build Templates for WC Builder Free and Pro](https://demo.hasthemes.com/plugins/wc-builder/wc-builder-prebuild-templates.zip)

== WC Builder Includes ==
* WooCommerce Template Builder (Basic)
* Option to select single product layout.
* Option to select an archive page template.
* Option to build an unlimited custom design for WooCommerce pages.
* 15+ Addons
* Rename Label
* Ajax add to Cart for Single Product
* Redirect to "Checkout Page" after "Add to Cart" features
* Unlimited Color and Font Variations.

== Installation ==
This section describes how to install the WC Builder plugin and get it working.

= 1) Install =

1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "WC Builder".
3. Install, then Activate it.

= OR: =
1. Unzip (if it is zipped) and Upload `wc-builder` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= 2) Configure =
1. After install and activate the plugin you will get a notice to install WpBakery Page Builder Plugin ( If allready install it then do not show notice. ).
2. Install and Activate the plugin
3. ‘WPBforWPbakery’ Category will be appear in WPBakery page builder
