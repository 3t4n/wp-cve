== Description ==

This plugin creates the additional sections on the front page in spediex themes. A easy plugin to single page and post in add shortcode data for themes by spediex Themes.  enjoy the full functionality of the Spediex For Theme Software by installing the Spediex For Theme.
Spediex For Theme is a plugin created to enhance the functionality of the WordPress theme created by themerevieweringsoftware.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for Spediex For Theme
3. Activate Spediex For Theme plugin from your Plugins page.
