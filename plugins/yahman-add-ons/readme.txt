YAHMAN Add-ons has Multiple functions.

== Description ==

Page views,Google Adsense,Analytics,Social,Profile,Table of contents,Related Posts,sitemap,SEO,JSON-LD structured data,Open Graph protocol(OGP),Blog card,Twitter timeline,Facebook timeline,Carousel Slider etc...

== Installation ==

1. Upload `yahman-add-ons` folder into your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can change the default settings and more under Settings > YAHMAN Add-ons
