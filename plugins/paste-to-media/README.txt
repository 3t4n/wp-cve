This is a simple plugin that allows uploading images by pasting them directly to the Media library.

== Description ==

When the author opens the Media library either to upload any images including featured images, this plugin gives the ability to use the paste option.
* Open Media library section on the sidebar or anywhere else where you can upload files
* Paste image or image URL from the clipboard to the page or search field
* Image is uploading to the media library after that

== Screenshots ==

1. You can copy any image or image url from any place online
2. Later just press the paste keyboard shortcut on the media section
3. You can also paste the image in the search field to get it uploaded

== Installation ==

1. Upload `paste-to-media` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Paste any image when the Media library is open on the screen
