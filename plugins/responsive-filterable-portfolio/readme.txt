== Description ==


This is a beautiful responsive portfolio with responsive lightbox plugin for WordPress blogs and sites. Admin can manage any number of videos, images, links into the portfolio grid.Admin can add, edit and delete videos, images, links from portfolio.admin can set media title to lightbox. Before add portfolio to WordPress blog, admin can preview a portfolio grid. Admin can set height, the width of portfolio media. Admin can also set the backgroud color of the portfolio.

== Installation ==

This plugin is easy to install like other plug-ins of Wordpress as you need to just follow the below mentioned steps:

1. upload responsive-filterable-portfolio folder to wp-Content/plugins folder.

2. Activate the plugin from Dashboard / Plugins window.

4. Now Plugin is Activated, Go to the Usage section to see how to use wordpress thumbnail slider.


### Usage ###

1.Use of wordpress responsive video gallery lightbox is easy after activating plugin go to Responsive Portfolio menu.

2.You can manage media by Manage media menu.

3.You can set settings for this plugin using Portfolio Settings menu.

4.You can add this portfolio to your wordpress page/post by adding this shortcode to [print_responsive_portfolio_plus_lightbox] 

OR you can add this to your theme by adding this code echo do_shortcode('[print_responsive_portfolio_plus_lightbox]'); to your theme
