Make and display your portfolio professionally. 

== Description ==

A lightweight plugin to make and display your work portfolio professionally. This is a block plugin that adds a new block in the Gutenberg page editor called "Portfolio". It supports all types of images.


**[Demos](https://bblockswp.com/demo/portfolio/)**


= Features =
- **Fully Customizable**: All the options you need to arrange the showcase to your liking are available here.


= How to use =
- First, install the Portfolio Block plugin
- Add the Portfolio Block from the block category called "Widgets" in the Gutenberg editor.
- You can change block settings from the right-side settings sidebar.
- Enjoy!

* For Installation help click on Installation Tab

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Portfolio Block**
3. Click on the **Portfolio Block** to add the block

= Download & Upload: =
1. Download the **Portfolio Block** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`portfolio-block.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **Portfolio Block** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
