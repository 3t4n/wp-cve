Shows the number of active users on the site

== Description ==

Plugin to show the current active visitors on the website

FUNCTIONS

- Plugin counter of current users on site
- Ability to change title
- Change the background and text colors
- Change the border radius
- Change elements width

== Installation ==

1. Select 'Add new' in the 'Plugins' menu to find and install The Visitors Counter Plugin.
2. Activate the plugin.

For full installation instructions, you can have a look at the [plugin web site](http://visitorcounterplugin.com/faq/).
