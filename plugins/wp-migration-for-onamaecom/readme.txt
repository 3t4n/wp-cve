== 説明 ==

お名前.com レンタルサーバーの「WordPressかんたんお引越し」をご利用いただくためのプラグインです。

プラグインを使用すると、別のサーバーで運用していたWordPressをお名前.com レンタルサーバーのサーバー上に移設することができます。


= 利用対象 =
お名前.com レンタルサーバーをご契約の場合、無料でご利用いただけます。

ご利用いただく際、お名前.com レンタルサーバーのコントロールパネル上からお引越し元のWordPressの情報を入力していただく必要があります。

なお、お名前.com レンタルサーバー以外のサーバーを利用されている場合、プラグインをインストールしても動作いたしませんのでご注意ください。

プラグインの使用方法は、お名前.com レンタルサーバーご利用ガイドをご確認ください。
