Addon Library Layouts Builder

== Description ==

Build layouts with the addons from the addon library plugin

== Installation ==

1. Make sure you have the addon-library plugin installed and activated
2. Download the plugin then upload 'addon-library-layouts-builder' folder to the '/wp-content/plugins/'  directory
   --OR--
   Download and install the plugin 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Layouts Builder section should appear in the side menu and "Addons for Wordpress" menu item in addon library.
