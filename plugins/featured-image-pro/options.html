Basic Options

  • animate

    The animate option enables animation for the grid when the grid resizes.

  • animateduration

    the amount of time it takes to animate the grid (only available in the shortcode)

  • border

    Optional border width. An integer value that defaults to 0 (none).

  • captionalign

    Align the caption text - left, center or right. Default value is center

  • captionheight

    Optional fixed caption height - defaults to none. A css value px, em, etc.

  • cat

    Filter the grid using one or more category. category may also be used.

  • category

    Filter the grid using one or more category

  • columnwidth

    Columnwidth - the width of a column in a grid. An integer value only.

  • debug

    This functionality is only available in the shortcode. It is not available in the widget.

  • debug_log

    This functionality is only available in the shortcode. It is not available in the widget.

  • debug_query

    This functionality is only available in the shortcode. It is not available in the widget.
