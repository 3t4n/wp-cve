Azigen! egy magyar webkettes oldal, a plugin ehhez készül.

== Description ==

Ez a plugin arra való, hogy minnél teljesebben ki tudd majd használni az Azigen! oldal lehetőségeit a blogodból.
Kezdetben ez most csak két admin dashboard widget lesz, az egyik egy magyar WordPress Planet, a másik pedig egy kedves kis idézetes oldal widgetje.


== Frequently Asked Questions ==

= Mire jó ez a plugin?  =

Egy jelenleg még fejlesztés alatt álló webkettes, csodás, tartalom beküldős oldal támogatást fog építeni a blogodba. 
Ez öröm-boldogság lesz azoknak, akik szeretnének látogatókat az oldalaikra csábítani, akik pedig nem, azok majd jól nem használják.

= Mi a gyakorlati haszna?  =

Jelenleg csak annyi, hogy két admin panelben lévő widgetet aktivizál. Az egyiken látszanak a magyar WordPress közösség friss hírei, a másikon pedig egy kis non-profit lélekmelegítés zajlik.

= Minek ez a plugin, miért nem építitek bele a dashboard.php-be ezeket és kész?  =

Gondoltunk rá, de úgy nem marad fejleszthető, és karbantartható. Tudod, ez az utolsó verzió amikor kézzel kell frissítened a WordPressed, ezentúl egy kattintással lehet majd installálni a dolgokat. Az első ilyen frissítésnél fellül íródnának a dashboard.php -ban beállított hírek.
Másrészt azért adjuk ki plugin formába, mert így az újabb változások telepítése szintén csak egy kattintás.


= Milyen újdonságok várhatóak?  =

Lesz távolbalátás, levitálás, szellemidézés, és vízenjárás...
 
Nos, a viccet félretéve. Az első frissítésnél bele fog kerülni egy RPC szerver cím, ami annyit tesz, hogy amikor új bejegyzést írsz, akkor küld a blogod egy jelent az azigen.hu -nak, hogy frissült. Ilyenkor jön egy fémből készült sok lábú szerkezet (web crawler), ami összegyűjti ezeket az információkat.

Ezután a következő lépés az lesz, hogy kibővítjük ezt közvetlen cikk küldéssel, API kulccsal, ami alapján azonosítani tudjuk a blogod az oldalainkon, így lehetnek személyes beállításaid, tudsz közvetlenül publikálni, stb.

== Installation ==

1. Töltsd fel a ZIP fájl tartalmát a `/wp-content/plugins/azigen/` könyvtárba
2. Aktiváld a bővítményt
