== Description ==
A easy plugin to import starter sites.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for Blockskit Import
3. Activate Blockskit Import plugin from your Plugins page.
4. Go to the Appearance -> Demo Import

= From WordPress.org =

1. Download blockskit-import.
2. Upload the 'blockskit-import' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc.)
3. Activate Blockskit Import from your Plugins page.

== Frequently Asked Questions ==
1. Where I can see the Settings?
Check all the available options inside Appearance > Demo Import
