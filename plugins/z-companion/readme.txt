Using Z Companion Import pre made websites in just one click install.

== Description ==

Z-Companion plugin is specially made for wpzita themes.This plugin boost up features in the theme.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/z-companion` directory, or install the plugin through the WordPress Dashboard > Plugins > Add New, Search for "Z Companion".
2. Install and activate plugin.

== Screenshots ==

1. Choose your desired demo.
2. Install and Activate required plugin.
3. Import website.
4. View website.
