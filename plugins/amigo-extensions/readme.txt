== Description ==
Amigo extension is to increase the awesome of our themes created by Amigo Themes. The main use of this plugin is to increase the functionality and features of amigo themes with a smooth user experience. This helps us to add a homepage and other functionality to the website. so you can build amazing websites that are suitable for business, portfolio, blogging & agency websites. You can see below listed free themes.


== Features ==
Slider Section
About Section
Service Section
Blog Section 
ect.
