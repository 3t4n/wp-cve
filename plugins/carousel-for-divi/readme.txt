The Carousel for Divi plugin allows you to transform every module in the Divi theme to a carousel. Just add a few classes and you're done.

== Description ==

### DEMOS ###

[youtube https://www.youtube.com/watch?v=XqWPOpuWOu8]

You can find more demos [here](https://demos.danielvoelk.de/divi-carousel/).

### FEATURES ###

* Transform Divi rows to a carousel

* Transform Divi columns to a carousel

* 3D Carousel [3D Carousel Demo](https://demos.danielvoelk.de/3d-carousel/)

[Official Product Page](https://danielvoelk.de/en/divi-carousel)

[Official Documentation](https://carousel.danielvoelk.de/)

### SETUP ###

Section:

*   Click on blue gear icon to access the Section Settings.
*   Go to Advanced -> CSS ID & Classes -> CSS Class. Here you add the dc-carousel class.
