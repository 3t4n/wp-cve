Yektanet Ecommerce plugin to integrate with yektanet advertising system

== Description ==

This is the official Yektanet Digital Marketing System plugin for WooCommerce that connects your WooCommerce website to Yektanet.

== Installation ==

First install and activate the plugin and then put your app id at the settings page.


== Frequently Asked Questions ==

= How do I sign up as your customer? =

Use this link to sign up: https://accounts.yektanet.com/login/?plt=yektanet

= where is my app id? =

you can find app id in your panel in yektanet after sign up.

= Is it possible that app id will change after a while? =

No. Your app id will always be the same and will not change.
