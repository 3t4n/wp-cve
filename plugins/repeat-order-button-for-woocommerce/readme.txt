Just add an "order again" button in Recent Orders list

== Description ==

This plugin is very simple to use. Just install and activate. Ready!

This plugin will add the "order again" button for all orders marked as "Completed".

Compatible with all themes and product types.

== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
2. Activate the plugin through the 'Plugins' menu in WordPress
