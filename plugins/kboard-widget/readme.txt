== Description ==

= 최다 사용자 무료 워드프레스 게시판 KBoard 위젯 입니다. =

* [무료 게시판 KBoard 플러그인 다운로드](https://www.cosmosfarm.com/products/kboard)
* [홈페이지](https://www.cosmosfarm.com)
* [사용자 커뮤니티](https://www.cosmosfarm.com/threads)
* [블로그](https://blog.cosmosfarm.com)

한국형 KBoard 위젯은 최신글, 최신댓글, 추천글, 인기글, 공지사항, 내가쓴글, 내가쓴댓글로 구성된 탭 위젯입니다.

여러 개의 위젯을 생성할 수 있으며 탭 구성도 자유롭게 변경할 수 있습니다.

플러그인 설치 후 위젯 설정 페이지에서 세팅할 수 있습니다.

PC 화면과 모바일 화면 모두에 최적화되어있습니다.

KBoard(케이보드) 게시판, 댓글 플러그인이 설치되어 있어야 합니다.

KBoard(케이보드) 게시판 추가 하시려면 공식 홈페이지를 방문해주세요.

KBoard(케이보드)는 홈페이지를 누구나 쉽게 만들 수 있도록 다양한 기능을 제공하고 있습니다.

공지사항, 캘린더, 포럼, 갤러리, 자료실, 비디오(동영상)재생, 웹진, 고객센터, 프랜차이즈, 상담게시판, 구글지도, 유튜브, 비밀글, 콘텐츠몰(컨텐츠몰), 콘텐츠샵(컨텐츠샵), 재능마켓 솔루션 모든걸 해결할 수 있습니다.

[코스모스팜 회원관리](https://www.cosmosfarm.com/wpstore/product/cosmosfarm-members) 플러그인을 사용하면 한국형 회원가입, 로그인, 소셜로그인, 정기배송, 정기구독, 정기결제, 자동결제, 유료회원, 쪽지, 본인인증, SMS, 문자보내기 등 수십가지 기능을 사용할 수 있습니다.

[워드프레스 커뮤니티](https://www.cosmosfarm.com/threads)에서 수 천개의 질의응답을 검색해서 사용 방법을 찾아볼 수 있으며, 새로운 질문을 커뮤니티에 올려주시면 답변을 받으실 수 있습니다.
