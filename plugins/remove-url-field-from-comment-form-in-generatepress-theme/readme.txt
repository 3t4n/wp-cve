This plugin helps to remove URL Field from comment form in GeneratePress Theme.

== Description ==
The plugin allows you to disable/Remove the URL/Website field from Comment form in GeneratePress Theme. The Remove URL Field from Comment Form in GeneratePress Theme plugin works in both free version and Premium Version of GeneratePress Theme.

Features:

1. Remove URL Field in Free Version of GeneratePress Comment Box.
2. Remove Website Field from Premium Version of GeneratePress Comment Box.

== Installation ==
#### From within WordPress
1. Visit 'Plugins > Add New'
1. Search 'Remove URL Field from Comment Form in GeneratePress Theme'
1. Install and Activate the plugin.

#### Manually
1. Download and unzip the file.
1. Upload the folder Remove URL Field from Comment Form in GeneratePress Theme plugin into 'wp-content/plugins/' folder.
1. Activate from the plugin page and Enjoy.
