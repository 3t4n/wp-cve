== Description ==

Con este plugin podés permitirle a tus clientes pagar con tarjetas de débito o crédito, con bajas comisiones.

Activar Getnet como método de cobro es muy fácil, sólo vas a necesitar tener una cuenta en Getnet y generar las credenciales para producción [desde acá](https://dashboard.globalgetnet.com.ar/dashboard/merchant/credentials).

Podés consultar los términos y condiciones [ingresando acá.](https://globalgetnet.com.ar/Contenido/exec/getnet/tyc/)

Para consultas, podés contactarnos al mail consultasecommerce@globalgetnet.com.ar

Encontrá toda la documentación técnica en nuestra Web:
https://centraldeayuda.globalgetnet.com.ar/hc/es-419/articles/14392615786765-C%C3%B3mo-integrar-Getnet-a-mi-tienda-WooCommerce


== Installation ==

Configurá Getnet siguiendo estos simples pasos:

1. Pedí tus credenciales de acceso ingresando [desde acá](https://dashboard.globalgetnet.com.ar/dashboard/merchant/credentials).
2. Descargá el plugin y elegí dónde guardarlo.
3. Dentro del panel de administración de WordPress dirigite a la pestaña WooCommerce y seleccioná pesos argentinos como moneda.
4. Abrí la pestaña Plugins > Añadir nuevo y hacé click en "subir plugin".
5. Seleccioná el plugin que descargaste antes y hacé click en Instalar ahora. Una vez instalado, seleccioná Activar.
6. Ahora dirigite a la pestaña WooCommerce > Ajustes > Pagos y activá Getnet como método de pago.
7. Ingresá el Título y Descripción, te recomendamos los siguientes:
   Título: Pagar con tarjeta de crédito o débito.
   Descripción: Servicio provisto por Getnet.
8. Luego completá los campos Client ID y Client Secret, con las credenciales que generaste.
9. Seleccioná Guardar cambios, ¡y listo!


Para consultas, podés contactarnos al mail consultasecommerce@globalgetnet.com.ar

Encontrá toda la documentación técnica en nuestra Web:
https://centraldeayuda.globalgetnet.com.ar/hc/es-419/articles/14392615786765-C%C3%B3mo-integrar-Getnet-a-mi-tienda-WooCommerce
