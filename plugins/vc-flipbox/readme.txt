== Installation ==

Installing this plugin as regular wordpress plugin. First you need to install Visual Composer plugin then install Flipbox Addon For Visual Composer.

This addon plugin works like other page builder addon plugin. Go to new page then click on Visual Composer builder then click on Add Element you should see Flip Box menu as Name "Flip Box" on Menu Top, click on it you will see shortcode panel.
