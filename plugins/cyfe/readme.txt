This plugin lets you see your WordPress data inside your Cyfe dashboard.

== Description ==

This plugin allows website owners to easily integrate their WordPress analytics into their Cyfe dashboard.

This enables you to monitor and share your WordPress blog stats (e.g. users, posts, comments) in real-time with Cyfe, the all-in-one business dashboard. To use Cyfe with your blog, simply install this plugin and sign up for a free account at https://www.cyfe.com.

You can track hundreds of additional metrics via Cyfe's other widgets like Google Analytics, Facebook, SERPs, Webmaster Tools, MailChimp, CSV, SQL, and more!

== Installation ==

1. Sign up for Cyfe for free at https://www.cyfe.com
2. Upload the directory called `cyfe` to `/wp-content/plugins/`
3. Go to the plugin's admin page (Plugins -> Cyfe Configuration on the left sidebar)
4. Copy/paste the API URL found there into your WordPress widget config panel inside your Cyfe dashboard
