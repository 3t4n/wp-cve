Automatically sync your data, like orders and inventory, from WooCommerce to Xero.

== Description ==

<strong>Save time and reduce errors by automatically syncing data from WooCommerce to Xero. Anytime someone places an order in WooCmmerce, have the order sync straight into Xero with any manual entry.</strong>

Realtime sync orders and customer data. Consfigure the plugin to send exactly what you want, when you want.Realtime
<ul>
	<li>Choose what order status gets sent</li>
	<li>Create new customers if they don't exist or use a generic customer</li>
	<li>Select tax codes direct from Xero</li>
	<li>Select what data from inventory is sync-ed and in which direction</li>
	<li>Much more...</li>
</ul>

<h4>Key Features</h4>

True two-way syncing between WooCommerce and Xero. Data syncing happens instantlly and without any manual editing.

<ul>
	<li>When an order is created, send to Xero</li>
	<li>Configure Data Sync Q directly from Wordpress with Xero data</li>
	<li>Sync inventory both ways (from WooCommerce to Xero or vice-versa)</li>
	<li>Payment from WooCommerce to Xero</li>
	<li>Full log history</li>
</ul>



== Installation ==

Installation is quick and easy.

e.g.

1. Upload `data-sync-q-woocommerce.php` to the `/wp-content/plugins/` directory or search Wbsync from the plugins area
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Wbsync key and save. If you don't have a key, get one from <a target="_blank" href='https://app.wbsync.com'>Wbsync</a>
