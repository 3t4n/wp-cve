Analytics code is easy to load your analytics code in all pages of your wordpress site.

== Description ==

Analytics code is simple plugin.Using this plugin you can easily load your analytics code in all pages of your wordpress site.

== Installation ==

If you have ever installed a WordPress plugin, then installation will be pretty easy:

1. Download the Analytics code plugin archive and extract the files
1. Copy the resulting Analytics code directory into /wp-content/plugins/
1. Activate the plugin through the 'Plugins' menu of WordPress
1. Enter your analytics code through option page.
