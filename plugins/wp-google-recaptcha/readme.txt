== Description ==

Added Google re-CAPTCHA in Wordpress in any form like comment form, login form, forgot password form, woocommerce form etc. 

== Installation ==

Download plugin from wordpress plugin directory.

e.g.

1. After activate plugin, create captcha key from google and update it in setting re cpatcha menu and save it. 

2. Contact form 7 Add shortcode for recaptcha [google_recaptcha] in form

== Frequently Asked Questions ==

= This plugin is work in wordpress basic form ? =

Yes, this plugin is usefull for wordpress basic form like login, register, comment, forgot password etc.

= This plugin is work in wordpress contact form 7? =

Yes, just need to add [google_recaptcha] in form .
