== Description ==

Easily add a sidebar menu to your widgetable sidebar. With this plugin you can create a sidebar menu.

Note : If you like the plugin please rate it.

== Installation ==

1. Upload `sidebar-menu-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add `Sidebar Menu ` to your widgetable area and configure the options.
