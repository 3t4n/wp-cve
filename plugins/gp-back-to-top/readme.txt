== Installation ==

1. Upload gp-back-to-top.zip to plugins via WordPress admin panel or upload unzipped folder to your wp-content/plugins/ folder
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to Settings -> GP Back To Top to setting
