== Description ==

Exclude Pages From Menu plugin provides easy option to remove any page from navigation menu.

The plugin provides option in the page edit screen in admin area which you can use to remove the page from menu display in the site front end.

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins -> Add New -> Search` and search for Exclude Pages From Menu).
* Alternatively, you can [download](http://downloads.wordpress.org/plugin/exclude-pages-from-menu.zip "Download Exclude Pages From Menu") the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* Use the `Settings -> Exclude Pages From Menu` to configure the plugin

== Frequently Asked Questions ==

= Why does the plugin not remove page from menu? =

Please make sure you have checked the plugin option "Exclude this page from menu." in the page edit screen.

Also please note it will only remove page from menu if the menu is created using either wp_nav_menu or wp_page_menu functions.
