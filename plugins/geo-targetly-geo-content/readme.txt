Change content based on visitor geolocation (country, state, city, lat/lng/radius)

== Description ==

[youtube https://youtu.be/BrgSoKrELr0]

Create geo targeted content for your website pages. Perfect for personalizing your website by country, state & city.

This plugin can allow you to deliver content from Geo Targetly or use Wordpress shortcodes to show/hide your own content. 

This plugin is made by [Geo Targetly](https://geotargetly.com "Geo Targetly")


* Paid account is required [see pricing](https://geotargetly.com/pricing "pricing")

* Get a 14 day free trial on sign up


**Features**

* Show content by country, state, city, IP address, latitude-longitude-radius

* Create multiple pieces of content each targeted at a particular geo location

* Add default content for non-targeted visitors

* Restrict content to only certain pages

* Create content in Geo Targety and inject into your pages

* Hide/show content based on location using shortcodes

* Built in analytics

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload 'geo-targetly-geo-content.php' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to [Geo Targetly](https://geotargetly.com "Geo Targetly") to create and account and setup your geo content
