== Description ==

Add Woocommerce and Easy Digital Download Menu Cart Widget to the popular free page builder Elementor.
Check out our Pro version on [Briefcase Elementor Widgets](https://briefcasewp.com/briefcase-elementor-widgets/
  ).
  
  - More Menu styles
  - More Icons
  - Customize all the Woocommerce Mini Cart. 
  - Modern Woocommerce Grid and EDD Grid to showcase your products
  - Create custom Woocommerce single product page. 
  - Create custom Woocommerce shop page.
  - Create custom EDD single product page. 
  - Create custom EDD shop page.
  - Woocommerce elements( Custom add to cart buttom, Woo price, Woo images, etc)
  - EDD elements( Custom add to cart buttom, EDD price, EDD details, etc)
  
  

== Installation ==

1. Upload `BEW Menu Cart` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!

== Frequently Asked Questions ==

= I installed the plugin but it does not work =

This plugin will only work with the Woocommerce or Easy Digital Download.
