This plugin will add a taxi booking form on your WordPress website. You will be able to manage your vehicles and orders through the back end.

== Description ==

Taxi Booking for WordPress is a complete, standalone booking system for distance-priced transportation services like taxis, limousines, shuttles, airport transfers, delivery etc.

With this WordPress plugin, you can add a Taxi Booking form on your WordPress website.

See [Taxi Booking for WordPress demo](https://tbwp.kanev.com/book-now) here.

The Pro version offers 2 service types: Address to Address distance-based fares, and Fixed fares between pre-defined destinations.

[Get Taxi Booking Pro for WordPress here](https://kanev.com/products/taxi-booking-for-wordpress).

With the Pro version of the plugin, you get multiple online payment gateways, translation capability into any language, and custom fields to collect more information or upsell products or services.
