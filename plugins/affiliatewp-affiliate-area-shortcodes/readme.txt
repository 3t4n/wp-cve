== Description ==

= WordPress Affiliate Program Plugin Shortcodes =

This plugin gives you fine-tuned control over your AffiliateWP-powered WordPress affiliate program. With access to over twenty shortcodes you'll be able to create the perfect experience for your affiliates, maximize conversions and boost sales.

= Requirements =
This plugin requires [AffiliateWP](https://affiliatewp.com/ "AffiliateWP").

= What is AffiliateWP? =

[AffiliateWP](https://affiliatewp.com/ "AffiliateWP") is the leading premium affiliate plugin for WordPress. It provides a complete affiliate management system for your WordPress website which seamlessly integrates with all major WordPress e-commerce and membership platforms. It aims to provide everything you need in a simple, clean, easy to use system that you will love to use.

= Shortcodes for your affiliate program =

The following shortcodes are provided for your affiliate program:

1. [affiliate_area_graphs].
Loads the `dashboard-tab-graphs.php` template file.

2. [affiliate_area_settings]
Loads the `dashboard-tab-settings.php` template file.

3. [affiliate_area_creatives]
Loads the `dashboard-tab-creatives.php` template file.

4. [affiliate_area_referrals]
Loads the `dashboard-tab-referrals.php` template file.

5. [affiliate_area_stats]
Loads the `dashboard-tab-stats.php` template file.

6. [affiliate_area_payouts]
Loads the `dashboard-tab-payouts.php` template file.

7. [affiliate_area_urls]
Loads the `dashboard-tab-urls.php` template file.

8. [affiliate_area_visits]
Loads the `dashboard-tab-visits.php` template file.

9. [affiliate_area_notices]
Displays the affiliate dashboard notices.

10. [affiliate_logout]
Allows an affiliate to see a log out link.

11. [affiliate_referrals status="paid"]
Allows an affiliate to see a count of their paid referrals.

12. [affiliate_referrals status="unpaid"]
Allows an affiliate to see a count of their unpaid referrals.

13. [affiliate_earnings status="paid"]
Allows an affiliate to see their paid earnings.

14. [affiliate_earnings status="unpaid"]
Allows an affiliate to see their unpaid earnings.

15. [affiliate_campaign_stats]
Allows an affiliate to see their campaign stats.

16. [affiliate_name]
Allows an affiliate to see their name as set by the "Display name" field in their WordPress profile.

17. [affiliate_name first_name_only="yes"]
Allows an affiliate to see their first name as set by the "First Name" field in their WordPress profile.

18. [affiliate_username]
Allows an affiliate to see their username.

19. [affiliate_website]
Allows an affiliate to see their website URL.

20. [affiliate_id]
Allows an affiliate to see their affiliate ID.

21. [affiliate_visits]
Allows an affiliate to see a count of visits.

22. [affiliate_conversion_rate]
Allows an affiliate to see their overall conversion rate.

23. [affiliate_commission_rate]
Allows an affiliate to see their commission rate.

== Installation ==

1. Unpack the entire contents of this plugin zip file into your `wp-content/plugins/` folder locally
1. Upload to your site
1. Navigate to `wp-admin/plugins.php` on your site (your WP Admin plugin page)
1. Activate this plugin

OR you can just install it with WordPress by going to Plugins &rarr; Add New &rarr; and type this plugin's name
