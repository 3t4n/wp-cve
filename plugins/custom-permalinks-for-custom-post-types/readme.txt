Remove base slug of Custom Post Types and change the permalink structure of Custom Post Types.

== Description ==

Custom Permalinks for Custom Post Types is a plugin for WordPress which allow you change the permalink structure of Custom Post Types.

It supports the following features:
* Remove base slug of all public Custom Post Types
* Supports multiple taxonomies for permalink
* Supports hierarchical taxonomies for permalink
* Allows to select primary term for permalink
* Allows to exclude Custom Post Types from being affected by this plugin

== Compatibility ==

This plugin is compatible with: WooCommerce, Polylang, WPML

== Privacy Policy ==

This plugin doesn't collect/store any user related information.

== Installation ==

1. Install the plugin through the 'Plugins' menu => Add New => Upload => Select the zip file => Install Now
2. Activate the plugin through the 'Plugins' menu in WordPress

Access the permalinks setting by going to Settings -> Permalinks.
