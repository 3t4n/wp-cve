== Description ==

Sequential Order Numbers for WooCommerce - Plugin for change woocommerce orders number. Create your own order number type.

= Features: =

&#9989; Custom order number text
&#9989; Possibility to use a lot of data in order number text:
&#9989; ID - Custom ID, start number can setup in option
&#9989; ID WC - WooCommerce ID, order ID that WooCommerce provide by default
&#9989; Date Time - Day, Month, Year, Hour, Minute, Second from order date
&#9989; Input - any text that you need 
&#9989; User Role - Name of role for user that order did
&#9989; Total price - order total price
&#9989; Product count - count of products in order
&#9989; Post meta - you can display any post meta from order

= How It Works: =
*check installation*


= Advanced Settings (Plugin Settings): =
* Plugin settings can be found in admin area, WooCommerce -> Sequential Order Numbers for WooCommerce

== Installation ==

= Step 1: =
* First you need to install the woocommerce plugin

= Step 2: =
* Install and activate plugin
* First of all go to Admin area -> WooCommerce -> Sequential Order Numbers for WooCommerce and check what global options you can manage
* Install the configuration that you need
* Click "Save", and the next new order will be changed according to your settings.
* Go to Woocommerce -> Orders to check how it work.
* That's it =)
