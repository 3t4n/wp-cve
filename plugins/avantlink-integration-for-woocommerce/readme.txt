== Description ==

The AvantLink Integration For WooCommerce plugin allows quick and simple integration of the AvantLink network tracking library into your WooCommerce website.

Users must have a valid, active merchant account with AvantLink to utilize this plugin.

Once activated, merchants can quickly and easily track affiliate sales and attribute commissions through their WooCommerce website.

== Installation ==

1. Activate the plugin through the `Plugins` menu in WordPress.
2. You'll see the Avantlink Logo appear in your Admin Menu; navigate to it.
3. On that page, input your Merchant ID and hit save. If you're not sure what your Merchant ID is, contact support@avantlink.com
