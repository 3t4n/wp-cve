== Installation ==
1. Upload the `hellowoofy.com` folder to the `/wp-content/plugins/` directory or install directly through the plugin installer
2. Activate the plugin through the 'Plugins' menu in WordPress or by using the link provided by the plugin installer
3. Visit the settings page in the Admin at `HelloWoofy.com` and copy your API key
