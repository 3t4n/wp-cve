== Description ==

[Cookie Notify]
Cookie Notification about using cookie files on Your website. Easy and fast configuration of position, view, colors, links, buttons and text content.

= Features include: =

* Custom position of Cookie Notify
* Custom shape - box / bar
* Custom information text
* Predefined information text to choose
* Custom link url to specified page for more cookie information
* Custom link text to specified page for more cookie information
* Custom link url to specified page with private policy
* Custom link text to specified page with private policy
* Predefined link texts to choose
* Custom accept button text
* Accept button styles to choose
* Predefined color themes
* Set cookie expiry time
* Option to accept functional cookies
* Inline CSS and Javascript that prevent number of external scripts loading (PageSpeed will be happy)
* No Frontend jQuery usage (pure JavaScript)
* Languages supported: EN, PL ... (POT file included)
* Small CSS injected in head.
* HTML and JS injected in footer.

= Usage: =

Install, customize and turn on.

== Installation ==

1. Install Cookie Notice either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Cookie Notify plugin page and configure visual options.
4. Configure content text - remember to change information links if needed.
5. Turn on notification
