== Description ==

This plugin automatically adds featured images of your posts into the RSS feed.

Features:

*   Automatically adds featured images of your posts into the RSS feed
*   Choose from all available image sizes
*   Choose from multiple image alignment options

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/mailerlite-featured-image-in-rss-feed` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Visit Settings > "Featured Image in RSS Feed" to configure the plugin

== Frequently Asked Questions ==

= Multisite supported? =

Yes of course, but please activate the plugin individually on every single blog.
