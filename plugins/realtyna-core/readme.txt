Realtyna Core plugin that adds some functionality to Realtyna Sesame Theme.

== Description ==

If you're using Sesame, Listy, IDXed or any other Realtyna theme, then you can install this plugin to get more features for your theme.

== Installation ==

For installing Realtyna Core please follow below steps:

1. Install it either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
