== Description ==

* Woocommerce payment gateway, without payment and obligations.
* Payment for the goods only after a call from the store manager.
* The payment gateway is suitable for online stores that have not decided on payment or if you offer advice on a product or service and only then offer the client payment methods.

== Installation ==

1. Make sure you have the latest version of the plugin installed. [WooCommerce](http://www.woothemes.com/woocommerce)
2. Unpack the archive and download the "without-payment-for-woocommerce" folder in your-domain / wp-content / plugins
3. Activate the plugin
