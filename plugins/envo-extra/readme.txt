== Description ==

Envo Extra add extra features and options to free [Enwoo](https://enwoo-wp.com/) theme like widgets, WooCommerce options, Elementor and Gutenberg widgets, one click demo import and much more.
Add one click demo import for [EnvoThemes](https://envothemes.com/) WooCommerce themes

== Installation ==

1. Upload `envo-extra` to the `/wp-content/plugins/` directory;
2. Activate the plugin through the 'Plugins > Installed Plugins' menu in WordPress dashboard;
3. Done!

== Frequently Asked Questions ==

= I installed the plugin but it does not work =

This plugin will only function with the [Enwoo](https://enwoo-wp.com/) theme and WooCommerce themes created by [EnvoThemes](https://envothemes.com/)
