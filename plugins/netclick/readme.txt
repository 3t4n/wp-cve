== Question fréquentes ==

= Est-ce que Netclick est gratuit ? =

Oui, vous pouvez profiter de la puissance de Netclick sans jamais rien débourser. Seules les fonctionnalités avancées sont payantes.

= Que me permet de faire Netclick ? =

Netclick vous permet de suivre efficacement le comportement des visiteurs de vos sites. Avec Netclick, vous êtes assuré d'utiliser la manière la plus simple pour analyser l'activité des internautes.

= Est-ce que je dois utiliser un logiciel sur mon serveur ? =

Non. Netclick est déjà hébergé sur ses propres serveurs. Toutes les données collectées sur votre site sont transmises à Netclick (https://netclick.io) à l'aide du code d'installation. Les données sont ensuite analysées et affichées sur votre tableau de bord sur Netclick.io.

== Installation ==

*Server Requirements:* PHP4 ou PHP5.

*Wordpress versions:* Wordpress 2.7 et supérieur.

Note: Vous devez [créer un compte gratuitement sur Netclick](https://netclick.io/register) pour utiliser Netclick.

Guide d'utilisation :

* Installer l'extension Wordpress et l'activer.
* Dans le menu administrateur, cliquer sur “Netclick” et copier-coller le code d'installation.
* Tout le reste se passe directement depuis [votre compte Netclick](https://netclick.io/register).
* Et voilà, vous avez accès en temps réel aux données de votre site.
