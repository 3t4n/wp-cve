== Installation ==
<strong>Using The WordPress Dashboard</strong>

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After activate plugin you will see "Service Box" menu at left side on WordPress dashboard and find "Service Box Options"<br />
<br />

<strong>Using FTP</strong>
1.Extract service-boxs.zip to your computer.
2.Upload the service-boxs directory to your wp-content/plugins directory.
3.After activate plugin you will see "Service Box" menu at left side on WordPress dashboard and find "Service Box Options"

<br />
<strong>How to use short-code?</strong><br />
When Service Box options setup done please copy & paste shortcode into your theme any where you want to display this service box.
