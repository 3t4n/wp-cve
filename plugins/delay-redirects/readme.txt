Simple WordPress Plugin for redirecting pages to any url with delay.

== Description ==

Super lightweight Wordpress plugin for creating redirect pages. 

= Use Cases = 
* Best for Movie Download Pages
* Best for APK Download Pages
* Best for Any download websites

== Installation ==
= Way 1: =
1. Upload plugin to the /wp-content/plugins/ directory
2. Extract the zip file.
3. Activate the plugin from wordpress plugins page. And you have done.

= Way 2: =
1. Go to plugins page in admin.
2. Click on Add New button.
3. Search for “Delay Redirects”.
4. Click install on Delay Redirects.
5. Activate the plugin
