== Description ==
私のサイトで無料で公開しているレスポンシブ メールフォーム（http://www.1-firststep.com/archives/462）のWordPressプラグインバージョンです。
Responsive Mailform for WordPress.

== Installation ==
1. プラグイン「Responsive Mailform」をWordPressにインストールする。
2. プラグインを有効化する。
3. 「設定」→「Responsive Mailform」のページで設定する。
4. 上記設定ページで生成されたコードをコピーし、メールフォームを設置したいページにテキストとして貼り付ける。
5. 自分で送信テストをしてみてサンクスページ(送信完了画面)が表示され、メールが届けばOK。

1. Install the plugin \'Responsive Mailform\' to WordPress.
2. Activate the plugin through the \'Plugins menu\' in WordPress.
3. Setting the \'Responsive Mailform\'.
4. Copy the generated code, paste to page. (text)
5. Do the send test myself.
