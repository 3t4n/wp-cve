Redirect 404 Error Page to Homepage

== Description ==

Simple redirect Wordpress plugin, all 404 error pages are redirected to homepage. Google may lower PageRank of site if there are 404 error pages(Page not found), this plugin will fix that issue and help to improve your sites ranking.

== Installation ==

The installation process is very basic:

1. Upload the unzipped plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all!

If you face issues with installation, you can read detail instructions here: http://codex.wordpress.org/Managing_Plugins#Installing_Plugins

== Frequently Asked Questions ==

= Do I have to configure anything? =
No. The plugin is ready once you install.

= What is the benefit of this plugin? =
If there are many 404 error from your site then search engines might consider your site is "broken", and drop your site ranking. You can check Google Analytics log to see your 404 error pages.

= Is this plugin working for https site? =
Yes, see this plugin in action on https://ProThoughts.com.
