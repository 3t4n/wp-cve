== Description ==

Perform a 301 redirect from an uppercase URL to the lowercase version for all non-admin URLs (that do not include a filename).

== Installation ==

= Installation =
1. In your WordPress Dashboard go to "Plugins" -> "Add Plugin".
1. Search for "LH Force Lowercase URLs".
1. Install the plugin by pressing the "Install" button.
1. Activate the plugin by pressing the "Activate" button.
1. That is it!
