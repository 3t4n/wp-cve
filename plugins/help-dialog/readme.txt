== Installation ==

You can install the Help Dialog Chat from your WordPress Dashboard or manually upload it through FTP.

= OPTION 1: Install Help Dialog Chat Plugin from WordPress Dashboard =

1. Navigate to Plugins -> Add New.
2. Search for 'Help Dialog' and install the plugin.
3. Activate the plugin in the Plugins menu.
4. Optional step: configure the plugin in the Admin menu, in Help Dialog Chat -> Need Help page.

= OPTION 2: Manually Upload Plugin Files =

1. Download the plugin file from the plugin page: help-dialog.zip.
2. Unzip the file help-dialog.zip.
3. Upload the 'help-dialog' folder to your '/wp-content/plugins' directory (do not rename the folder).
4. Optional step: configure the plugin in the Admin menu, in Help Dialog Chat -> Need Help page.
