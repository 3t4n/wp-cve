== Description ==

This plugin is an addon for WooCommerce to implement a payment gateway method for accepting **Credit Cards Payments** By merchants through **Network Merchants Payment Gateway via Authorize.Net Emulation**.

This plugin uses AIM and Uses CURL to pass data to payment gateway and hence named as **Network Merchant Inc Gateway via  Authorize.Net Emulation**


= Features =
1. Very Simple Clean Code plugin to add a Network Merchants Payment Gateway to woocommerce
2. No technical skills needed.
3. Prerequisite visualized on screenshots.
4. Adds Transaction ID, Authorization Code, Response Reason to Order Note.
5. Can be customized easily.
6. Can work with sandbox/live Merchant One Payment Gateway accounts for testing purpose.
7. This plugin currently **Supports accepting in USD**.
8. This plugin does not store **Credit Card Details**.
9. Feature to accept the type of card you like with dynamic card logo at checkout.
10. This plugin can work as authorize/authorize and capture based on plugin settings.

== Installation ==

1. Upload 'webmicro-nmi-woo-addon' folder to the '/wp-content/plugins/' directory
2. Activate 'Network Merchant WooCommerce Addon' from wp plugin lists in admin area
3. Plugin will appear in settings of woocommerce
4. You can set the addon settings from wocommmerce->settings->Checkout->Network Merchant Cards Settings
5. You can use 4111111111111111  as testing card no.
6. You can test in sandbox mode with NMI username as "demo" and NMI password as "password"


== Frequently Asked Questions ==

1. You need to have woocoommerce plugin installed to make this plugin work.
2. You need to obtain API Login ID = your account username and Transaction Key = your account password from  Network merchants INC Payment Gateway.
3. This plugin works on test & live mode of Network Merchants.
4. This plugin readily works on developmentment server.
5. This plugin does not requires SSL.
6. This plugin does not store Card Details anywhere.
7. This plugin requires CURL installed. 
8. This plugin does not support Pre Order or Subscriptions 
9. This plugin does not support Refunds in woocommmerce interface
