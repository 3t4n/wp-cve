The simple plugin to add Minicart on your WooCommerce website. 

== Description ==
The plugin will allow you to add Minicart on your website. You can change position of Minicart from plugin options. There's 5 built in cart icons for you to select from. Shorcode is now available in free version.

=== Pro Version ===
Pro version provides more icons, an option to upload custom cart icon, custom styling.
<a href="https://ahmadshyk.com/item/woocommerce-minicart-pro/">Go PRO</a>

== Installation ==
1. Minicart for WooCommerce required WooCommerce
2. Go to plugins in your dashboard and select “Add New”
3. Search for “woo-minicart”, Install & Activate it
4. Go to Woo Minicart settings page to choose minicart icon and position.
