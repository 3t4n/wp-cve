== Description ==

Show the business card in different styles on your pages or posts.

= Features =
- **Fully Customizable**: All the options you need to arrange the showcase to your liking are available here.


= How to use =
- First, install the Business Card plugin
- Add the Business Card block from the block category called "Widgets" in the Gutenberg editor.
- You can change block settings from the right-side settings sidebar.
- Enjoy!

* For installation help click on Installation Tab

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Business Card**
3. Click on the **Business Card** to add the block

= Download & Upload: =
1. Download the **Business Card** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`business-card.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **Business Card** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
