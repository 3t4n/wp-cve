Yandex Turbo модуль позволяет гибко настроить RSS 2.0. выгрузку для сервиса «Яндекс Турбо» страницы (https://yandex.ru/)

Функции
=======

* Создание нескольких выгрузок с разными настройками
* Неограниченное количество adNetwork
* Настройка типов материалов
* Настройка порядка сортировки

Установка
=========

* Сделайте резервную копию своего сайта.
* Скопируйте папку плагина 'ya_turbo/' в директорию с плагинами вашего wordpress.
* Включите "Ya Turbo" на странице управления плагинами.
* Для каждой выгрузки доступна ссылка _domain.com/?feed=ya-turbo&name=#name#. Где #name# это название вашего фида.
