Use Posts Modified Date Plugin to Add "Last Updated Date" in WordPress.

== Description ==

Use Posts Modified Date plugin and Simply show post updated date before content in single post!!!

WordPress using for many purposes. Most of the bloggers want to show last update date in blog post. Example, when you publish an article in your blog, there are often times you might need to make updates. This plugin will help you to display post updated date before content in every single post.
<blockquote><p>If you want to change the title "Last Updated on:" or want to show this bellow content or need anything modification just contact with us! And don't forget to give 5 stars ratings!!!

If you have any questions then ask here <a href="https://www.usefulblogging.com/add-last-updated-date-wordpress-blog-posts/">https://www.usefulblogging.com/add-last-updated-date-wordpress-blog-posts</a>. </p>

</blockquote>
For example, if your article <strong>Posted on June 1, 2016</strong>, and you found out five days later that there was an update to the article. You can just edit the article, and it will show "<em><strong>(Last Updated On: June 5, 2016)</strong></em>" before content that article. Still there showing original published date in post info. So you're going to show both dates published and updated to the readers. And this will help you to notice your readers that blog is always updated.

Just simply install Post Updated Date plugin and it will be automatically display Last Updated Date in your WordPress posts.


== Installation ==

1. Upload the 'post-updated-date' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You’re done!
