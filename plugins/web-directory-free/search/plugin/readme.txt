== Description ==

WooCommerce Search & Filter plugin - powerful, flexible and easy to build your custom search forms for WooCommerce. The plugin provides additional functionality to WooCommerce plugin to allow your site visitors make searching of products by categories, products attributes, tags and price.

- Price search: range slider, checkboxes, radio buttons, select dropdowns, inputs.

- Search by keywords: enter placeholder, show keywords examples, autocomplete suggestions.

- Search by categories, tags, product attributes: checkboxes, radio buttons, dropdowns list, heirarhical dropdown, multi dropdown options.

- Search by ratings: show counter, select stars color.

- Reset button and search button.

- Dynamic recount of categories, tags, attributes, prices and ratings.

- Dependent search: search fields depends on taxonomy terms

- On sale, in stock and featured checkboxes.

- up to 5 columns layout. Position and size search filters as you wish.

- Make each element own columns width.

- Limit height of lists: categories, tags, attributes you can cut by height and make scrollable radios or checkboxes.

Look at our [demo](https://www.salephpscripts.com/wordpress-search/demo/) | [Documentation](https://www.salephpscripts.com/wordpress-search/demo/documentation/)

Shortcodes:

`[wcsearch]`

`[wcsearch-products]`
