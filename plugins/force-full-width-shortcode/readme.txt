Extend any element in your page/post to the full browser width with simple shortcode

== Description ==

Extend image (or iframe or other elements) in your page/post to the full browser width with simple shortcode

 [jg-ffw] Your element [/jg-ffw] 

Demo: http://force-full-width-shortcode.jacobgoh.com/

** Only works when the page/post does not have sidebar

GitHub: <a href="https://github.com/jacobgoh101/force-full-width-shortcode">https://github.com/jacobgoh101/force-full-width-shortcode</a>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `force-full-width-shortcode.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
