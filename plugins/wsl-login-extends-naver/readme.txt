== Description ==

이 플러그인은 Wordpress Social Login 플러그인 에서 naver 로그인과 kakao 로그인을 할수 있는 플러그인입니다.

( 반드시 Wordpress Social Login 플러그인을 설치해야 합니다. )

naver 로그인 및 kakao 로그인을 설정하는 방법은 아래 url에 자세히 설명해 놓았습니다.

사용팁 : http://sir.co.kr/gnucommerce_tip/6


== Installation ==

이 섹션은 wsl-login-extends-naver를 설치하여 사용하는 방법에 대해 설명하고 있습니다.

1. Wordpress Social Login 플러그인을 설치하지 않았다면, 설치합니다.
2. 관리자 `플러그인` 메뉴에서 wsl-login-extends-naver를 활성화 합니다.

This section describes how to install the plugin and get it working.

1 . It requires Wordpress Social Login plugin.  
2 . Activate the plugin through the 'Plugins' menu in WordPress
