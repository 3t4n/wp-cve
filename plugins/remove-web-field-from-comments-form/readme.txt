This plugin adds a new setting in the last position of Settings > General, this option will allow you to check if you want to remove the web field from the comments form.

== Description ==

“Remove Web Field From Comments Form” is a small plugin that allows you remove the web field from all the comments forms so you will never have to deal again with unwanted website links. You can activate it by checking the checkbox at the bottom of Settings > General. This plugin wont work if the value is already defined elsewhere to avoid errors.

== Installation ==

The Plugin can be installed in two ways.

= Manual instalation =

1. Download and upload the plugin files to the /wp-content/plugins/remove-web-field-from-comments-form directory from the WordPress plugin repository.
2. Activate the plugin through the ‘Plugins’ screen in WordPress admin.
3. Go to Settings > General and scroll down to see the new option to configure it.

= WordPress interface instalation =

1. Open the plugin Interface in the wordpress admin and click on “Add new”.
2. In the Search box enter “Remove Web Field From Comments Form” and hit Enter.
3. Click on “Install” to install the plugin.
4. Activate the plugin
5. Go to Settings > General to configure it.
