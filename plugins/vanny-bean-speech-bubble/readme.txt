Allows you to enter captions inside of speech bubbles on top of images. 

== Description ==

Inspired by the wonderful expressions babies make, this plugin allows you to impose speech bubbles on top of images in order to add captions.
It does not have to be a baby or anything about kids for that matter; it works wonderfuly for images with lots of expressions.

Please feel free to visit <a href="http://www.vannybean.com">VannyBean</a> for any questions or more information.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Download and unzip the plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In a blog post or page you should now see a caption button on the rich text editor menu (far right).
1. Click the icon, then click on top of the image where you want the caption to appear.
