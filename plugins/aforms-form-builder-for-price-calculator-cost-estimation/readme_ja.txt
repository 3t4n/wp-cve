== Description ==

**AForms -- Form Builder for Price Calculator & Cost Estimation** はお客様の興味を引き付け、商品・サービスへの理解を促し、申込に繋げるフォーム作成ソフトです。
見積シミュレーションフォームやカスタム注文フォームを作れます。

== Feature List ==

- javascriptプログラムが60KB(gzipなし)。違和感ゼロのロード
- 100%レスポンシブ・モバイルフレンドリー
- ブラウザサポート：Chrome、Firefox、Edge、IE11、iOS Safari、Android Chrome
- ウィザード形式とフロー形式のナビゲーション
- 単価の動的な計算（+ - * / など）
- リアルタイムの見積表示
- インスタントバリデーション（リアルタイムバリデーション）
- オンラインフォームビルダー
- 「ラベリングロジック」で分かりやすい条件分岐
- 柔軟な税の取り扱い（内税・外税、複数税率、端数処理）
- 確認画面に対応
- サンキューメール、通知メール

[拡張ソフト](https://a-forms.com/ja/category/ext/)を使って見積フォームをグレードアップすることもできます。

- PDF：PDF見積書を自動発行
- スタイル：フォームの見た目とレイアウトをカスタマイズ
- アップロード：フォームからファイルをアップロード
