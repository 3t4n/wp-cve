=== Политика конфиденциальности для сайта. Согласие под формами Contact-Form 7 ===
Contributors: aleksinSky
Tags: политика конфиденциальности, 152ФЗ, 152 Федеральный закон
Requires at least: 4.6
Tested up to: 6.0
Stable tag: 1.29

Adds a page with a privacy policy to the site. Also, he adds to the Contact Form 7 consent to the processing of personal data. The plugin is intended only for Russian sites.

== Description  ===

Добавляет на сайт страницу с политикой конфиденциальности. Также, добавляет под Контакт Форм 7 согласие на обработку персональных данных. Плагин предназначен только для русскоязычных сайтов. При входе пользователям показывает оповещение о использовании сайтом cookies. Подходит для 152ФЗ. 
По вопросам работы плагина пишите на это почту: webmaster@aleksinsky.ru. Через онлайн консультант на нашем сайте консультации не даем.

==  Installation  ==

1. Установите плагин. 
2. Заполните обязательные данные о компании.
