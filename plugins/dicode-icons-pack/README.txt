== Description ==
Dicode Icons Pack by Designinvento provides ability to add custom font icons to your website from all time top icon libraries. You can use these icons with any page builder. plugin offer ability to activate selective libraries from and you can search your desired icon from dedicated library search, Dicode Icons Pack is fully integrated with **Elementor** Icon Manager

== Installation ==


1. Go to Wp admin -> plugins -> click add new and search **Dicode Icons Pack**
2. Install and activate plugin
