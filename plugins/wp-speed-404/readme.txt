WP Speed 404 can either suppress or log any 404 errors affecting your site under wp-content, wp-admin and wp-includes
(configurable). 404 errors are generally very 'expensive' on WP sites as they cause WP to run on every attempt, slowing
down your site and causing additional load on your web hosting environment.

== Description ==

404 errors are generally very 'expensive' on WP sites as they cause WP to run on every attempt, slowing
down your site and causing additional load on your web hosting environment.

WP Speed 404 can either suppress or log and email any 404 errors affecting your site under wp-content, wp-admin and
wp-includes (configurable).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-speed-404` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the 'Speed 404' screen to configure the plugin
