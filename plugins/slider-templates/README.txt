== Description ==

Automatically import Slider Templates from slider-templates.com to your website with just one-click!
This plugin is created for Slider-Templates.com users or Codeless Themes customers.

If you are a Codeless theme customer you can automatically install templates without the need of ST account.

To get full-access to templates you should have a ST account. With a FREE account you can download only 1 slider!
Get premium account on slider-templates.com to access all sliders and import them automatically to your website.

= Slider-Template Service =

This plugin uses an external service, <a href="https://slider-templates.com">slider-templates.com</a> but only in cases you consent the connection to Slider-Templates Server.

<a href="https://slider-templates.com/wpautoterms/terms-and-conditions/">Terms & Conditions</a> - <a href="https://slider-templates.com/wpautoterms/privacy-policy/">Privacy Policy</a>

= Privacy Policy =

This plugin collects users information (IP, Location, Cookies) when the User consent it with the 'CONNECT' button or Sign In using
Slider-templates.com Email & Password.

Anytime users can block (disconnect) the plugin from communicating with Slider-Templates Server.

When connecting or Sign In, plugin automatically connects with server to get templates and to identify which templates is allowed to be downloading from that user using the actual user level.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `slider-templates.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Connect with Slider-templates.com
4. Sign In if you are not a Codeless Theme customer.
