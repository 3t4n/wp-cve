== Description ==

<a href="https://crestaproject.com/cresta-help-chat-demo-page/" taget="_blank">Plugin Demo</a> | <a href="https://crestaproject.com/downloads/cresta-help-chat/" taget="_blank">Plugin Homepage</a>

With <strong>Cresta Help Chat</strong> you can allow your users or customers to contact you via WhatsApp simply by clicking on a button. Users may contact you directly in private messages on your WhatsApp number and continue the conversation on WhatsApp web or WhatsApp application (from mobile).
You can use the floating button which is positioned at the bottom right of the site or a button inside post or pages using the <strong>[cresta-help-chat text="Need Help?" icon="yes" position="top"]</strong> shortcode.
Cresta WhatsApp Chat is compatible with WooCommerce and WPML.

<strong>Some features</strong>
<ul>
	<li>Add the button to open WhatsApp Chat at bottom right of your site</li>
	<li>Use the shortcode <strong>[cresta-help-chat]</strong> to add the WhatsApp button in your posts, pages or custom post type</li>
	<li>Choose which pages to show the WhatsApp Chat button</li>

== Installation ==

1. Unzip cresta-whatsapp-chat.zip
2. Upload the folder 'cresta-whatsapp-chat' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress Dashboard
4. Go to your WordPress Dashboard under "Main Menu -> Settings-> Cresta Help Chat" to set the options


== Frequently Asked Questions ==

= Where I find the options page? =
After installing the plugin, using the WordPress Admin menu, go to "Settings-> Cresta Help Chat"

= Can I hide the WhatsApp Chat box in a few pages/posts? =
Yes, just edit the page/post you want to hide the box, find the metabox called "Cresta Help Chat" and check the box "Hide Cresta Help Chat in this page".

= Will my WhatsApp number be visible to all users? =
Yes, every user of your website will be able to see the number of WhatsApp you are using (for the way WhatsApp works). For this reason we advise you to not use a WhatsApp number that you want to keep private.
