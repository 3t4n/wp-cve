== Description ==
A very simple plugin for the maintenance mode. All You have to do is turn it on, so that the logged in users and those logged in without permission to manage the website see the message about the technical break. 

== Installation ==
1. Upload `pryc-wp-maintenance` dir to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
