== Description ==

Easy UPI Payment plugin (for WooCommerce ) helps you accept payments online from your Customers instantly & directly into your bank account (without paying any Commission or Charge : 100% Free) via UPI applications (like: BHIM App, Google Pay, PhonePe, Paytm, Whatsapp Pay, or via any other UPI based application).

== Installation ==

yoursiteurl/wp-admin > Plugins > Add New > Upload > Install > Activate > WooCommerce > Settings > Payments > Easy UPI Payment > Setup
