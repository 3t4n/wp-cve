Perfectly Integrates latest font awesome icons with your wordpress site as well as adds an Icon insert button in Tiny MCE wordpress editor.

== Description ==
Perfect Font Awesome Integration plugin seamlessly adds latest font awesome icons to your wordpress site. 

How to use this plugin ? 

Install the plugin.
Go to Posts > Add New
In the visual editor you will find [FA ICON] tab.

Following settings are available : 

1) Fa Icon Code:(Eg: fa fa-pencil)
List of all codes are available on : 
https://fontawesome.com/icons

2) Color:(Eg: #333333) leave blank for default

3) Size : Choose from 2x to 5x larger.

4) Alignment : If your icon is off from the text, use this option.

5) Border: Gives a border to the icon.

6) Float: Make the icon float left or right.

7) Animate: Give a spinning animation to font awesome icon. Useful in loading type icons.

8) Rotate/Flip: Rotate the icon 90 to 270 degrees, Flip vertically and Horizontally.


Press ok to insert the shortcode in post.

== Installation ==
Install Perfect Font Awesome Integration from Wordpress.org.

Start using it through the [FA ICON] tab in wordpress Tiny MCE Editor.
