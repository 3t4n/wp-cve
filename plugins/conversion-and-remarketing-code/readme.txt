== Description ==

Easily place tracking code in the header or footer of your entire site or individual posts/pages. This is ideal for embedding remarketing and analytics tags that need to be on every page, as well as conversion code that needs to only be on specific pages. Also supports CSS and JS code.

== Installation ==

1. Upload folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. For embedding code on every page, go to Settings->Remarketing Code.
1. For embedding on individual pages, go to the "Conversion and Remarketing Code" section when editing a page/post.
