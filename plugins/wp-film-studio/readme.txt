WP Film Studio is a WordPress Movie Maker/Production Plugin. 

== Description ==
WP Film Studio is a WordPress Movie Maker/Production Plugin  which allows you to display unlimited movies and trailers in your website. This plugin facilitates you with multiple options to attach the details & archive pages of the movies to your website. This plugin also supports Elementor page builder. Using the Elementor addons you will be able to place the movies or trailers anywhere on the pages according to your sweet will. Don't worry, these actions take no time. 

Remember, you are dealing with Entertainment. So, design and outlook is the most important thing what you have to be very careful of. And we care a lot about those things, whatsoever.

== Installation ==
This section describes how to install the WP Film Studio -  WordPress Movie and Trailer plugin and get it working.

= 1) Install =

= Modern Way: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "WP Film Studio".
3. Install, then Activate it.

= Old Way: =
1. Unzip (if it is zipped) and Upload `wpfilm-studio` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= 2) Configure =
1. After install and activate the plugin you will get a notice to install CMB2 Plugin ( If already install it then do not show notice. ).
2. To install the plugin click on the "Button" Install CMB2.
3. After install and activate the both plugin you will get a Dashboard Menu(WpFilm).
