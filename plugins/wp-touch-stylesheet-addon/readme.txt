Adds a custom mobile only stylesheet for WPtouch.

== Description ==


Custom Stylesheet Extension for WPtouch is a plugin that allows you to add a custom stylesheet to your WordPress theme 
that will control the css for mobile only visitors. You can change the location of the stylesheet in the admin. You can use the default stylesheet that is created once you activate the plugin.

** Note **
Default stylesheet location: 
/wp-content/uploads/wp-touch-stylesheet-addon.css


== Installation ==

Installation of this plugin is simple

* Upload the plugin to the plugins folder or install from the admin
* Activate the plugin!
* Voila, you're up and running.
* If you want to change the location of the stylesheet go to Settings > Custom Stylesheet WPtouch
