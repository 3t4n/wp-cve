Copy term of any type with a click!

== Description ==

This plugin allows you to clone terms of any taxonomy. Duplicate any custom taxonomy term, including built-in categories and tags. Copy taxonomy term of any type with a click - plugin adds a quick links to each term. 

== Installation ==

Use WordPress' Add New Plugin feature, searching "Duplicate Taxonomy Term", or download the archive and:

1. Unzip the archive on your computer
2. Upload `duplicate-term` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
