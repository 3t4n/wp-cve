Tayori Form Plugin For WordPress.


== Description ==

Enables you to put inquiry button in your wordpress page and send inquiries.
http://tayori.com

Tayoriお問い合わせボタンを簡単に設置することができます。
http://tayori.com

== Installation ==

1. Upload files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

1. `/wp-content/plugins/`にファイルをまとめてアップしてください。
2. 'プラグイン'メニューからTayoriを有効にしてください。
