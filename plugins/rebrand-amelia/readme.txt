== Description ==
Amelia is an automated booking specialist that allows customers to book appointments or make payments with a simple plugin. Thanks to Rebrand Amelia, you can remove any Amelia branded colors or messaging and replace it with your own. Change the plugin’s name and description and make the calendar look like it’s native to your website, among other features. 

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 5.6 or greater

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress(This will add new Sub Menu in admin panel's side menu name Rebrand` under Amelia menu).
3. Go to Amelia > Rebrand
