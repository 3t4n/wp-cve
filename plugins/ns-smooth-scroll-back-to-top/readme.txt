This plugin allow to add a smooth scroll back to top button on your theme.
Install plugin, click on Active button and your Back to top button is ready.

== Description ==

Install and customize it! You can go to NS Back To Top and choose color, icon settings and much more.

You can set:

* change position of back to top (**NEW**)
* text o FontAwesome to use like arrow
* background color
* background color hover
* text color
* text color hover
* border color
* border color hover
* the speed to use for back to top

*We test our plugins with many other. If you find incompatibility with one specific plugin, please inform us at report@nsthemes.com.*

https://www.youtube.com/embed/10yyOvDubUw

== Installation ==

1. Upload the entire 'ns-smooth-scroll-back-to-top' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Settings' -> 'NS Back To Top'


== Frequently Asked Questions ==

= Can i customize it? =

Yes you can customize it from setting.

= Button back to top is always visible? =

No the button show when you scroll down in your site.
