Display user meta information for the currently logged in user with a simple shortcode.

== Description ==
Use this handy little plugin to display various user meta information for the currently logged in user with a simple shortcode. Array data in user meta are displayed as comma-separated list.

1. Shortcode list page.

== Installation ==
This section describes how to install the plugin and get it working.

1. Download and install the plugin from WordPress dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Open 'User Meta' in menu and pick your meta tag from the list. (Click on the selected shortcode to copy to Clipboard.)
4. Paste the shortcode into your post or page.
