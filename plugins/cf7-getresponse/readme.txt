Simple way to integrate GetResponse mailing lists to Contact Form 7. Save your subscribers in a proper mailing list.

== Description ==

Integrate Contact Form 7 with [GetResponse](https://renzojohnson.com/contributions/contact-form-7-getresponse-extension). Automatically add form submissions to predetermined lists in GetResponse, using its latest API. This GetResponse Extension supports multiple mailing lists and API Keys.

= Key Features =

* Easy to use
* Use a different mailing list per contact form
* Single opt-in - no confirmation email send to subscribers
* Double opt-in - confirmation email send to subscribers
* [Opt-in checkbox](https://renzojohnson.com/contributions/contact-form-7-GetResponse-extension/GetResponse-opt-in-checkbox) - add the chance to opt-in/opt-out
* Support for [custom fields](https://renzojohnson.com/contributions/contact-form-7-GetResponse-extension/GetResponse-custom-fields)
* Latest GetResponse API V3
* Constantly updated

= Requirements =

1. Self hosted WordPress.org installation (4.0 or higher)
2. Contact Form 7 (4.2 or higher)
3. GetResponse account




== Installation ==

1. Upload the entire '[contact-form-7-mail-chimp-extension](https://renzojohnson.com/contributions/contact-form-7-GetResponse-extension)' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Once the plugin is active, you will see a new 'tab' inside Contact Form 7 admin interface.

You will find '[GetResponse: Subscriber List Details](https://renzojohnson.com/contributions/contact-form-7-GetResponse-extension)' menu in your Contact Form 7 editor.

For basic usage, you can also have a look at the [Contact Form 7 GetResponse Extension homepage](https://renzojohnson.com/contributions/contact-form-7-GetResponse-extension), you can also have a look at the [Contact Form 7 homepage](http://contactform7.com).
