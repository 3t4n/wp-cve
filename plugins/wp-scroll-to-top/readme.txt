Scroll To Top plugin does the job with just one click a button appears when a person scrolls to the middle of the page to be clicked on.


== Description ==

 Scroll to top plugin makes the scrolling of a page easier. It appends an icon at bottom of the page which attracts user attention. When the user click on this button it smoothly scrolls to the top of Webpage. Wp Scroll To Top plugin gives you custom settings which no other plugin gives.


* Fluid/responsive icon.
* Compatible with all Wordpress themes.
