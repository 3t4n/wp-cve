== Description ==

Post thumbnail and post list plugin will help you to Show your recent published wordpress post with thumbnail and lists style. Show your post with individual category. No coding required, Mobile friendly and fast performance. You can use this plugin any of your wordpress sidebar widget.


** Plugin Features **
This plugin has some awesome options, you customzie from widget settings.

* Post categories
* Post Count
* Post order
* category Posts
* Post date, comments, author
* Two Post Layout
* 3 Flat Colors



== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>WP Post Thumbnail & Post List Plugin</strong>" activate it.<br />

After activate plugin you will see "WP Post Thumbnail & Post List Plugin" in widget list. Just drap and drop widget to your sidebar<br />
