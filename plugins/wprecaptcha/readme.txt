Add Google reCaptcha to WordPress forms. Easy to add, advanced security for your forms.

== Description ==

WP reCAPTCHA protects your site from spam and abuse. It uses advanced risk analysis techniques to tell humans and bots apart.

WP recaptcha is the only WP Plugin curretly offering reCAPTCHA v3 for Gravity Forms.

WP recaptcha Supports all Google reCAPTCHA versions:
* Version 3
* Version 2 - checkbox
* Version 2 - invisible

WP recaptcha is very simple to use! just select the page that contains the form - WP recaptcha will automatically generate the required recaptcha code.

[View Demo](https://demo.wprecaptcha.com/)

== Installation ==

1. If You have any Google reCAPTCHA integrations for your forms - please disable the integrations.
2. If you have any other Google reCAPTCHA plugins - please deactivate or uninstall the plugins before use wpRECAPTCHA.
3. Upload the `wr-recaptcha` folder to the `/wp-content/plugins/` directory.
4. Activate the plugin via the 'Plugins' menu in WordPress.
5. Go to WP reCaptcha "Settings" tab
