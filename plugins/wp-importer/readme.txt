= Create Unlimited Pages Through The CSV File =

WP Importer plugin provide an option to create unlimited posts/pages through the CSV file. Also plugin does provide an option to create any custom post type pages in bluk.

CSV (Comma Separated Values) is a file format for data storage which looks like a text file. The information is organized with one record on each line and each field is separated by comma. 

Using csv file, user can easily manage pages without do any manual task from admin. On a click "Import Now" button, your post will be create automatically, It will save your valuable time and money as well :)

== Installation ==

Installing the plugin:

1.  Unzip the plugin's directory into `wp-content/plugins`.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  The plugin will be available under Tools -> WP Importer on
    WordPress administration page.
