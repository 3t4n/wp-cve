== Description ==
Allows you to meet the minimum compliance requirements for the [EU cookie legislation](http://ec.europa.eu/ipg/basics/legal/cookies/index_en.htm#section_2) introduced in 2011 without sacrificing functionality.

Features:

* Displays easily dismissed cookie consent notification for non-logged in users.
* Colour scheme and notification positioning options.
* Adjust the message visitors see when they're being notified that your site uses cookies.
* Optionally link to your own cookie policy page.

== Installation ==
1. Download the zip file and decompress it.
2. Upload "eu-cookie-consent" to the "/wp-content/plugins/" directory.
3. Activate the plugin through the "Plugins" menu in WordPress.
4. Click *Cookie Consent* in the admin menu.
