== Description ==

**Cart Link for WooCommerce** is a simple yet powerful plugin which will skyrocket your sales in no time. Create, customize and save the direct cart links adding the predefined products setup to the cart. Share the cart links with your customers and make their **carts be automatically filled with the selected items**. Set prices and quantities of the products added to the cart to create special offers and increase the sales.

= How the custom Cart Link works =

Create a new campaign and choose the product or products combination which should be added to the cart once the customer clicks the special cart link. Set the products' details including their variations, quantity, price and apply the discounts if needed. Decide what should happen next and which page the customers should be redirected to after the link is clicked.


= Possible Cart Link Usage Examples =

* Use the custom cart link for the call-to-action buttons on your site and landing pages
* Send the promo links with the products discounts already applied to your newsletter subscribers
* Create Black Friday & Cyber Monday special deals available only to a specific audience
* Create user friendly custom Add to Cart link in WooCommerce without adding any additional parameters
* Prepare the individual wholesale and retail pricing offers for your customers
* Promote the ready cart setups and product combinations in social media campaigns
* Place the products set links in the blog posts
* Create a direct checkout link to speed up the checkout process
* Combine the products into special-priced packages and boost your sales with cross-selling

and this is just the beginning!


= What you can do with Cart Link for WooCommerce - Key Features =

* Create unlimited Cart Link campaigns
* Generate the individual cart links which once clicked will fill the cart with the selected products
* Customize the direct cart link URL
* Change or adjust the products' quantity and price
* Add the analytics to the links - plugin passes the UTM parameters after redirection
* Use any type of products including their variations
* Clear the previous cart content and remove the products which customer added to the cart before hitting the direct cart link (optional)
* Choose the page your customers should be redirected to after clicking the cart link
* Edit or modify the already existing campaigns
* Add, duplicate and delete easily the campaign products with quick action buttons

Give it a try and find out that our Cart Link for WooCommerce plugin is exactly what you are looking for!


== Installation	 ==

This plugin can be easily installed like any other WordPress integration by following the steps below:

1. Download and unzip the latest zip file release.
2. Upload the entire plugin directory to your /wp-content/plugins/ path.
3. Activate the plugin using the Plugins menu in WordPress sidebar menu.

Optionally, you can also try to upload the plugin zip file using Plugins -> Add New -> Upload Plugin option from the WordPress sidebar menu. Then go directly to point 3.
