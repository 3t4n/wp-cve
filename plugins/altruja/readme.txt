== Description ==

Altruja allows embedding simple, proven donation forms and fundraising tools to non-profits and political parties.

== Installation ==

1. Register for an account at https://www.altruja.de/register
2. Install this plugin, and navigate to Settings -> Altruja in your WordPress backend
3. Provide the email address you used to register to Altruja
4. Login to https://www.altruja.de/myaltruja, navigate to "Pages" and then the page you would like to embed, click the donation button to the top right, and then click "Enable". You can now collect donations.
