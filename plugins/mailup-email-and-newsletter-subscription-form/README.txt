== Description ==

= ITALIANO =

Il plugin di MailUp per WordPress permette di inserire sul proprio sito o blog WordPress un form per l’iscrizione degli utenti a newsletter, campagne email e SMS. 
Il plugin ha lo scopo di:

*   facilitare le operazioni di raccolta contatti e gestione del database 
*   far comunicare, in modo semplice e automatico, la piattaforma di invio MailUp con WordPress. 


Grazie a questo plugin, i dati inseriti dagli utenti nel form di iscrizione sono salvati in modo automatico sull’account MailUp. Si potranno quindi inviare campagne email e SMS a tutti i contatti raccolti tramite WordPress direttamente utilizzando la piattaforma di invio MailUp. 

= ENGLISH =

_With this plugin you can create a subscription form on your WordPress website or blog and let your visitors subscribe to your email and SMS campaigns._

_This plugin aims to:_

*   _simplify data collection and database management operations, and_ 
*   _automatically connect your MailUp platform with your WordPress website_

_All the data entered by users in the form are automatically saved in your MailUp account. You can send your email and SMS campaigns to all the users who have subscribed through the form directly from your MailUp platform._
_This plugin allows you to create and customize your subscription forms in every detail: title, description, subscription button, form fields, terms and conditions._



== Come utilizzare il plugin / How to use the plugin ==

= ITALIANO =

1. Aver attivato una piattaforma MailUp ([disponibile anche in prova gratuita](https://lp.mailup.com/it/prova-gratuita/?b=1&utm_source=wordpress&utm_campaign=trial))
2. Scaricare da questa pagina il plugin e collegare l’account MailUp a WordPress 
3. Iniziare a progettare il modulo di iscrizione. In Impostazioni generali, selezionare la lista e il gruppo MailUp in cui raccogliere i destinatari. 
4. Personalizzare il form aggiungendo titolo, descrizione e pulsante di iscrizione 
5. Nel tab Campi del form, personalizzare le informazioni da richiedere ai destinatari e i termini e condizioni. Definire gli eventuali campi obbligatori. 
6. In Impostazioni avanzate, definire se si intende attivare l’invio di una richiesta conferma indirizzo, personalizzare i messaggi di feedback e customizzare ulteriormente il modulo tramite CSS (qualora si desideri utilizzare uno stile diverso da quello ereditato automaticamente da WordPress) 
7. Posizionare il form utilizzando lo shortcode [mailup_form] oppure tramite widget (si può utilizzare un solo form per pagina del sito) 

Per maggiori dettagli, fare riferimento alla sezione *Screenshots of the plugin instructions* in fondo alla pagina oppure alla [documentazione tecnica](https://help.mailup.com/display/mailupUserGuide/WordPress/?b=1&utm_source=wordpress&utm_campaign=trial). 

= ENGLISH =

1. _Have a MailUp platform ([or activate a free trial](https://lp.mailup.com/en/free-trial/?b=1&utm_source=wordpress&utm_campaign=trial))_
2. _Download from the present webspage the plugin and connect MailUp account to WordPress_
3. _Start designing the sign up form. In General settings, select the MailUp list and group for collecting the recipients_
4. _Customize the form by adding title, description, and subscription button_
5. _Use the Form Fields tab to customize the information that you want to collect from your visitors and insert terms and conditions. Define the required form fields (optional)_
6. _In Advanced settings, you can activate an address confirmation request, customize the feedback messages and change the CSS file (if you want your form to have a different style as compared to your WordPress website)_
7. _Place the form using the [mailup_form] shortcode or the widget (you can add only one form per webpage)._

_For additional details, please refer to the Screenshots of the plugin instructions section or check the [technical documentation](https://help.mailup.com/display/mailupUserGuide/WordPress/?b=1&utm_source=wordpress&utm_campaign=trial)._

== Installation ==

= ITALIAN =

Di seguito i passaggi per installare il plugin MailUp for Wordpress:

1. Accedere all’account admin del proprio sito WordPress
2. Selezionare “Aggiungi nuovo” nel tab Plugin 
3. Ricercare MailUp for Wordpress nella barra di ricerca
4. Cliccare su “Installa adesso” e selezionare “Sì” per confermare 
5. Cliccare su “Attiva”

Il plugin è aggiunto alla sidebar principale con il nome “MailUp”
Per maggiori informazioni su come installare il plugin, [visitare la pagina di approfondimento](https://help.mailup.com/display/mailupUserGuide/WordPress). 

= ENGLISH =

_Here are the steps to install MailUp for WordPress plugin:_

1. _Sign-in to your WordPress admin account from your Wordpress website or blog_
2. _Go to the Plugins tab and click on “Add New”_
3. _Search for the MailUp plugin on the search tab_
4. _Click on “Install now” and then "Yes" to confirm_
5. _Click on "Activate plugin"._

_The plugin is added into the main sidebar, with the name “MailUp”._
_For further details on how to install the plugin, please refer to the [technical documentation](https://help.mailup.com/display/mailupUserGuide/WordPress)._


== Frequently Asked Questions ==

= [ITA] DEVO ESSER CLIENTE MAILUP PER UTILIZZARLE IL PLUGIN? =

Per usare il plugin è necessario aver attivato una piattaforma MailUp ([anche in prova gratuita](https://lp.mailup.com/it/prova-gratuita/?b=1&utm_source=wordpress&utm_campaign=trial)) e disporre delle credenziali di accesso. 

= [ITA] VORREI CAMBIARE L’ASPETTO DEL FORM CREATO CON IL PLUGIN: COME POSSO FARE? =
Il plugin eredita automaticamente lo stile del proprio sito WordPress. Per modificarlo, occorre andare nella sezione Impostazioni avanzate del plugin e modificare il file CSS esistente. 

= [ITA] È POSSIBILE TRADURRE QUESTO PLUGIN? =
Il plugin eredita automaticamente la lingua del proprio sito WordPress. Per modificarla, occorre modificare la lingua stessa del sito nella sezione Impostazioni di WordPress. 

= [ITA] POSSO DISABILITARE IL PLUGIN? =
Sì, puoi disconnettere in qualsiasi momento il plugin dalla sezione Impostazioni avanzate. Disconnettendo il plugin i form che hai incluso nel tuo sito smetteranno di funzionare.

= [ITA] E' compatibile con WPML ? =
Sì, una volta salvato il form torverai le stringhe da tradurre in WPML sotto la voce "traduzione strighe."  

= [ENG] Do I have to be a MailUp customer to activate this plugin? = 
 
_If you want to use this plugin it is necessary to have a MailUp platform (or start the free trial) and have the login credentials._

= [ENG] How can I change the appearance of the form created with the plugin? = 
 
_The plugin has the same style as your WordPress website. If you want to change it, you shall go to the Advanced Settings section of the plugin and change the CSS file._
 
= [ENG] Can I translate the language used in the subscription form created with this plugin? = 
 
_The plugin has the same language as your WordPress website. If you want to change it, you shall go to the Settings section of WordPress and change the language._

= [ENG] Can I disable the plugin? = 
 
_Yes, you can disable the plugin whenever you want, by going to the Advanced settings section. Please be aware that by disabling the plugin all the subscription forms you created on your WordPress website will stop functioning._ 

= [ENG] Is the plugin comaptible with WPML? = 
 
_Yes, once you have saved the form, you will find the strings to be translated in WPML under the heading "strings translation."_
