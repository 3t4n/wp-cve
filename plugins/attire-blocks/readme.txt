Attire Blocks is a collection of beautifully designed, highly customisable gutenberg UI blocks and ready-made layouts to create pages and posts with just a few clicks.



=== Description ===

=== Installation ===

1. Upload the `attire-blocks` folder to your `/wp-content/plugins/` directory or alternatively upload the attire-blocks.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your computer.
2. Activate the Attire Blocks WordPress plugin through the 'Plugins' menu in WordPress.
3. Use Attire Blocks on your next page or post.
