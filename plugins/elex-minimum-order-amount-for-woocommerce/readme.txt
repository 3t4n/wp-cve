== Description ==
ELEX WooCommerce Minimum Order Amount plugin helps you to configure minimum and maximum order amount based on WordPress user roles.
WooCommerce Minimum Order Amount plugin allows you to display custom messages to the end customer when restrictions are applied.
If you have any custom requirement on restricting WooCommerce minimum order, please feel to contact <a href="https://elextensions.com/ecommerce-development-services/">ELEX Development Services</a> explaining your business case.
If you have any queries please contact <a href="https://elextensions.com/support/">ELEX support</a>

== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. That's it! you can now configure the plugin.
