== Description ==

**Ultimate Colors** helps WordPress users, even beginners to **change colors on their WordPress website** without coding. The plugin **works with any theme** and **integrates with the WordPress Customizer to preview in real-time**.

The power of the plugin comes from the flexibility to **select any element on your website to change colors for**. Once the elements are added to the list, they will be instantly available in the Customize with a set of settings that allows you to customize the color.

With **Ultimate Colors**, you can fully control the colors of your WordPress website with no coding knowledge!

> If you like Ultimate Colors, you might want to check out [**Ultimate Fonts**](https://wordpress.org/plugins/ultimate-fonts/) which allows you to change fonts on your WordPress website.

### Features

* **Live preview with Customizer**: Smooth integration with WordPress Customizer to give you preview the changes in real-time.
* **Optimized for performance & SEO**: The plugin is well coded and optimized for the best website loading speed. Enjoy the beautiful look and high SEO score!
* **Works with any theme**: Made for users, Ultimate Colors works with any theme. No conflicts, no extra setup. Just install and enjoy!
* **Developer & User friendly**: The plugin is easy to use and easy to integrate into themes if theme authors want to have color options for their users.

Love the plugin? Check out our [WordPress themes](https://gretathemes.com)!

== Installation ==

1. Go to Plugins > Add New
1. Search for Ultimate Colors
1. Install and activate the plugin
