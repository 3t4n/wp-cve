== Description ==

### Leadster – Plataforma para aumentar sua Captação de Leads

Criamos uma Metodologia Própria de Marketing Conversacional unindo 5 anos de aprendizado na construção de chatbots com as melhores práticas de otimização de conversão online.

Transforme seu formulário em uma conversa, a ferramenta altera a experiência do seu cliente tornando o processo de conversão muito mais interativo e dinâmico.

A ferramenta realiza Teste A/B com diferentes chamadas para descobrir quais as campeãs de conversão para seu público.

* Casos de Sucesso – [Veja aqui!](https://leadster.com.br/geracao-de-leads/cases)
* Mais detalhes da ferramenta - [https://leadster.com.br/ferramenta](https://leadster.com.br/ferramenta)

== Frequently Asked Questions ==

= É necessário ter uma conta Leadster para usar o Plugin? =
Sim, crie sua conta e tenha 14 dias de teste grátis [Criar conta](https://app.leadster.com.br/users/sign_in)
Não é necessário Cartão de Crédito.


== Support Javascript snippet code ==

Este plugin tem como objetivo facilitar a instalação da ferramenta chatbot da Leadster.
Para isso, usamos 'https://cdn.leadster.com.br/' para carregar nosso snippet de chatbot em seu site, sob os termos da lei e [Políticas de Privacidade da Leadster](https://leadster.com.br/privacidade).


== Installation ==

1. Faça upload `/leadster/` para o diretório `/wp-content/plugins/` ou adicione-o a partir do menu de busca de plugins;
2. Ative seu plugin;
3. Vá para Configurações - Leadster Script;
4. Digite seu identificador obtido no painel de implementação (Painel)[https://app.leadster.com.br/implementation];
5. Clique em "Salvar e Ativar";
6. Caso trabalhe com algum mecanismo de cache (Memcached, LiteCache...) limpe o cache para atualizar o HTML;
