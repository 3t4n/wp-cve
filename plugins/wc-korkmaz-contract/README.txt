== Description ==

Bu eklenti, WooCommerce ile mesafeli satış sözleşmesi oluşturmanıza, ön bilgilendirme formu hazırlamanıza, kendi talepleriniz doğrultusunda yasal sözleşmeler yaratmanıza, ödeme ekranında bu sözleşmelerin otomatik olarak doldurulmasına ve PDF formatında e-posta yoluyla gönderilmesine olanak tanır.
* Yasal sözleşmeleri admin panelinden ekleme ve düzenleme imkanı.
* Sözleşmeye istenilen bölüme istenilen bilgileri yerleştirme olanağı.
* Sözleşmelerin ödeme sayfasında otomatik olarak oluşturulması.
* Sipariş onaylandığında PDF formatında sözleşmenin otomatik gönderilmesi.
* Admin ve müşteri sipariş sayfasında geçmiş sözleşmelerin görüntülenmesi.
* E-posta ile gönderilen PDF'lerin uploads klasörüne kaydedilmesi.

[youtube https://www.youtube.com/watch?v=wKfnfilEIk4]

== Installation ==

Eklentinin yüklemesi ve kullanımı son derece kolaydır.

Eklentiyi buradan indirin veya WordPress eklentiler bölümünde arayarak yükleyin ve etkinleştirin.
Eklentiyi indirdikten sonra sitenizin /wp-content/plugins/ klasörüne arşivden çıkararak yükleyin.
Admin panelinizdeki eklentiler bölümüne giderek, eklentinin yanında bulunan 'Etkinleştir' düğmesine tıklayın.
Admin panelinde 'Sözleşmeler' başlığı altında yer alan alana sözleşmeler ekleyin.
