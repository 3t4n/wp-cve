Get better targeted ads from Google's AdSense with this plugin. You can also have Google's AdSense Ignore specific sections of your post by using the included shortcodes [ignore] and [/ignore]

== Installation ==
1. Download the .zip file
2. Unpack into wp-content/plugins/ folder
3. Activate the plugin
4. Define your settings in the options page

or

1. Login into your wp-admin section
2. From the Plugins menu choose 'Add New'
3. Search for 'Better Adsense Targeting'
4. Click on the 'Install' link for this plugin

== Description ==
Using the Google AdSense Section Targeting will help the ads being displayed be more releveant to the content on your pages. This plugin lets you determine the areas of your site that will be used to select ads. You can find out more about Section Targeting at the following URL:

<a href="https://www.google.com/adsense/support/bin/answer.py?hl=en&answer=23168" target="_blank">What is section targeting and how do I implement it?</a>
