== Description ==

A powerful Video Gallery plugin that allows you to embed videos from YouTube, Vimeo and Dailymotion through categories. You can manage them through a neat and user-friendly settings panel. Also included a help section for your reference.

= Features =
* Embed videos from YouTube, Vimeo and Dailymotion
* Easy to install and use.
* Works with your theme's files and UI style.
* You can style as you need.
* A complete help guide for your support.
* Responsive & latest UI.
* Two types of layouts.
* View videos in popup view.
* Can order videos as you need with numbers.
* Shortcode for full gallery, categorised gallery and single video.
* Customizable video thumbnail and player size.

== Installation ==

You can download and install Rio Video Gallery using the built in WordPress plugin installer. If you download Rio Video Gallery manually, install the plugin using the steps given below

1. Upload 'rio-video-gallery' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can see a new section named "Video Gallery" appeared in sidebar.
4. Place the shortcode displayed in settings panel in any page or post.
5. You are done.

== Frequently Asked Questions ==

= How can i filter video gallery based on a category =

In settings tab under the video gallery, you can generate shortcode based on video categories.

= Displaying 404 Error =

After installing plugin, Please update the permalink structure with "/%category%/%postname%/". If you are having the same structure, just press the "Save Changes" button once more.
