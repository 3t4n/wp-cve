== Description ==

Divi Supreme Maps provides the power you need to create and manage customized maps quickly and easily. Take your websites to the next level with robust control over style and functionality. This completely free plugin integrates smoothly with OpenStreetMap & Leaflet Maps (Google & Bing coming soon).

With a simple interface and a great builder, you can design interactive maps that match your website’s style and exceed your customers’ expectations. Add fully responsive full-screen or fixed-width maps. Use unlimited markers/pins to showcase multiple locations, specific regions or provide store locators and directions.
Features:

= Divi Leaflet Map Module =

* Pinpoint Geolocation Accuracy
* Fast and Streamlined (loading only the related JavaScript)
* Unlimited Markers/Pins
* Markers/Pins Clustering
* Custom Latitude and Longitude
* Custom Marker/Pin Images
* Add Links to Markers/Pins
* Popup Styling
* Show Popup on Hover/Click
* Auto-Load Markers/Pins/Popups (so user can close)
* Advanced Map Control Settings (show zoom control, use scroll wheel zoom or double-click zoom, center map and more)
* Soon to Integrate with Google Maps & Bing Maps
* World-Class Support
* More on the way: Let us know what features you’d like—we want to build the plugins you want!

= Divi Google Map Module =
* Preset Google Map Styles to choose from
* Fast and Streamlined (loading only the related JavaScript)
* Marker Font Styling
* Unlimited Markers/Pins
* Custom Marker/Pin Images

With the power of Divi Supreme Maps at your fingertips, you can provide maps tailor-made to your users’ specific needs and preferences and make your websites stand out.

== Installation ==

Installing "Supreme Maps" can be done either by searching for "Supreme Maps" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Do I need Divi Theme for this to work? =
You must have Divi theme, Extra Theme or the Divi Builder installed in order for this plugin to work. [Divi by ElegantThemes](https://www.elegantthemes.com/affiliates/idevaffiliate.php?id=34051&tid1=wordpress) (affiliate link).

= Does it work with the Divi Visual Builder? =
YES! At least Divi 3.1 onwards where they introduced the Divi Developer API.

Divi Theme >= version 3.1
Extra Theme >= version 2.1
Divi Builder >= version 2.1
PHP version >= 7.0

= Is this plugin free? =
Yes! Supreme Maps is absolutely free. There is no hidden charge.
