Automaticlly adds a flexible KakaoTalk Plusfriend button and/or 1:1 Chat button to your WordPress website.

== Description ==

이 플러그인은 워드프레스 사이트에 카카오톡 플러스친구 추가하기 버튼과 1:1 채팅 버튼을 사용할 수 있게 도와줍니다.

= Features Include: =

* 간편한 설치와 셋업.
* 플러스친구 추가 버튼 및 1:1 채팅 버튼.
* 다양한 설정 옵션

= Support Me =

* Contribute on [Github](https://github.com/wpguide/wp-kakao-plusfriend).

== Installation ==

**Through Dashboard**

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **plusfriend** in the search box and click on search button.
3. Find WP Kakao Plusfriend plugin.
4. Then click on Install Now after that activate the plugin.
5. Go to Settings -> Kakao Plusfriend.

**Installing Via FTP**

1. Download the plugin to your hardisk.
2. Unzip.
3. Upload the "wp-kakao-plusfriend" folder into your plugins directory.
4. Log in to your WordPress admin panel and click the Plugins menu.
5. Then activate the plugin.
6. Go to Settings -> Kakao Plusfriend.
