== Giới thiệu ==
Việc cài đặt nhiều plugin hiện rất phổ biến đối với người dùng WordPress. Điều này có thể gây chiếm tài nguyên website và làm chậm tốc độ tải trang của bạn. Tệ hơn nữa, điều này cũng có thể mang lại trải nghiệm xấu cho khách hàng, ảnh hưởng đến xếp hạng SEO của website.
Để giúp bạn vượt qua những khó khăn đó, Chúng tôi đã đưa ra một bộ công cụ để tăng sức mạnh hiệu quả cho website WordPress của bạn. Bạn chỉ cần một plugin để sử dụng tất cả các tính năng được tích hợp sẵn.

=== Tính năng ===
Mặc dù các tính năng có sẵn đã được kiểm tra và phê duyệt trước khi xuất bản, chúng tôi đánh giá cao phản hồi của bạn để cải thiện plugin tốt hơn nữa.
- Liên hệ nhanh.
- Tùy chỉnh thanh trang toán WooCommerce.
- Tùy chỉnh trang thanh toán WooCommcer dành cho thị trường Việt Nam.
- Chèn mã vào Header / Footer.
- ...

== Bảo vệ riêng tư ==
Với cấu hình mặc định, bản thân plugin này không:
- Theo dõi người dùng.
- Ghi mọi dữ liệu cá nhân của người dùng vào cơ sở dữ liệu.
- Gửi bất kỳ dữ liệu nào đến các máy chủ bên ngoài.
- Sử dụng cookie.

== Installation ==
Bạn có thể cài đặt All in One Tools từ Bảng điều khiển WordPress của mình hoặc theo cách thủ công qua FTP.

= Hướng dẫn cài đặt từ Bảng điều khiển WordPress =
1. Chọn 'Plugins -> Cài mới'.
2. Tìm kiếm từ khá `All in One Tools` và cài đặt.
3. Kích hoạt plugin từ menu Plugins.
4. Cấu hình cài đặt của plugin.

= Hướng dẫn cài đặt thủ công =
1. Tải file nén plugin dạng: `aio-tools.zip`.
2. Tải file nén `aio-tools.zip` vào thư mục website `/wp-content/plugins` (không được đổi tên thư mục này).
3. Kích hoạt plugin từ menu Plugins.
4. Cấu hình cài đặt của plugin.
