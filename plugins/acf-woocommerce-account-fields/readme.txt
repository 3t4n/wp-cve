== Description ==

Add Advanced Custom Fields to the Woocommerce registration form and edit profile form.

This plugin supports all ACF field types including:

*   Basic
*   Content
*   Choice
*   Relational
*   jQuery
*   Layout

== Installation ==

Getting started is as easy as can be:

1. Upload the plugin files to the `/wp-content/plugins/acf-wc-account-fields` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create or edit a Field Group in Advanced Custom Fields.
4. Under Location Rules, select: "Woocommerce Account Fields is equal to Field Group" and hit save
5. Then head over to Woocmmerce -> Settings -> Accounts & Privacy and scroll to the bottom.
6. Select the corresponding ACF form to the location in which you want it displayed


== Frequently Asked Questions ==

= Do I have to create a field group in advanced custom fields =

Yes. These same field groups will be the fields used in your forms.
