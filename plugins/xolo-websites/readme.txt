== Description ==

= FREE TEMPLATES FOR ELEMENTOR PAGE BUILDER =

Create the professional designed pixel perfect websites for every business such as a blog, portfolio, agency, landing page, application page, freelancer's, magazine, and more.


This plugin gives you access to 10+ pre-made full website templates for your favorite page builder such as Elementor.


= GET A WEBSITE LIVE IN 3 CLICKS! =

1. Install and activate Xolo Websites Plugin
2. Pick a website that suits your needs
3. Import the website
4. Done!

Use this Prebuilt templates as a base for your project and don’t waste time starting from scratch!

_<a href="https://wpxolo.com/ready-website/">See list of all available templates to import »</a>_

== Installation ==

= Option 1. =

- Upload the entire `xolo-websites` zip file to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Xolo Websites" —> Click install.
- After plugin is downloaded, click active plugin

= After the plugin active =

- You will find 'Appearance > Xolo Websites' menu in your WordPress admin panel.
