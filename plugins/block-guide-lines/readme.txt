This plugin add visibile guide lines to gutenberg blocks, now the block are more readable ;)

New Styles! With labels and minimal only with lines:
Go in Settings > Block Guide Lines > Select the style

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Blocks Guide Lines"
3. **Activate** Blocks Guide Lines from your Plugins page
4. Enjoy editing
