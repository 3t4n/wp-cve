Easily add ManyChat widgets to your site.

== Description ==

Block Editor for ManyChat is the missing link in getting your site connected to your ManyChat Facebook Messenger Bot.  Forget the pain of trying to figure out how to get the correct Javascript embeded into your website header. When you install and activate this plugin you will just need to enter your Facebook page id and we will do the rest.

In addition to setting up the Javascript for you, we have included blocks to allow you to choose from the widgets you have created inside your ManyChat account.  Compatible with WP 5 blocks and Divi Modules!

== Installation ==

1. Upload `wp-manychat` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit Settings -> Block Editor for Manychat
1. Enter your Facebook page id
1. Create/Edit a page/post/widget and select the ManyChat widget you would like to display.
