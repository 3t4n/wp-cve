== Description ==

[Demo](https://tishonator.com/plugins/carousel-3d-slider)

Configure a Responsive 3D jQuery Carousel Slider and Insert it in any Page or Post as a Shortcode. Admin slide fields for title, text, image.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `carousel-3d-slider` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Admin Panel -> 'Carousel 3D Slider' -> fill Slider fields
4. Insert the shortcode [carousel-3d-slider] to display Slider to any page or post
5. Enjoy!
