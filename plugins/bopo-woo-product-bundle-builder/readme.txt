== Description ==

Bopo – Woo Product Bundle Builder (Best for WooCommerce product bundle) is built for shop owners to create amazing deals that combine several products to put for sale together. This is such a powerful plugin for boosting sales of different product types, and at the same time offering customers some reasonable benefits. Bopo – Woo Product Bundle Builder is a smart tool for enhancing cross-selling strategies on your online WooCommerce shop. It is also optimized for search engines, increases site ranking, and ensures a higher conversion rate.

###FEATURES:

- **CREATE PRODUCT BUNDLES WITH BOPO WOOCOMMERCE PRODUCT BUNDLE BUILDER **: You can add new WooCommerce product bundles or create bundles in existing products in WooCommerce, you can create bundles for simple and variable products. Customers can add bundle items you assigned in the backend, purchase bundles at a discount price, the bundles will help you to motivate more purchases and traffic..

- **Shipping fee**: Select to calculate the shipping fee in the whole bundle or each bundle item..

- **Rules**: Select which products appear in the bundle item list.

	- Default product: Assigning a product as the default bundle item.
	- Product: Selecting products to appear on the bundle item list.
	- Exclude product: Select which products can not appear on the bundle item list.
	- Categories: Selecting products of certain categories to appear on the bundle item list.
	- Exclude categories: Select which categories can not appear on the bundle item list.
	- Tags: Select which products whose tags are enlisted in the tag field to appear on the bundle item list.
	- Exclude tags: Not select which products whose tags are enlisted in the tag field to appear on the bundle item list.

- **Display**: Sort suggested items in the bundle list, you can sort by:

    - Price: Sort suggested bundle items in the list by price. You can sort by ASC or DESC
    - Title: Sort suggested bundle items in the list by title. You can sort by ASC or DESC
    - Rating: Sort suggested bundle items in the list by rating. You can sort by ASC or DESC

- **Quantity**: Allow to set the default quantity of the current bundle item.

- **Bundle Discount**: Set discount for the bundle by percentage or fixed price.

- **BUNDLE SHORTCODES:**: You can use the shortcode to display the Bopo bundles everywhere you want.

&#9658; **DESIGN BUNDLES WITH BOPO WOOCOMMERCE PRODUCT BUNDLE BUILDER**: The information displayed on a bundle item: You can enable to show on a bundle item include:

- **Quantity**: Quantity of current bundle item.

- **Stock**: Number of products in the stock of current bundle item.

- **Rating**: Star rating of current bundle item.

- **Short description**: Short description of current bundle item appeared when hovering.

- **Link to a single product**: Choose action when clicking on the title of the bundle item. It can be:

    - No link.
    - New tab.
    - Popup: Display a list of suggested bundle items under a popup.

- **Bundle template**: There are 2 bundle templates to choose vertical or horizontal.

- **Bundle popup**: Popup displays all variables and single products. When clicking on the image, if it is a simple product, it will be set as the bundle item or if it is a variable product, a popup will show variations of that variable product.

- **Design button and the bundle item popup**: Change the product button and Popup change product

    - Text: Change the text of the bundle button and the text to change products in the bundle list popup.
    - Color: Change the color of the text in the button and the bundle list popup.
    - Background color: Change the background of the button and the bundle list popup.
    - Number product per page: Change the number of product display per page.

### PRO VERSION:

- **More items in bundle**: Can add up to 60 item on bundle.

- **Quantity for item**: User can select quantity to buy for each item of bundle.

- **Item product variation**: Include/Exclude product variation from bundle item.

- **Item gallery**: View product gallery on select item popup.

- **Filter product**: Search and filter product on select item popup.

- **Calculate product**: Item product price based on regular price/sale price/current price.

- **Optional item**: The bundle item can be set to optional and not required to buy this item.

- **Variations product**: Additional a type of variation select.

- **Item on cart**: Add setting to remove product if this already on cart when add bundle to cart.

- **Cart edit**: Edit product item, quantity on cart.

- **Order edit**: Edit product item, quantity on checkout page.

3. Activate the plugin through the 'Plugins' menu in WordPress
