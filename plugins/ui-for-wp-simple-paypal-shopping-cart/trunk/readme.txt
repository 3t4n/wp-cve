Generates the short code for WordPress Simple Paypal Shopping Cart.

== Description ==

The plugin adds a button in the richtext editor which allows you to easily generate the short code for *WordPress Simple Paypal Shopping Cart*.
It saves you typing the short codes and therefore reducing the chance of typos.
The plugin is very easy to use. You just fill out a form and it will generate the necessary shortcode.
