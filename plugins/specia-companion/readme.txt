== Description ==

Specia Companion is created for <a href="https://speciatheme.com/" rel="nofollow">Specia Theme</a>. The plugin set frontpage sections, It allow edit customizer settings for the theme. Extend your theme functionalities with one click import & enjoy free stock images. Build your website with Specia.

**Features:**
&#9989; Setup Header Section
&#9989; Add Slider Section
&#9989; Add CTA Section
&#9989; Add Service Section
&#9989; Add Features Section
&#9989; Add Portfolio Section
&#9989; Setup Footer Section

You can check out our themes demo <a href="https://speciatheme.com/themes/" rel="nofollow" target="_blank">Click Here</a> for a more detailed overview of the features.

== Installation ==

= Option 1. =

- Upload the entire `specia-companion` zip file to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Specia Companion" —> Click install.
- After plugin is downloaded, click active plugin



== Frequently Asked Questions ==

= How to Install Plugin ? =
Just upload the `specia-companion.zip` to the `/wp-content/plugins/` directory, and then activate it from 'Plugins' menu in WordPress.

= This plugin only work for Specia themes ? =
Yes, This plugin is specially make for Specia Themes . You can install theme for better compatibility.
