= Plugin Description =
Just install the plugin and enter its menu to automatically generate all important pages - 1 parent page "info" and several sub-pages "Privacy Policy", "Terms of conditions", "Contact us", About us". Then automatically creates a menu-links for them and appends it in footer.
You can set that on either Multisite-Wide or per single site. Also, plugin has an option to show a basic <b>Privacy Pocily</b> consent notice to first-time visitors.

See all available options and their description on plugin's settings page.
