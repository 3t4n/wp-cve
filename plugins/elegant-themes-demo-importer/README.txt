== Description ==

Elegant Blog Themes Demo Importer plugin helps you import demo content for various free themes of [elegantblogthemes](http://elegantblogthemes.com/) . Elegant Blog Themes are dedicated to creating high quality, easy to use WordPress themes. If you want to import demo content for any of the Elegant Blog Themes free version, then this is the plugin for you. Remember this only works with elegant blog Themes (themes) . This plugin is based on [One Click Demo Import Plugin](https://wordpress.org/plugins/one-click-demo-import). Install the elegant blog Theme Demo Importer Plugin, One Click Demo Import Plugin and you are good to go.

== Installation ==

**From your WordPress dashboard**

1. Visit 'Plugins > Add New',
2. Search for 'Elegant Blog Themes Demo Importer' and install the plugin,
3. It will also need 'One Click Demo Import' plugin so install that too
3. Activate both of the plugin from your Plugins page.

== Frequently Asked Questions ==

= What does this plugin do? =

This plugin will help you import demo content for various free themes of elegantblogthemes.com

= Will I need any other plugin? =

Yes, This plugin depends on One Click Demo Import plugin, Install both of the plugin and you can import any demo content of Elegant Blog Themes (themes)
