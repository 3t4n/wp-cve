Ads.txt Manager is a plugin to help WordPress sites easily take advantage of the Ads.txt Manager service.

== Description ==

Ads.txt Manager is a free way to manage, update, and easily maintain your ads.txt file.
Save time by eliminating manual file updates and authorize sellers to ensure you have the most up to date file possible.

== Installation ==

To properly install Ads.txt Manager please follow the instructions below.

1. Download the plugin from the wordpress.org marketplace
2. Create an Ads.txt Manager account at [https://svc.adstxtmanager.com/auth/register](https://svc.adstxtmanager.com/auth/register)
3. Make sure that Permalinks are enabled for your site, this plugin will not work unless permalinks are enabled.
4. Update your plugin with your Ads.txt Manager Id.

Then you're done!
