== Description ==

The Carousel Slider With Text is the most powerful, responsive, and intuitive plugin to create a carousel slider with text. It is the best post slider for WordPress that allows you to create an attractive and eye-catching blog post using a carousel slider.

Using the WordPress Post slider plugin, you can play with the core of WordPress. You can not only display your blog posts in a carousel slider but also can present them to your user in a responsive way. That means it will display very nicely on any device. 

This plugin has emerged as the most dynamic way that can be used to display visual media on your website. The best part of this WordPress post slider carousel is it allows you to create beautiful sliders and tell stories without requiring any code. It also has 100% touch and swipe support. 

HOW DO YOU MAKE A SLIDER IMAGE RESPONSIVE IN WORDPRESS?
To create a responsive carousel slider in WordPress, you’ll need to follow these 5 steps:

1. Make some posts with relevant images.
2. Install and activate the carousel slider with text. 
3. Create a slider with your selected posts.
4. Configure slider settings and get the shortcode.
5. Publish and display.

Extra Features
Autoplay Controls

It comes with a few animation effects and autoplay options. This allows switching between the slides after a period of time. You can customize the slider’s autoplay value, hover effect, direction, and loop effect. 

Custom Query

Allows you to add posts or sources of posts; even you can add custom posts also. With this feature, you can include or exclude posts as you want. Another benefit of this feature is it allows ordering your post by Ascending / Descending. 

Content Customization

With this plugin, you can optimize your content also. It allows you to show/hide the content title, post description, date, thumbnail, using the title HTML tag, and hide the slider description on the mobile screen.
