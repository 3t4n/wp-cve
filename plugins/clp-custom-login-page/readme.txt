== Description ==
Custom Login Page plugin allows you to customize any essential element on WordPress login page. It utilizes powerful customizer to implement changes in real-time. Change of a logo, graphic background, typography or just a simple tweak cannot get easier than this.

<h3>Features</h3>
<ul>
	<li>Carefully crafted predefined templates to choose from: minimalist, flat, light, dark and more.</li>
	<li>Customizable layout - from full width, 50-50 to 2/3 layout.</li>
	<li>Logo customization: disable logo, custom text with chosen typography or a graphic logo.</li>
	<li>Background: color, gradient, pattern, image or YouTube video.</li>
	<li>Custom Login Page footer to display Copyright or any content.</li>
	<li>Integrated Unsplash Gallery - choose from more than 1.5 millions free hi-res images.</li>
	<li>Form: width, height, spacing, color, blur, border radius, complete typography, borders, shadows.</li>
	<li>Form Inputs and Labels: colors, hover & focus colors, border radius, disable password icon, disable/enable "remember me" checkbox.</li>
	<li>Form Submit Button: custom text, color, hover color, size, alignment, borders.</li>
	<li>Form Footer: alignment, font size, change texts.</li>
	<li>Messages: complete customization of login, register and lost password messages including colors.</li>
	<li>Custom CSS: insert any custom CSS code for expert customization.</li>
	<li>Login Page is 100% responsive.</li>
	<li>Import / Export of the customizer setttings.</li>
</ul>

== Installation ==

<h3>Install from WordPress Plugin Dashboard</h3>
<ul>
	<li>Navigate to the 'Add New' in the plugins dashboard</li>
	<li>Search for 'Custom Login Page'</li>
	<li>Click 'Install Now'</li>
	<li>Activate the plugin on the Plugin dashboard</li>
	<li>Click Customize login page to enter customizer.</li>
</ul>

<h3>Upload in WordPress Plugin Dashboard</h3>
<ul>
	<li>Navigate to the 'Add New' in the plugins dashboard</li>
	<li>Navigate to the 'Upload' area</li>
	<li>Download <code>custom-login-page.zip</code> and select <code>custom-login-page.zip</code> from your directory</li>
	<li>Click 'Install Now'</li>
	<li>Activate the plugin in the Plugin dashboard</li>
	<li>Click Customize login page to enter customizer.</li>
</ul>

<h3>FTP Upload</h3>
<ul>
	<li>Download <code>custom-login-page.zip</code> and extract the <code>custom-login-page.zip</code> zip file to your computer</li>
	<li>Upload the extracted <code>custom-login-page</code> directory to the <code>/wp-content/plugins/</code> directory</li>
	<li>Activate the plugin in the Plugin dashboard</li>
	<li>Click Customize login page to enter customizer.</li>
</ul>
