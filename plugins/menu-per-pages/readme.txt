== Description ==

Wp Menu Per Pages Plugin let you provide to access a menu on your selected page.

Features:
	
1. You can Selec your choice menu on each page.
2. This plugin Can be integrated in any theme.
3. Easy Installation.
4. Error Free.


== Installation ==

The automatic plugin installer should work for most people. Manual installation is easy and takes fewer than five minutes.

1. Download the plugin, unpack it and upload the 'menu-per-pages' folder to your wp-content/plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. First create multiple menus for working properly.
4. Go to your choice page and select a specific menu for this page.
5. Menu will automatically change when you click on a page.


== Frequently Asked Questions ==
How to use this plugin.
Just install the plugin, go to on your page and select the menu and save it.
