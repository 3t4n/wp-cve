Posts widget! Add list of recent posts from your CUSTOM POST TYPE.

== Description ==

It's a very simple plugin to display recent posts on your website sidebar widget and also there are an option to choose posts of your custom post type.

Note : please don't forget to leave your valuable feedback and suggsations for make my plugin much more better compare to other plugins.

= `Features` = 
 * An option to show comment count
 * An option to define hide widget title
 * An option to choose custom post type
 * Display post from specific category 
 * An option to exclude/include specific posts.
 * An option to define post filter order.
 * An option to define post content limit.
 * An option to define to display number of posts

== Installation ==

Installing the plugin:

1.  Unzip the plugin's directory into `wp-content/plugins`.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  The categories post will be available on widget "WP Posts Widget" (Apperance >> Widget)
