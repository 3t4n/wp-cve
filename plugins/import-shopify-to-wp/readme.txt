Easily transfer your Shopify Store to WooCommerce

== Description ==
Easily transfer your Shopify Store to WooCommerce by using a tool that is 100% free.

### How to use it:
1. Go to <a href="https://shopifytowp.com" rel="friend">shopifytowp.com</a>.
2. On this website you'll find a form where you must enter your Shopify store URL and a valid email.
3. After submitting the form, you'll be asked to install our Shopify App into your store.
4. Once the application is installed, you can submit a request to export the products, orders and customers.
5. An email, containing the download URL, will be sent to you when it's done. Follow it to get the store data in a ZIP archive.
6. Go to WordPress dashboard -> Tools -> Shopify Importer.
7. Select the Zip archive in the upload field and press "Upload". Once the upload is done, you'll be redirected to the next step where you can see the products list.
8. Press "Start Import" and wait for the process to finish.
9. Done.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Import Shopify To WP'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `shopifytowp-importer.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `shopifytowp-importer.zip`
2. Extract the `shopifytowp-importer` directory to your computer
3. Upload the `shopifytowp-importer` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin's dashboard
