== Description ==
<strong>Fami Sales Popup</strong> is an influential selling tool which helps to boost your sales. Built with the concept of social proof, the app displays purchase activities on your store via real-time notification popups. When customers know what other people are buying from your store, it creates a positive influence and motivates them to buy your products.

We develop this plugin completely free of charge and will constantly update new features as required by the community. We do not offer any paid versions for additional features, if you need any other features please send us a request, we will update the plugin according to your needs.

== Installation ==
1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `Fami Sales Popup` from Plugins page

== Main Functions ==
- Enable/Disable sales popup
- Enable/Disable sales popup on mobile devices
- Set the minimum/maximum time threshold to show someone just bought
- Customize display text
- Customize the list of products
- Customize the list of addresses
- Add multiple addresses with just one click
- Customize random display time by seconds, minutes, hours and days
- Ready for multiple languages

== Frequently Asked Questions ==
= Why can't I see the popup displayed? =
Make sure you have it turned on and have added the products to the list

= Why does the popup not show up on the mobile devices? =
Make sure you don't turn off the popup on mobile devices
