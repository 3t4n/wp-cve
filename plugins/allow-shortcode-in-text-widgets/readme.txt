Allow shortcode to be used in text widgets

== Description ==

Allow Shortcode in Text Widgets is a very simple plugin that does one thing... Allows users to add shortcode to text widgets.

There are no settings. Simply install and activate the Allow Shortcode in Text Widgets plugin, then use whatever shortcode you like within a text widget

== Installation ==

1. Go to Dashboard >> Plugins >> Add New >> Search
2. Search for "Allow Shortcode in Text Widgets," click "Install Now," then "Activate Plugin"
3. There are no settings. Simply add your desired shortcode to a text widget

== Frequently Asked Questions ==
