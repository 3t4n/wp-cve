== Description ==
The clima widget is a free weather forecasting widget for your beautiful wordpress website.

* Supported in Spanish and English
* The widget is responsive, perfect for mobile and desktop
* The widget style like background and color can be customized
* No development skills are needed
* High performance and lightweight code
* SEO friendly

** For special customization requests please contact this mail -> danielogo@tiempo3.com

The clima widget plugin is relying on a 3rd party as a service, the widget is using weather services provided by tiempo3.com.
* Service provider link - https://www.tiempo3.com
* Service provider terms of use link - https://www.tiempo3.com/terms
* Service privacy policy link - https://www.tiempo3.com/privacy


== Installation ==

1. Upload `clima.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the clima widget to your page from the widgets tab
