== Description ==

**Tranzly** is a **AI DeepL Translation Wordpress plugin** .It helps you to automatically generate a multilanguage website in just a few clicks!

- This plugin uses [DeepL](https://www.deepl.com/) API to translate the wordpress posts' and pages' content, The title, The content and the slug as well.
- However, The API Pro key is chargeable, You can find out more details about the api [here](https://www.deepl.com/pro/), Tranzly is not a service or a product provided by DeepL.
- Please read bellow the disclaimer section.

Tranzly uses the best machine translation engine **DeepL API** which trains artificial intelligence to understand and translate texts and that allows your website visitors to tap into the power translation text using AI. **The Future is Now.**
Also, with Tranzly You can **boost your international Traffic**, **Sales and Rankings* and Open the doors of global markets for your Website or Online shop.
Tranzly helps you save a lot of money by no longer having to hire expensive translation agencies. Thanks to Artificial Intelligence and ever-improving neural machine translations such as those offered by DeepL.

**Note**: The DeepL API is accessible with a DeepL Pro subscription. The API is an interface that allows this plugin to send texts to the DeepL servers and to receive high-quality translations.Instantly.


== Features ==

* DeepL Api: Integarte the revolutionary AI Translation solution into your multilanguage website.
* Posts & Pages: Translate your WordPress Posts and Pages automatically and generate perfectly SEO-Optimized Content
* Woocommerce: Localize all your WooCommerce Products and create awesome multilingual online shop experience.
* Language Switcher Widget: Place the Language switcher in any area of your website using Shortcode.
* SEO By Yoast: Create multilanguage SEO optimized content by translating Yoast meta fields. 
* Gutenberg support: Tranzly is fully compatible with Gutenberg Editor.
* Manual translation: You can use this plugin to manually translate your website pages or posts.
* 9 Languages : Deliver your content to the 9 most spoken langauges in the world,.. Imagine the possibilities.

== Installation ==

1. Copy the Tranzly folder into wp-content/plugins
2. Activate the plugin through the plugins' menu
3. Configure Tranzly through its admin options panel.
