Spreadsheet Price Changer for WooCommerce and WP E-commerce - Light

== Description ==

A WooCommerce / WP E-commerce product fast input spreadsheet editor for product price and sales price.

This free trimmed version of the famous [Product manager/bulk product editor](https://holest.com/bulk-product-manager-for-woo-commerce) that allows only change of the regular and the sales price; Mass prices update by amount or percentage by filter criteria; View of other product fields;
It supports both WooCommerce and WP E-commerce. GUI is like Excel. It also has an export feature for prices. This is the right thing for you if your users give you a blank stare when you're trying to explain to them how to update prices using the default shop interface.
UI features: frozen columns, column reordering, sorting.  
EDITABLE / MASS-UPDATE: 
WooCommerce: Price, Sales Price; 
WP E-commerce: Price, Sales Price
VISIBLE / EXPORTABLE FIELDS: WooCommerce: Price, Sales Price, Tags, Attributes (Each pivoted as a column), SKU, Category, Shipping class, Name, Slug, Stock, Featured, Status, Weight, Height, Width, Length, Tax status, Tax class, Image (image is not exportable)
WP E-commerce: Price, Sales Price, Tags, SKU, Category, Name, Slug, Stock, Status, Weight, Height, Width, Length, Taxable, local and international shipping costs, Image (image is not exportable)

If you like this web app/plugin you can consider the full version [Excel-like Product manager/bulk product editor](https://holest.com/bulk-product-manager-for-woo-commerce). With it you can edit all fields. You can import CSV. You create custom columns for almost anything related to the products including fields added by other plugins like YITH, Price Ranges, etc.... These defined columns will also be importable/exportable as default columns. It also provides PHP scripts you can use to set up auto-import by CRON task.

== Installation ==

1. Go to wp-admin > Plugins > Add New
2. Use search on 'excel-like-price-change-for-woocommerce-and-wp-e-commerce-light' or go to 'Upload' if you already have zip package with you to install plug-in
3. Activate the plugin through the 'Plugins' menu in WordPress
4. When you access plugin first time it will first calculate available memory. This can take a minute.
5. Use plugin
