== Description ==

The plugin handles the GIF upload process without losing the GIF animation in the uploaded gif and all generated subsizes.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start uploading GIF images from media and subsizes will keep the animation.
4. GIF Editor settings is listed under Media Page.
