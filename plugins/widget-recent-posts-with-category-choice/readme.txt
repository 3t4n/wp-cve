Simple Recent Post Widget with Advanced choice.

== Description ==

**Don't work? Open ticket, we answer in max 48h.**

Simple Recent Post Widget with this filters:

• Post Type
• Post Format
• Categories (only for "Post Type" post)
• Tags (only for "Post Type" post)
• Authors

You can also:

• Display post Thumbnail
• Display sticky post
• Display post Data
• Display author
**• Exclude widget in the same cpt page choice for widget**

**Use same class CSS of default WP Recent Post Widget.**
You can choice to include or exclude post for Post Format, Categories, Tags and Authors.
**From 1.3 version you can override default template with child theme on your-theme/widget-recent-post-advanced/frontend-1.php**

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins page directly.
2. Activate the plugin through the 'Plugins' page in WordPress.
3. Add Widget "Advanced Recent Posts" and setup properly.

You can use this id widget-recent-posts-in-category-*
*number of widget, you can show in browser console
