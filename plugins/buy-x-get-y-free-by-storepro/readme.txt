== Description ==

###  Buy X Get Y Plugin

Are you looking to add some promotional products to your cart? Here you go! The Buy X Get Y by StorePro allows you to achieve your goal. You can add the desired free product to the cart along with the main product with the help of simply arranged plugin settings.


= Installation =

Activate the Buy X Get Y plugin. Go to the Buy X Get Y settings and follow the Steps.

-Enable 'Buy X Get Y' globally
-Enable Cart Message (Optional)
-Add custom Cart Message
-Select the Main Product
-Select the Quantity of Main Product
-Select the Free Product
-Select the Quantity of Free Product
-Save

That's It. You are Ready to Go!
