You take the control of how users are tracked! It supports multiple and single answers. 
 
== Description ==

Modern Polls is easy to use!
You can customise the default Poll-Theme to your needs and Install multiple different versions of your own Theme.
The Backend is a modern, userfriendly and self-explaning Interface.
A Poll can have a Startdate and Time as well as an Enddate and time, automatically!
Use the Polls on Posts or whole Pages.

Review the Voters and Analysis of your Polls in the Backend Info. It includes the number of Voters as well as analysis in Charts.

Modern Polls uses its own Database Tables for better Backups, Exports, selfmade analysis and data integrity.

== Frequently Asked Questions ==

= How can i create a Template or change an existing one =

To create a Template, you can find a Readme.txt in the Templates-Folder (../wp-plugins/modern-polls/resources/views/templates).
I strongly recommend using the existing templates as a guide

To change an existing one, your can edit the corresponding .css and .hml Files in the template folder.

= Are there restrictions and/or limitations to an Paid/Pro Version?  =

NO! This Plugin is completely Free and there will never be any kind of Paid-Content.

= Are there restrictions for adding answers? =

There is no limit for the amount of answers. You can add as many answers as you want.

= Can I embed a poll manually? =

Yes. Post the shortcode to any Post or Page: [mpp id = "ID"], where ID is the ID of the Poll.

= Can I add the same poll in different pages? =

Yes. You can add the same poll on different Pages and Posts.

= Are there limitations for adding polls? =

No, there is no limit.
You can add as many polls as you want. This Plugin has no limitations.

= Where to change the settings? =

Settings you can change are located under the "Settings"-Tab in the Modern-Poll Menu on the Sidebar.
