== Description ==
Hide your long ugly affiliate links with your own short and easy to remember links or redirect your old posts to a new post or external link

This plugin can help you in

*   Hiding your affiliate link and converting your long affiliate links to a short easy to remember link
*   Redirect your old post to an external link or a new post
*   Ability to redirect one source URL to randomly selected one of the many destination URLs set
*   Keep track of the number of visits to these links 

More options to come (if you need a feature then all you need to do is suggest the feature and it will be added)

== Installation ==

1. Upload `link-to-url-post` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto the Settings > Link to URL to change setting
