Show Visitor IP - Simply display visitor IP Address & visitor another location info using by IP on post or page, anywhere using shortcode.

== Description ==

This plugin simply display Visitor IP Address & visitor location info using by visitor IP on post or page, anywhere using shortcode.

Very simple to install, simple to use, light weight.

== Installation ==

Use WordPress Add New Plugin feature, searching "Show Visitor IP", or download the archive.

1. Log in to your website administrator panel
2. Go to Plugins page, and add new plugin
3. Upload plugin
4. Click Install Now button
5. Then click Activate Plugin button

Shortcodes:
<code>[show_ip]</code>
<code>[svip_location]</code>
<code>[svip_location type="countryCode"]</code>
<code>[svip_location type="region"]</code>
<code>[svip_location type="lat"]</code>
<code>[svip_location type="long"]</code>
<code>[svip_location type="city"]</code>
<code>[svip_location type="countryName"]</code>
<code>[svip_location type="timeZone"]</code>
<code>[svip_location type="currency"]</code>
