== Description ==

Discounts are one of the best ways to entice more customers. WooCommerce MailChimp Newsletter Discount is a perfect WooCommerce plugin that will assist you in obtaining your visitors' email addresses simply by offering them a discount. It is one of the most effective ways to increase your email list, retain and convert a visitor into a potential customer.

The WooCommerce MailChimp Newsletter Discount plugin generates Forms and Popups (PRO) that integrate seamlessly with WordPress. As a result, if you own an e-commerce site, a business website using our responsive and customizable forms will look great on any device.

When a visitor subscribes with their email address, the WooCommerce MailChimp Newsletter Discount plugin adds the email address to your MailChimp Audience List and automatically sends them a unique coupon code to their registered email address.


== Features ==

Here is a list of incredible features of the WooCommerce MailChimp Newsletter Discount plugin.


* Easy and quick to create and integrate subscription form to your website
* When new subscribers subscribe, it automatically syncs them to your Audience in your MailChimp account
* Manage your subscribers and audience list in MailChimp
* Display your forms anywhere on your site using Shortcodes and WC MailChimp Subscribe (Our newly added Gutenberg Block)
* Customize the forms display fields, width, alignment, Top Text, Top Text Color, and more
* Select from a number of display fields, such as Email, First Name, and Email, or First Name, Last Name, and Email
* It automatically sends discount codes to new subscribers
* Customize the discount rules, including discount types, coupon value, products, and categories where the coupon is valid, minimum, and maximum purchase.
* Change the color of the Error Text, Success Text, Subscribe Button, Subscribe Button Hover Color, Subscribe Button Text Color, and Subscribe Button Label.

= Shortcode parameters =

**Use Following parameters with shortcode**

<code>[wc_mailchimp_subscribe_discount]</code>

* **Width:**
[wc_mailchimp_subscribe_discount width="100%"] (For 100% width)

* **Button Width:**
[wc_mailchimp_subscribe_discount btn_width="auto"] ( For button auto width. You can use '100px', '200px' )

* **Button Align:**
[wc_mailchimp_subscribe_discount btn_align="center"] (Align button center. If you want to align button left then use 'left' or if you want to align 'right' then use 'right' )

* **Layout:**
[wc_mailchimp_subscribe_discount layout="vertical"] (For vertical align the form form fields use 'vertical'. For horizontal use 'horizontal')

* **Top Text:**
[wc_mailchimp_subscribe_discount top_text="Get 10% Discount"] (Use top_text followed by text that you want to show text above the form)

* **Top Text Align:**
[wc_mailchimp_subscribe_discount top_text_align="left"] (Use top_text_align to show the top text left or right of the form. You left for left aligned. Use right for right aligned)

* **Top Text Color:**
[wc_mailchimp_subscribe_discount top_text_color="#ccc"] (Use color code for which the top text would be shown )

== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "WooCommerce MailChimp Newsletter Discount".
3. Click install.
4. Click activate.
5. Once the plugin is installed then you can see Newsletter Discount on the left navigation bar of WordPress Dashboard.

== Frequently Asked Questions ==

= What are the things that I need from MailChimp for setup this plugin? =
You only need your MailChimp API key. Once you put your MailChimp key and click on the list id link. It would show all the list id from your MailChimp account. So you don't need to put that manually.

= Will this plugin work with my theme? =
Yes, absolutely the plugin should work with your theme without any issue.

= Do you have any pro version of this plugin? =
Yes, absolutely we have the pro version of this plugin with extended features. You can purchase the pro version right from [here](https://zetamatic.com/downloads/woocommerce-mailchimp-newsletter-discount/?utm_src=woo-mailchimp-newsletter-discount/)

While you are checking our pro version we are working another cool feature of the pro version.

= Why I am not getting the discount emails even after subscribing? =
There might be different reasons for that.
* First please wait for 4-5 mins after subscribe. This might take some time to get an email.
* Then check your spam folder of your email. Might be the email has been reached out there.
* If that doesn't solve your issue then check whether all the normal WordPress related emails are working fine or not.

= Is there any test email feature available? =
Yes, absolutely. You can add your email to the plugin settings and try to subscribe. Your email will be never added to the MailChimp list. You can check whether the coupon emails are working fine or not without subscribe to the newsletter list.

= I have a suggestion for the plugin. How should I contact you? =
We are always looking for feedback from you. Whether its a completely new plugin or adding some custom functionality to any existing plugin, we are always ready to do that. You can contact us at accounts@zetamatic.com

= Can I use the popup in the free version? =
No, the popup is available in the [pro version](https://zetamatic.com/downloads/woocommerce-mailchimp-newsletter-discount/) of the plugin.

= Not receiving emails? =
Check SMTP settings of your Web Host / WordPress.

= How to use it as a form? =
To use this plugin as a form, you just need to copy the following shortcode: [wc_mailchimp_subscribe_discount], and paste it on the post/page/sidebar, that you want the form to appear.

= Can it be used as a Gutenberg block? =
Yes, We have added support for the gutenberg block in the 0.3.4 version onwards. Now you can easily display forms anywhere on your site using the WC MailChimp Subscribe block (Our newly added Gutenberg Block).
