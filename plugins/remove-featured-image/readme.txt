To show/hide/remove featured images on individual posts or from all posts.

== Description ==

Some times developer wants to show/remove/hide the featured images from single post or from all posts details page. Also some times developers want to hide from all the posts and only want to show specific post page. This plugin will fullfill there requirement. 


== Installation ==

For a manual installation via FTP:

1. Extract the downloaded plugin and upload to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' screen in your WordPress admin area

To upload the plugin through WordPress, instead of FTP:

1. You can also [download](https://downloads.wordpress.org/plugin/remove-featured-image.zip "Download Remove Featured Image") the plugin.
2. Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate.
3. Activate the plugin through the 'Plugins' screen in your WordPress admin area

== Frequently Asked Questions ==

= To remove featured image from single post or page =

1. Open the post/page in edit mode. 
2. Go to right side bar Remove Featured Image section, Select the option(`YES/NO`) and update.

= To hide featured image on all single posts/Pages =

1. Go to 'Remove Featured Image' Section in left sidebar.
2. Select the option(`YES/NO`) to remove featured image from all posts or pages.
3. Click Save.
