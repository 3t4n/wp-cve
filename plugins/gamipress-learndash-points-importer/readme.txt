Tool to migrate LearnDash user points to GamiPress points.

== Description ==

GamiPress - LearnDash Points Importer lets you import the user points balance from LearnDash to [GamiPress](https://wordpress.org/plugins/gamipress/ "GamiPress")!

= Features =

* Import LearnDash user points to any desired GamiPress points type.
* Choose how to process the user points, by summing both balances or keeping only the LearnDash balance.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.

== Frequently Asked Questions ==

= How to import the LearnDash user points? =

1. Navigate to WP Admin area -> GamiPress -> Tools -> Import/Export tab.
2. At box "LearnDash Points Importer" you will find settings to set to which points type perform the import.
3. Click the button "Import LearnDash Points" and wait until process gets finished.
4. That's all!
