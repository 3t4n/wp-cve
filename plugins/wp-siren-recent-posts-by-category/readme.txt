== Description ==

WP Siren Recent Posts by Category is a highly customizable widget, that can display your all/favorite categories posts of your blog.With highly customizable options and features, this plugin will allow you to add number of posts by category, excerpts, thumbnails, icons, style, colors and comments. Check out the following options you can pull over with this plugin.


1) General

a) Title
b) No of Posts
c) Select a Category



2) Excerpts

a) Include Excerpts
b) Excerpts Length



3) Thumbnails

a) Include Thumbnail
b) Thumbnail Float
c) Thumbnail Width
d) Thumbnail Height


4) Icon

a) Add Icon
b) Icon Width
c) Icon Height



5) Style

a) Widget Background Color
b) Widget Text Color
c) Widget Link Color
d) Widget Title Color


6) Comments

a) Include Comments       
b) Comments Text



7) Credits [Optional]

Used to enable/disable credits (Powered by WPSIREN hyperlink)


== Installation ==

1. Go to Plugins > Add New
2. Select Upload, Choose the zip file and click on Install Now
3. Activate it.
4. Go to widgets dashboard. Drag & Drop the widget, Set Options and Save it.

== Frequently Asked Questions ==

1. Can i use this widget as Recent Posts Widget ?
A. Yes, Select ALL from the 'select category' dropdown
