A plugin for setting the featured video in SiteOrigin's Focus theme.

== Description ==

Focus Videos is a plugin for SiteOrigin's Focus theme. A meta box is added to posts below the WordPress editor. Add videos from YouTube, Vimeo or self-host in the WordPress Media Library. Videos are prominently displayed above the theme's main content area. 

Focus Videos integrates with Focus and offers a premium video field, perfect for sites using s2Member, offering a teaser to all users and premium content to members.

== Installation ==

Download the plugin ZIP file to your desktop. Install from Plugins > Add New > Upload Plugin > Activate.

== Documentation ==

[Documentation](https://siteorigin.com/focus-documentation/) is available on SiteOrigin.
