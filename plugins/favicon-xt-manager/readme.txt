Upload and install a Favicon image to your WordPress website. When using this simple WordPress plugin you can easily upload and use your own Favicon icon for your Wordpress website.

== Description ==
[Favicon XT-Manager WordPress plugin](http://xtthemes.com/favicon-xt-manager/ "Favicon XT-Manager WordPress plugin Link at xtthemes.com") Favicon XT-Manager is an easy to use WordPress plugin to add a favicon to your WordPress website. A favicon is the little icon that browsers display next to a page's title on a browser tab, or in the address bar next to its URL. By using this simple plugin you will be able to easily upload a favicon to your Wordpress site.

= Favicon XT-Manager WordPress Plugin Features =
* Very simple and easy to use.
* No complicated menus or settings.
* Simply upload an image of the correct size to use as your own favicon logo.
* Easily Enable or disable the favicon on your website.
* Option to Store two favicon logos, you select which favicon to use on your website.
* Compatible with most versions of WordPress.

= How Does It Work? =
This plugin works by providing PHP codes that enable the display of an icon to the server. As users access pages on your website, Wordpress will load action hooks known as `wp_header` as requested. As each page is loaded so to are the `wp_header` this enables the display of your favicon on each page request. Any pages such as home page, post pages, archives, category pages or custom pages will each load the Wordpress action hooks `wp_header`. As the hook is enabled, web browsers will then display the favicon image on your website.

== Installation ==
This section describes **how to upload Favicon image icon on your WordPress website**. Read the following sections carefully to install and use this WordPress plugin to change favicon image of your website.

= Search and Install from your WordPress dashboard/admin panel =
1. When you are logged into your WordPress, from the dashboard/admin panel go to the 'Plugins' menu and click the 'Add New' button.
2. From the Plugins installation page, locate the Search Plugins, enter 'Favicon XT-Manager' in the search bar and click 'Search Plugins.'
3. You will find 'Favicon XT-Manager plugin By Chris Bennett' in the list.
4. Click the 'Install Now' button to install the plugin or click 'Details' to read more. The 'Install Now' link is also available on the details popup window.
5. Once the plugin is installed, you need to activate the plugin to use it on your website.

= Manual Installation =
1. To install the plugin manually, visit [Favicon XT-Manager WordPress plugin](https://wordpress.org/plugins/xtthemes-favicon-manager/ "WordPress plugin to change favicon image") description page.
2. Click the 'Download Version 1.0' button on the page to download and save the plugin zip file named `xtthemes-favicon-manager.zip` to your computer.
3. Login to your WordPress admin panel. Go to Plugins -> Add New -> select 'Upload' navigation on the page. Click the 'Browse' button, select the downloaded `xtthemes-favicon-manager.zip` file from your computer and click the 'Install Now' button. WordPress will upload and automatically extract and install the plugin in the appropriate directory for you.
<br />
**OR**
<br />
Extract the `xtthemes-favicon-manager.zip` file and upload all the files inside 'xtthemes-favicon-manager.zip' folder to your website's `wp-content/plugins` directory. Go to plugins and activate the Favicon XT-Manager plugin from the list.
4. Reload the WordPress Admin Panel and you will see a new menu 'XtManager Favicon' on the left sidebar menu of you administrator panel.
5. Click and open the 'phpxt Favicon' page. Here you can upload your own favicon images to your WordPress website.
6. You can also completely enable or disable the favicon code by selecting the respected dropdown menu.

= What is exactly is a Favicon? =
A favicon is the little icon that browsers display next to a page's title on a browser tab, or in the address bar next to its URL. By using this simple plugin you will be able to easily upload a favicon to your Wordpress site.

= What are standard image resolutions for a favicon? =
For IE, Microsoft recommends 16x16, 32x32 and 48x48 favicon.ico file, for iOS, Apple recommends resolutions at most 180x180 while some other platforms look for PNG files with various resolutions, like the 96x96 pixels.

= Can I use multiple favicons in a browser using this plugin? =
No! You can only use one favicon at a time within a browser. But you can upload two favicon's to your Wordpress admin area and select which favicon to use.

= What requirements are needed to use this plugin? =
You should have an image in the Format: .png, .gif or .ico with a standard size 16x16, 32x32 or 48x48 to use as a favicon.

= Can I turn off the favicon display? =
Yes! On the plugins page you can find the dropdown menu to enable or disable the favicon logo. Just choose the appropriate option and save.

= How can I deactivate or uninstall the plugin? =
Go to the plugins menu and deactivate the plugin from the installed plugins list.
