Five calculation options or all at once

The shipping costs will be calculated for each rule, with different results depending of the choosen option:

(once):

If one or more products matches the selection criteria, the cost will be applied once, no matter how many products do so.

* [qty]:

The calculated cost will be the result of multiplying the amount of products that matches the selection criteria by the value entered.

* weight:

The calculated cost will be the result of multiplying the weight of products that matches the selection criteria by the value entered.

* [group]:

The calculated cost will be the result of multiplying the number of groups (see the help section group-by that matches the selection criteria by the value entered.

% (percentage):

The percentage will be calculated based on the product price before taxes. If there is a sale price, it will be used instead.

Composite:

We can set the four calculations at once: once, * qty, * group and %. Those that are set with value 0 will not influence the calculation.

Be aware with grouping:
Keep in mind that if we set grouping products in any way, these will be taken for the calculation of costs as a single block: either all the members of the group, or none (see the group-by help section).
