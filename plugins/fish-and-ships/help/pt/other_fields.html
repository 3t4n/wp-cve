Outros Campos

Fora da tabela das regras existem outros campos que vamos explicar em baixo:

Titulo do Método:

Este é nome do custo de envio que o seu cliente vai ver (se não o mudarmos com uma Acção Especial)

Estado do Imposto:

Devemos aplicar impostos sobre os custos de envio?

Agrupar por Global: [PRO]

Ao desmarcar esta opção vai poder definir um critério de agrupamento diferente em cada selecção

Agrupar por [para todos os selectores]:

A forma como os produtos no carrinho são agrupados para analizar as condições das regras (ver agrupar por)

Tipo de cálculo:

Aqui pode escolher entre aplicar o custo de todas as regras correspondentes, apenas as mais cara, ou a mais barata.

Custo de envio Min:

Caso esse método de envio seja aplicavél, pode definir o custo minimo para esse método.

Custo de envio Max:

O custo maximo que esse método de envio não vai exceder em caso algum.

Sistema de registos (Escrever logs:)

Se o activar, um registo de todos os cálculos para este método vai ser guardado durante 7 dias, para o ajudar a compreender ou a depurar as suas definições. Também é muito útil para nos pedir ajuda ou para nos ajudar a depurar algum erro que possamos ter feito.

Podemos activá-lo para todos os utilizadores, ou apenas para os utilizadores com o papel de administrador (recomendado se quisermos depurar uma loja aberta).
