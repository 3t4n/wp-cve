WP Analytics Tag Manager is a plug-in that you can easily manage tags embedded Google Analytics, such as Yahoo! analysis.


== Description ==
WP Analytics Tag Manager allows you to manage the embedded tags such as GoogleAnalytics. 
There is no need to edit the PHP file. 

There are also the following functions. 
* Output Page setting (all page or single page[PostType=page])
* Output Position setting
* Excluding IP address setting 
* Excluded Hosts name setting 
* Status setting (ON • OFF) 


WP Analytics Tag ManagerはGoogleアナリティクス、Yahoo!アクセス解析などの埋め込みタグを簡単に管理できるプラグインです。
管理画面で設定すれば、テーマのPHPファイルを編集しなくても各ページのヘッダーもしくはフッターに出力されます。
Adwordsコンバージョンタグ、Adwordsリマーケティングタグ、Yahoo!プロモーション　コンバージョンタグ、YDN　コンバージョンタグ、YDN　リターゲティングタグ。


•基本機能
* 出力ページ設定(全ページ　or 指定固定ページ)
* 出力位置設定（ヘッダー or フッター)
* 解析タグ設定
* 除外IPアドレス設定
* 除外ホスト名設定
* ステータス設定（ON•OFF）

== Installation ==
1. Upload `WP Analytics Tag Manager` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. You can set a AnalyticsTag by settings page.
