Create a new custom field for "GravityForms" plugin called "Date/Time" field.

== Description ==

Create a new custom field for "GravityForms" plugin called "Date/Time" field:

- Add a new field called "Date/Time" field to GravityForms fields.
- User can drag that field into the form and use it.
- The field will allow user to select both Date and Time for one field.

Upload the Date/Time Fields for Gravity Forms plugin to your blog, Activate it, check it.
