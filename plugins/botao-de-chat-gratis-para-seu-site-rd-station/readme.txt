== Description ==

O Botão de Chat Grátis para Seu Site da RD Station ajuda você a transformar os visitantes do seu site em contatos dentro do WhatsApp da sua empresa! É gratuito e, ao instalá-lo, você facilita conversas e recebe os leads por e-mail. 

<h3>A importância do WhatsApp para sua empresa</h3>
<p>Quase 94% dos celulares brasileiros tem o aplicativo do WhatsApp instalado no celular. São 169 milhões de usuários utilizando o mensageiro todos os dias. E por que não estar onde seus clientes estão?</p>
<p>Atenda seus leads em uma comunicação rápida, fácil e preparada para converter ainda mais!</p>

<h3>Como usar o botão do WhatsApp no seu site</h3>
<p>Inserir o Botão de WhatsApp no seu site é muito simples. Veja o passo-a-passo que fizemos para você:</p>

[youtube https://www.youtube.com/watch?v=2CkqKL-DEaE]

<br>

<ul>
    <li>Na aba Plugins, clique em “Adicionar Novo”.</li>

    <li>Na aba de Pesquisa (canto superior direito), escreva “RD Station” e dê enter.</li>

    <li>Selecione o plugin “Botão de Chat Grátis para seu site”.</li>

    <li>Clique em <strong>"Instalar Agora"</strong>.</li>

    <li>Espere a instalação terminar e clique em <strong>"Ativar"</strong>.</li>

    <li>O plugin <strong>Botão de Chat Grátis para Seu Site</strong> aparece na aba de plugins instalados e no menu lateral.</li>

    <li>Para configurar, selecione o menu lateral <strong>“Botão Chat | RD”</strong>.</li>

    <li>Preencha o campo com seu email — é por ele que você recebe seus leads</li>

    <li>Ative a opção mostrar o Botão de Chat, preencha os dados e salve as alterações.</li>

    <li>Você também pode colocar uma uma mensagem amigável para estimular as conversas!</li>

    <li>Depois é só ir para a página do site e conferir o botão de WhatsApp instalado.</li>
</ul>
