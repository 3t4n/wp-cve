== Description ==
Using this plugin we can convert MyISAM storage engine type to InnoDB . We always recommend backing up your MySQL database before using this plugin.

== Installation ==

1. Upload the 'simple-myisam-to-innodb' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Plugin can be used from "Simple MyISAM to InnoDB" menu in left panel.
