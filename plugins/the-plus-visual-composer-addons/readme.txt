== Installation ==

1. Go to plugins in your dashboard and select 'add new'
2. Search for 'ThePlus Addons' and install it
3. Now You will have all options available in your Visual Composer Panel.
4. Check Setting of plugin to know extra options.
5. You can start building website using brand new addons.
