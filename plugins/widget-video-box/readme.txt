== Description ==
Simple plugin to add video into your widget box. Supports Youtube, Dailymotion, Vimeo and many other sites that provide embed code.. just don't forget to edit the embed code to the size of your widget box

Tested up to 3.5.2


#### Functionality
* Simple plugin to add video into your widget box.


== Installation ==
Follow the steps below to install the plugin.

1. Upload the files to your /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag and drop "Video Box" into your preferred widget area
4. Grab embed code from desired site and edit the height and width to your desired size
5. Paste embedd code into the widget editor and click "Save"

Editable fields include:
Title - [for the widget]
Embed Code - [remember to edit sizes]
More Text - [goes under the video]
Link - [will create a link for the More Text]
