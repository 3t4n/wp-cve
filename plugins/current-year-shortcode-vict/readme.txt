A shortcode to display the current year. Never worry about updating the current year on your site again.

== Description ==
A shortcode to display the current year, for example in the footer section. 
When using this shortcode you don't have to manually update the year on your website, when a new year begins.

Just add the shortcode where you need it, within the text. Don't forget to use the [ ] brackets.

**[vict_year] displays the current year**


== Installation ==
1. Upload the plugin files to the \\\'/wp-content/plugins/\\\' directory, or install the plugin through the WordPress plugins utility directly.
2. Activate the plugin from the plug-in menu in WordPress.
3. Add the shortcode wherever needed.

== Frequently Asked Questions ==
= Q: What shortcode do I use? =
A: [vict_year] displays the current year wherever you want.

= Q: Can I use the code only for my site footer section? =
A: No, you can use the shortcode wherever you want. But ofcourse it is a really good idea to use the shortcode in the footer/copyright section.

= Q: where can I use the shortcode? =
A: anywhere you can use an shortcode in WordPress.
