WIDEO视频播放器，整合dplayer,支持mp4,flv,hls等协议视频播放

== Description ==

Wideo视频播放器是是将开源dplayer整合wordpress到开发的一款视频播放器，设置简单、功能强大。
主要功能：
1、可根据自己网站风格定制播放器的功能样式；
2、支持mp4,flv,m3u8等常规视频的播放；
3、支持优酷、腾讯、爱奇艺等网络视频的在线播放；
4、开启视频解析的情况下支持优酷、腾讯、爱奇艺等网络视频的解析播放；
5、支持视频弹幕；
6、支持多集视频，列表可以选择在视频底部或者右侧；
7、支持插入广告。

== Frequently Asked Questions ==

= 我该如何使用该插件？=

该插件已经整合到经典编辑器，可以直接点击按钮插入视频；如果使用古腾堡编辑器的话需要用到短代码【wideo】视频地址【/wideo】，视频地址可以为本地视频或者优酷、腾讯等在线视频。

= 使用解析后，手机端有广告怎么办？ =

解析的地址是站长在网络上搜集的，只是验证了可用性，并没有验证是否存在广告，大家可以根据自己的需要进行更换。
