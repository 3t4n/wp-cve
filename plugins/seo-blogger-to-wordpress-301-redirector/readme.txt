== Description ==

This plugin will 301 redirect all incoming traffic from your Blogger account to your newly setup Wordpress account. Move from Blogger to wordpress with great ease.

It also has a module that'll download all images from your picasa/blogger account to your server. Obviously, it will update all the image paths so they don't disappear.  

Please read the [documentation](https://suhastech.com/the-easiest-blogger-to-wordpress-migration-guide-picasa/) before you continue.

Features:

*   Works with Blogger custom domains and blogspot.com domains
*   Search Engine Friendly Redirects. You'll never lose traffic
*   Has a template generator for blogspot.com domain users
*   Fixes some common "Blogger Importer" errors

PS: We now provide professional [Mongo to SQL Migration Service](https://mongotosqlmigration.com/). 

== Installation ==

1. Unzip the package, and upload 'seo-blogger-to-wordpress' to the "/wp-content/plugins/" directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it. All your URLs will now be redirected.



== Frequently Asked Questions ==

= Will I retain my Page Rank and traffic? =
Yes, you will. Your Page Rank and traffic will be transferred, provided you've followed [the tutorial](https://suhastech.com/the-easiest-blogger-to-wordpress-migration-guide-picasa/) correctly. I guarantee it.

= I'm using custom domain on Blogger, would this work? =
Yes, Absolutely, all your Blogger URLs will 301 redirected to your current wordpress URL. 

= I'm using myblog.blogspot.com domain, would this work? =
Again, Yes. We will tell the search engine bot to crawl the wordpress URLs instead (using rel="canonical") and meta refresh real visitors to the Wordpress URL. 

= Can I import all my pictures that uploaded earlier in Blogger? =
Yes, There's an importer to do just that.

= I'm afraid to migrate my blog. Can you do it for me?
No, we have stopped it.
