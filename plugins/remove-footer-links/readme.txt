== Description ==
Simple plugin to remove footer links
Features of this plugin
- Auto detect footer credit link and remove.


== Installation ==
**Direct from Dashbaord:**
<ul>
<li>Please go to your website Dashboard</li>
<li>Plugins and Add new</li>
<li>Search for Remove Footer Links</li>
<li>Install it and activate it.</li>
</ul>

**Manually Installation:**
<ul>
<li>Please go to https://wordpress.org/plugins/remove-footer-links/</li>
<li>Download Zip file from there</li>
<li>Please go to your website Dashboard</li>
<li>Plugins and Add new</li>
<li>Uplaod zip file there</li>
<li>Install it and activate it.</li>
</ul>
