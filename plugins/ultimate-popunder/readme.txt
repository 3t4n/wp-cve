Add PopUnder links to your WordPress site and also track these events in your external marketing tools.

== Description ==
Generate a flexible PopUnder AD on your WordPress site using the [lalund.js logic](https://goo.gl/eEWhf8). This allows you to use several URLs as rotating test logic for the links. This plugin creates a custom post type to add your links along with weighting and logic. You can edit global details about the functionality under the default settings menu.

= Special Features =
You can track the data from the PopUnder activity in 3rd party systems:

* Google Analytics
  * dataLayer.push event
* Google Tag Manager
  * Univeral Analytics event

== Installation ==
Install the plugin and then edit the settings under the default WordPress Settings menu.
