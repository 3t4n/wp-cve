== Description ==

This plugin sends an email to the administrator for each login. The goal of this plugin is to improve the security of your website and keep under control all the accesses to your website.

You can customize the email template and exclude from notifications the admin access.

== Installation ==

1. Upload `login-alert.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

OR

Just grab the Login Alert plugin from the built in plugin directory and choose "Install now", hit ok, then select the "Activate Plugin" link.
