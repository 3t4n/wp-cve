Яндекс Вебмастер для вашей консоли WordPress

== Description ==

Данный плагин позволит вам легко просматривать информацию Яндекс Вебмастер прямо в консоли вашего сайта

Яндекс. Вебмастер — сервис Яндекса для вебмастеров, панель инструментов для оценки индексации сайта и настройки описания сайта в результатах поиска Яндекса. Используется для показа статистики сайта.


== Installation ==

1. Распакуйте архив с плагином
2. Загрузите папку с плагином в директорию /wp-content/plugins/ на вашем сервере
3. Активируйте плагин в разделе плагины в панели администрирования WordPress
