== Description ==
#### WordPress Popup Plugin

FireBox is a WordPress Popup Builder Plugin that boosts conversion rate. It helps you convert your visitors into subscribers and customers with eye-catching popups. Deliver the right message to the right person with precision targeting and smart triggers without any coding knowledge.

[FireBox Website](https://www.fireplugins.com/)  |  [Sample Popups & Demos](https://www.fireplugins.com/templates/)   |  [Docs](https://www.fireplugins.com/docs/firebox/)

#### A new popup editing experience

FireBox is based on the new WordPress Block Editor, Gutenberg which brings a whole new popup editing experience. With blocks, you can insert, rearrange, and style your popups with very little technical knowledge. Instead of using custom code, you can add a block and focus on your content. 

#### Create any kinds of campaigns

Easily create a scroll popup, exit popup, opt-in popups, contact form popup, modal, lightbox, floating bars, sticky bars, sidebars, scroll boxes, slide-ins, fullscreen overlays, welcome mats, lightboxes, EU cookie bars & more.

#### Smart Triggers
Detect your visitors' behavior and use smart triggers to display the right campaign to the right person at the right time. Our WordPress Popup Maker can help you create page load popup, scroll popup, exit popup and more.

* **[On Page load](https://www.fireplugins.com/docs/firebox/popup-triggers/the-page-load-trigger/)** - Automatically show a popup when the page is fully loaded.
* **[On Page Ready](https://www.fireplugins.com/docs/firebox/popup-triggers/the-page-ready-trigger/)** - Automatically show a popup when the page is ready.
* **[On Scroll](https://www.fireplugins.com/docs/firebox/popup-triggers/the-scroll-depth-trigger/)** (Pro) - Show a popup when the visitor has reached the specified amount of scroll depth set in percentage or pixel.
* **[On Element Visibility](https://www.fireplugins.com/docs/firebox/popup-triggers/the-scroll-depth-trigger/)** (Pro) - Show a popup when a specified element(s) enters the viewport after scroll down.
* **[On Exit Intent](https://www.fireplugins.com/docs/firebox/popup-triggers/the-exit-trigger/)** (Pro) - Show a popup when the visitor intends to leave the page by tracking the mouse movements. 
* **[On Click](https://www.fireplugins.com/docs/firebox/popup-triggers/the-click-trigger/)** - Show a popup when the user clicks on a button, link, or banner.
* **[On Hover / Mouseover](https://www.fireplugins.com/docs/firebox/popup-triggers/the-hover-trigger/)** - Show a popup when the visitor moves their mouse over a specified part of the page.
* **[On AdBlock Detect](https://www.fireplugins.com/docs/firebox/popup-triggers/the-adblock-detect-trigger/)** (Pro) - Show a popup when the visitor has been detected using an AdBlocker. Supported AdBlockers: AdBlock, Adblock Plus, uBlock Origin, and uBlocker.
* **[On Idle / User Inactivity](https://www.fireplugins.com/docs/firebox/popup-triggers/the-idle-trigger/)** (Pro) - Show a popup when the mobile visitors go idle for a specific amount of time.
* **[On External Link Click](https://www.fireplugins.com/docs/firebox/popup-triggers/the-external-link-click-trigger/)** - Show a popup when the users click on external links.
* **[Floating Button](https://www.fireplugins.com/docs/firebox/popup-triggers/the-floating-button-trigger/)** - Show a floating button that triggers your popup.
* **[Manually](https://www.fireplugins.com/docs/firebox/extending/firebox-javascript-api/#examples)** - Show a popup programmatically using Javascript.
