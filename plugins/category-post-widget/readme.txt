== Description ==

Category Post widget displays posts from a specific category. Number of posts to be displayed can be customized.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `category-post.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag and drop the Category Post Widget in to required sidebar.
