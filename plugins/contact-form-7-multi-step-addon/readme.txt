Enables the Contact Form 7 plugin to create multi-step forms, adds new shortcodes.

== Description ==

WordPress Plugin by [ThemeREX](https://themerex.net/) extends the functionality of Contact Form 7. It adds a Multi-Step opportunities without reloading the page. The visitor can fill out the form step-by-step. Also the form adds additional shortcodes to Contact Form 7. Now you have shortcodes of step start and step end, progress bar, next and previous buttons.

= Key advantages =

**Responsive design.** It's optimized to be viewed perfectly on different devices.

**A well thought-out shortcode settings.** New elements are simply built in into your Contact Form 7.

**No Reloads.** The visitors does not have to reload the page for the next/previous step.

**Fast.** Doesn't slow down the site. Plugin zip itself is just 17kb.

= Main features =
* Quick Installation
* Easy to Use

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress. You'll find 'Contact Form 7 Multi-Step Addon' on your main WordPress dashboard.

==Plugin Usage==

1. Navigate Contact - Contact Forms.
2. Add new or edit existent form.
3. Separate steps by Step Start and Step Finish shortcodes.
4. And next and previous buttons by adding Next and Prev Shortcodes on proper steps.
5. You may also add a form filling Progress Bar by adding a progressbar shortcode.
