Easy solution for building attractive pages with WPBakery Page Builder.
== Description ==
This plugin contains almost all the addons that are mostly use to build pages.It contains 20+ most using addons for WPBakery Page Builder to build your website in few click.These addons contains very simple and easy settings and new user can easily use it.

Note: This plugin requires <a href="http://codecanyon.net/item/visual-composer-page-builder-for-wordpress/242431?ref=labibahmed" target="_blank"> WPBakery Page Builder</a>.<br>
