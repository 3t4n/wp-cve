WooCommerce Custom Order Status plug-in allows you to create and manage new order statuses for WooCommerce.

== Description ==

<strong>Ni WooCommerce Custom Order status</strong> plugins helps you to improve your order system. You can easily create additional order status, edit, update and delete with integration of current WooCommerce Version. 

<p>Order stats report show the order status, order count and order total, filter by order days and sort by order total, order count and order status.</p>

<p>Send the custom order status email on custom order status changed,  provide the option to enable or disable the <strong>custom order status email</strong>, enter the subject line and email message.</p>
Example:

Some shops need more order statuses like "Awaiting Shipment", "Building" etc. You can easily add this yourself using this plug-in. It puts at your disposal customizable labels to highlight and differentiate different steps of your users' checkout process and to have an immediate overview on orders in your shop. You can create unlimited statuses for your shop

The WooCOmmerce Custom Order status plugin made everything in front of you and give a chance to check the flow of "Order" placement to delivery Product.  We created our plugin in very simple forms and easy to use. So You can easily manage everything without a much more complicated development knowledge. 


Woocommerce Custom order  status plugins Features:

*	Easily Create New Custom order status
*	Get status on an order page
*	Edit, Update and delete custom orders
*	Fast And Easy Install
*	Assign custom color to order status.
*	Create an unlimited order status
*	Very flexible
*	User Friendly 
*	Fully customizable

How To Get Started:
1.	Go to Wordpress>>Plugin>>Add new Plugin
2.	Search or Find "Ni Woocommerce Custom Order Status"
3.	Click On Install

The Ni Woocommerce Custom Order Status Plugins has full stop and complete solutions of order statuses.
