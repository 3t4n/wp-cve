== Description ==

GetChat.App is a super simple and effective WhatsApp Chat Button for your website.

Add WhatsApp Chat, Facebook Messenger or an email button to your website in seconds

### Features

* Easy to install and use
* Works on both Mobile and Desktop
* Users are sent to their WhatsApp app to chat with you (mobile and desktop)
* Left or right position
* Enter your own WhatsApp or WhatsApp Business number
* Enter your Facebook Page ID to enable the Facebook Messenger chat button
* Enter your email address and email subject to enable the email button
* World-class support from the developer should you need it!

GetChat.App WordPress plugin makes use of third party code located on https://getchat.app in order to build the chat box on your website. The terms of service can be found [here](https://getchat.app/terms-of-service).

### External dependencies

Get Chat App relies on an external JS file that is loaded from getchat.app. This file is copyright and allows the user to use the chat service and JS servers.

== Installation ==

### Download the Get Chat App plugin ZIP file.  
**Installing the ZIP File**  

1. Log in to your WordPress admin panel.   
2. Go to WordPress Plugins page and click on Add New button, then Upload the .zip file that you downloaded.
3. Then click "Install Now" and then "Activate"
4. From the left panel of Dashboard, click on the link called "Get Chat App"
	
**Installing via WordPress Plugin Search**  

1. Log in to your WordPress admin panel.   
2. Go to WordPress Plugins page and search for "Get Chat App"
3. Then click "Install Now" and then "Activate"
4. From the left panel of Dashboard, click on the link called "Get Chat App"


That's it! WhatsApp Chat will now be enabled on your site!

== Frequently Asked Questions ==

= How does this work? =

Once a user types a message in the WhatsApp chat box on your site, they are instantly taken to their WhatsApp app on their phone with the message and your number pre-loaded for them. The message is sent using their WhatsApp app which is then received on your WhatsApp app.

= What about Desktop users? =

WhatsApp Web will open instead of the Mobile App. Your mobile number and their message will be pre-loaded and sent to you via their web.whatsapp.com
