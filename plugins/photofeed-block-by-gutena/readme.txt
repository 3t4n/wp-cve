== Description ==

Gutena PhotoFeed is a free and simple plugin for WordPress that allows you to display your Instagram photos in a gallery. You can set the number of columns and give required spacing between the images, as well as choose a hover effect.

== Features ==

* Pull Instagram images by Instagram Access Token
* You can set spacing between Instagram images
* You can set the number of images to display 
* Choose hover color with opacity and effect for gallery
* You can enable/disable caption for images
* You can set custom caption color

== Installation ==

1. Visit 'Plugins > Add New'.
1. Search for 'Gutena PhotoFeed' and install it.
1. Or you can upload the `photofeed-block-gutena` folder to the `/wp-content/plugins/` directory manually.
1. Activate Gutena PhotoFeed from your Plugins page.
