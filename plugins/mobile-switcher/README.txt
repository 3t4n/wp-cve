Mobile Switcher plugin allow to set themes for mobile phones and tablets

== Description ==

The plugin allows you to set different themes for display on mobile devices.  
With this plugin you can select a theme that will be displayed if a user visits your site with a Tablet PC, mobile phone or desktop.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
