== Description ==
Show the visitor's information in a widget, including the IP address, country, country flag, city, region, language, operating system, and browser. 

You can **choose** the information to be shown. It's shown in the visitor's **language** automatically.

= How to use it? = 
1. Go to the admin menu **Appearance** > **Widgets**. 
1. Then **drag** the IP address widget to your **Sidebar**. 

Here is a live demo: [https://free-proxy-list.net/blog/](https://free-proxy-list.net/blog/)

= 3rd Party API =
- The plugin is relying on our [IP Widget API](https://www.find-ip.net/ip-script) service
- It gets the visitor's information by that API

== Installation ==

To use the widget you should do this:

1. Download and unzip the last version of this plugin.
2. Upload the ip-address-widget folder to /wp-content/plugins/
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Dray this widget to your sidebar in the 'Widgets' menu
