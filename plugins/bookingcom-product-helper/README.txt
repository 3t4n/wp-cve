== Description ==

The Booking.com Product Helper allows you to embed any Booking.com affiliate product on your website. With this plugin, you simply paste the embed code from the [Affiliate Partner Centre](https://www.booking.com/affiliate-program/v2/index.html?aid=1657077&utm_source=wordpress&utm_medium=marketplace&utm_campaign=booking_product_helper) and generate a shortcode, which can be used anywhere on your WordPress website.

= Key features =

* Add any Booking.com affiliate product
* Use the shortcode anywhere on your website
* Lightweight and easy to use
* Multi-lingual (over 40 languages)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/bookingcom-product-helper` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the 'Settings'->'Booking.com Product Helper' screen to configure the plugin
4. Other instructions you may find in the informational column on plugin page


== Frequently Asked Questions ==
= Can I use the Booking.com Product Helper even if I’m not a Booking.com affiliate? =

No, unfortunately not. You need to sign up to the Booking.com Affiliate Program to be able to use the Booking.com Product Helper.

= What’s the Booking.com Affiliate Programme and how can I join? =

The Booking.com Affiliate Program ensures you can earn commissions on any reservation made through the Booking.com products & links integrated in your site or blog. You can join by going to Booking.com and signing up to the program.
