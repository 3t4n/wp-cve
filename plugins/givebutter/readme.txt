Embed Givebutter widgets using shortcodes.

== Description ==
Givebutter Widgets is the most powerful way to drive donations to your campaigns by putting calls to action right where your donors already are â€“ on your website! Donors can submit a transaction while remaining on your site, without being redirected.

Widgets created from the Givebutter dashboard are dynamic, which means any changes you make on Givebutter will update on your Widget, without needing to update your site's code. You can embed Givebutter Widgets on specific parts of your website by copying and pasting provided HTML code into the editor area of the website where you'd like them to appear.

This plugins allows to embed widgets using shortcodes. E.g. [givebutter-widget id="X"]

== Installation ==
1. Download the latest zip file and extract the ***givebutter-widget*** directory.
2. Upload it to your ***/wp-content/plugins/*** directory.
3. Activate ***Givebutter Widget*** on the ***Plugins*** menu in WordPress.
4. Navigate to ***Settings > Givebutter Widget*** and enter your ***Givebutter Account ID***.
