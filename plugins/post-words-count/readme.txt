== Description ==

Post Words Count is a simple plugin which shows the total words of a post and thumbnail of the post at All Post section in dashboard. So, you can easily find out the total words of a post and the including or excluding the thumbnail of the post.

### Features Include:

* **Show Total Number of Words of each Post.**
* **Show the Featured Image (thumbnail) of each Post.**
* **Sort posts based on total pots words**
* No extra settings are required
* Provide extra information about your Posts.
* It is very light-weight

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Go to Plugin Menu from your dashboard.
2. Click on Add New
3. Now upload "post-words-count.zip" file and click install now.
4. Finally active it. It is done.
