== Installation ==

1. Install the plugin through the Plugins interface or by uploading the `scroll-bar-with-back-to-top` directory to your `/wp-content/plugins/` directory.
2. Activate the Scroll Back to Top plugin.
3. You're done, the scroll back to top button should appear on all pages after you scroll down a little bit.  If you'd like to customize the design you may continue.

Optional Customization Steps

* A new setting page will be located within Settings > Scroll Settings, navigate to that page to for customization options.
